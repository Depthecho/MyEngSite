Главная страница:
Просто дефолтное представление всей информации о сайте в целом и предложение юзеру пройти введение по сайту (т.е. по каждому разделу обучения), которое он по своему желанию сможет пройти ну или же закрыть,(под вопросом) также юзеру будет предоставлена возможность пройти урок из 4 составляющих этого сайта (т.е. 10 карточек-слов, пару текстов, что-то из грамматики и тест на закрепление всего пройденного), также на главной странице для авторизированного юзера будет отображена его активность и возможность оставить отзыв о данном сайте
-----------------------------------------------------------------
Функционал:
Доп функционал к профилю:
	Кастомная модель User на основе AbstractUser с дополнительными полями:
		wants_intro - флаг, хочет ли пользователь видеть вводный тур
		last_active - время последней активности
		daily_goal - ежедневная цель по изучению слов

Обучение:
	Lesson - урок, содержащий 4 компонента
	WordCard - карточки слов (10 штук на урок)
	TextMaterial - текстовые материалы (2 на урок)
	GrammarTopic - грамматические темы
	Quiz и QuizQuestion - тест для закрепления
	UserProgress - прогресс пользователя по урокам

Для неавторизованных пользователей:
	Простое приветствие с описанием возможностей сайта
	Кнопки регистрации/входа
	Модальное окно с предложением пройти введение (после регистрации)

Для авторизованных пользователей:
	Вводный тур (если пользователь еще не отключил его):
	Пошаговое знакомство с интерфейсом
	Объяснение структуры уроков
	Возможность пропустить в любое время

Dashboard с активностью:
	Последние пройденные уроки
	Прогресс по текущему уроку
	Статистика изученных слов/текстов
	Индикатор выполнения дневной цели

Система уроков:
	Каждый урок состоит из 4 последовательных этапов:
	10 карточек-слов (изучение лексики)
	2 текста (чтение и понимание)
	Грамматическая тема

Тест на закрепление
Прогресс сохраняется после каждого этапа
После завершения всех 4 этапов урок отмечается как пройденный

Обратная связь:
	Форма для оценки сайта
	Возможность оставить текстовый отзыв
	Рейтинг от 1 до 5 звезд

Профиль:
В профиле будет предоставлена информация о самом юзере, что личная, что его активность
Будет возможность обновить свою информацию, а также настроить приватность, чтобы другие люди не могли видеть некоторые данные и (возможно) вашу активность
Также будет сделана система достижений и бейджиков за усердную активность (на данный момент бронза-серебро-золото)
На странице профиля можно будет перейти на карточки, тексты и тесты, которые уже были выполнены юзером ранее
Добавим функцию друзей и подписчиков, а также возможность приглашать друзей(через ссылку-приглашение)
И будет реализована возможность делиться своим результатом с друзьями


Карточки:
	В разделе "Карточки" будет 3(4, возможно) вкладки:
		1ая - это отображение имеющихся карточек
		2ая - добавление своих карточек, т.е. юзер вводит слово на родном языке, потом на английском и сохраняет эту карточку
		3ья - викторина, т.е. из добавленных юзером карточек будет составлена викторина, где ему будет предоставлен выбор с родного языка на англ или наоборот и следом будет сама викторина, т.е. ему будет показано слово, а в ответ надо ввести перевод данного слова
4ая - словарь, импортированный через API или возможность учить слова по категориям(например по профессиям, программист, врач и т.п.)
=====================================================================================
Повторение по алгоритму spaced repetition (как в Anki) – чтобы слова не забывались.
Голосовое произношение – TTS (Text-to-Speech) для прослушивания слов.
Категории карточек – возможность сортировать слова по темам (еда, путешествия и т. д.).
-----------------------------------------------------------------
Функционал:
Вкладка "Мои карточки"
	Отображение всех созданных пользователем карточек
	Фильтрация по категориям
	Поиск по словам
	Сортировка по дате добавления/частоте повторений

Подвкладка "Добавление карточек"
	Форма для создания новых карточек:
		Поле для слова на родном языке
		Поле для перевода на английский
		Выбор категории из существующих или создание новой
		Валидация вводимых данных
		Подтверждение успешного добавления

Подвкладка "Викторина"
	Настройки викторины:
		Направление перевода (родной→английский или английский→родной)
		Количество вопросов (10/20/50/все)
		Выбор категорий для включения в викторину
	Процесс тестирования:
		Поочередный показ карточек
		Поле для ввода ответа
		Проверка правильности с показом верного ответа
		Статистика по завершении теста

Подвкладка "Словарь"
	Выбор категорий слов из доступных через API:
	Профессии (программист, врач и т.д.)
	Повседневная лексика
	Деловой английский
	Просмотр слов выбранной категории
	Добавление понравившихся слов в личную коллекцию
	Фильтрация по сложности
	Словарный API (например, WordsAPI или DictionaryAPI)
		Получение:
			Переводов слов
			Примеров употребления
			Синонимов/антонимов
		Использование для:
			Автодополнения при создании карточек
			Расширения словарной базы
			Проверки правильности переводов
	Text-to-Speech API (например, Google TTS или IBM Watson):
		Произношение добавленных слов
		Настройка скорости/голоса (под вопросом )
		Возможность прослушать примеры предложений
		Реализация:
			Кнопка "Прослушать" для каждой карточки
			Кэширование аудиофайлов для часто используемых слов
	API для категоризированных слов (например, Lexicala)
		Получение:
			Тематических подборок слов
			Готовых наборов по профессиям/интересам
			Распространенных фраз и идиом

		Использование для:
			Быстрого пополнения словарного запаса
			Изучения специализированной лексики
Возможная система повторений (Spaced Repetition)
Алгоритм повторений (по аналогии с Anki):
	Расчет интервалов повторений на основе:
		Количества правильных ответов
		Сложности слова для пользователя
		Времени последнего повторения
		Формулы для расчета:
			Нового интервала
			Фактора легкости
			Даты следующего повторения

	Реализация в интерфейсе:
	Ежедневная подборка карточек для повторения
	Новые слова
	Слова, требующие повторения
	Процесс повторения:
		Показ слова
		Оценка сложности (Легко/Средне/Трудно)
		Корректировка расписания повторений
	Визуализация прогресса:
		График изученных слов
		Процент запомненных слов
		"Горячие" слова (которые часто забываются)
