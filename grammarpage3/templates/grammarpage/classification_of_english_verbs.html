{% trans "Classification of English Verbs" %}

{% trans "By meaning and function" %}

{% trans "English verbs can be classified in several ways based on their meaning, function, and grammatical properties." %}

{% trans "By meaning and function:" %}

  • {% trans "Notional verbs:" %} {% trans "express actions or states and can function as simple predicates." %}
    {% trans "Examples:" %} to work, to read, to live, to speak.
  • {% trans "Auxiliary verbs:" %} {% trans "help form grammatical structures but don't have full meaning on their own." %}
    {% trans "Examples:" %} to be, to have, to do. ({% trans "e.g." %} "I am reading", "He has finished", "Do you like?").

{% trans "By relation to direct object:" %}

  • {% trans "Transitive verbs:" %} {% trans "require a direct object to complete their meaning." %}
    {% trans "Examples:" %} to give (something), to buy (something), to write (something).
  • {% trans "Intransitive verbs:" %} {% trans "do not require a direct object." %}
    {% trans "Examples:" %} to sleep, to walk, to arrive.

{% trans "Note: Some verbs can be both transitive and intransitive depending on usage." %}
