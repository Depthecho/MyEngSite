{% translate "The Subject in English Grammar" %}

{% translate "The subject is the essential core of a sentence that indicates what the sentence is about - the person, thing, or concept that performs the action or is being described." %}

{% translate "Core Definition" %}

[Subject] + [Predicate]

{% translate "Key Characteristics:" %}

  • {% translate "Typically a noun, pronoun, or noun phrase" %}
  • {% translate "Answers the questions who? or what?" %}
  • {% translate "Determines verb agreement in number and person" %}
  • {% translate "Usually precedes the verb in declarative sentences" %}
  • {% translate "Can be simple (single word) or complete (including modifiers)" %}

{% translate "Examples:" %}

  • Birds fly.
  • The old oak tree stood tall.
  • She is reading a book.

{% translate "Primary Functions" %}

{% translate "1. Agent of Action" %}

{% translate "The doer/performer in active voice constructions:" %}

  • The chef prepared the meal.
  • Rain fell all day.

{% translate "2. Theme/Experiencer" %}

{% translate "The entity affected by or experiencing the action:" %}

  • The window was broken.
  • I heard a strange noise.

{% translate "3. Subject of Description" %}

{% translate "The entity being described or identified:" %}

  • This book is fascinating.
  • She became a doctor.

{% translate "Grammatical Properties" %}

{% translate "Property" %} {% translate "Description" %} {% translate "Example" %}
{% translate "Number Agreement" %} {% translate "Determines verb form (singular/plural)" %} He runs vs They run
{% translate "Case" %} {% translate "Nominative case for pronouns" %} I went vs He saw me
{% translate "Position" %} {% translate "Typically precedes the verb in statements" %} The sun rises
{% translate "Omission" %} {% translate "Optional in imperative sentences" %} (You) Sit down!

{% translate "Special Cases" %}

{% translate "1. Dummy Subjects" %}

{% translate "Existential and weather expressions with 'it' or 'there':" %}

  • It is raining.
  • There are many options.

{% translate "2. Inverted Subjects" %}

{% translate "Subjects following verbs in questions or special constructions:" %}

  • Where are you?
  • Never have I seen such beauty.

{% translate "3. Complex Subjects" %}

  • {% translate "Gerund subjects:" %}

    Swimming is good exercise.

  • {% translate "Infinitive subjects:" %}

    To err is human.

  • {% translate "Clause subjects:" %}

    What you say matters.

{% translate "4. Common Errors" %}

  • {% translate "Subject-verb disagreement:" %}

    Wrong: The list of items are long.
    Right: The list of items is long.

  • {% translate "Case errors:" %}

    Wrong: Me and John went.
    Right: John and I went.
