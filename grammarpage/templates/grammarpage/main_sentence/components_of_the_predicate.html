{% translate "Components of the Predicate" %}

{% translate "The predicate consists of the verb and all the words that complete its meaning. These components work together to express what the subject does or what happens to it." %}

{% translate "Core Predicate Structure" %}

[Subject] + [Verb] + [Object] + [Complement] + [Modifiers]

{% translate "While only the verb is required, most predicates contain additional elements to complete their meaning." %}

{% translate "1. The Verb" %}

{% translate "The essential component that expresses the action or state of being." %}

{% translate "Types of Verbs in Predicates:" %}

  • {% translate "Action verbs: Show physical or mental action" %}

    She writes novels.

  • {% translate "Linking verbs: Connect subject to complement (be, seem, become)" %}

    He is happy.

  • {% translate "Auxiliary verbs: Help form tenses and moods (have, do, will)" %}

    They have been studying.

{% translate "2. Objects" %}

{% translate "Elements that receive or are affected by the verb's action." %}

{% translate "Direct Objects" %}

{% translate "Directly receives the action of the verb (answers 'what?' or 'whom?')" %}

  • She baked a cake.
  • We watched the movie.

{% translate "Indirect Objects" %}

{% translate "Receives the direct object (answers 'to/for whom?')" %}

  • He gave me the book.
  • She sent her friend a letter.

{% translate "Object or Preposition?" %}

{% translate "When the receiver comes after 'to' or 'for', it becomes a prepositional phrase rather than an indirect object:" %}

He gave the book to me.

{% translate "3. Complements" %}

{% translate "Words that complete the meaning of the subject or object." %}

{% translate "Subject Complements" %}

{% translate "Follow linking verbs and rename/describe the subject" %}

  • She is a doctor. (Noun)
  • The soup smells delicious. (Adjective)

{% translate "Object Complements" %}

{% translate "Follow and modify direct objects" %}

  • They elected him president.
  • We painted the door red.

{% translate "4. Modifiers" %}

{% translate "Optional elements that add information about time, place, manner, etc." %}

{% translate "Adverbial Modifiers" %}

  • She runs quickly. (Manner)
  • We'll meet tomorrow. (Time)
  • They waited at the station. (Place)

{% translate "Prepositional Phrases" %}

  • The book on the table is mine.
  • We traveled to Paris.

{% translate "Predicate Patterns" %}

{% translate "Pattern" %} {% translate "Structure" %} {% translate "Example" %}
Intransitive Subject + Verb Birds fly.
Transitive Subject + Verb + Direct Object She reads books.
Ditransitive Subject + Verb + Indirect Object + Direct Object He gave me advice.
Linking Subject + Linking Verb + Complement They are students.
Complex Transitive Subject + Verb + Object + Complement We consider him reliable.

{% translate "Common Errors" %}

  • {% translate "Missing components: Incomplete predicates" %}

    Incorrect: She gave.

    Correct: She gave a presentation.

  • {% translate "Wrong complement type: Using adverb instead of adjective after linking verb" %}

    Incorrect: The food tastes deliciously.

    Correct: The food tastes delicious.

  • {% translate "Misplaced modifiers: Creating confusing relationships" %}

    Unclear: He almost drove his kids to school every day.

    Clear: He drove his kids to school almost every day.
