{% translate "Grammatical Forms That Can Function as Subjects" %}

{% translate "Subjects in English can take various grammatical forms, from simple nouns to complex phrases and clauses, each serving as the focal point of a sentence." %}

{% translate "1. Nouns and Pronouns" %}

[Noun/Pronoun] + [Predicate]

{% translate "Characteristics:" %}

  • {% translate "Most common subject forms" %}
  • {% translate "Pronouns must be in nominative case (I, he, she, we, they)" %}
  • {% translate "Proper nouns (specific names) and common nouns (general categories)" %}
  • {% translate "Can be concrete or abstract" %}

{% translate "Examples:" %}

  • Mary loves music. (proper noun)
  • Happiness is contagious. (abstract noun)
  • They arrived early. (pronoun)
  • Something seems wrong. (indefinite pronoun)

{% translate "2. Noun Phrases" %}

[Determiner] + [Modifiers] + [Noun] + [Predicate]

{% translate "Characteristics:" %}

  • {% translate "Noun plus its modifiers" %}
  • {% translate "Can include articles, adjectives, prepositional phrases" %}
  • {% translate "May contain relative clauses" %}
  • {% translate "Can be simple or complex" %}

{% translate "Examples:" %}

  • The big red ball bounced away. (with adjectives)
  • The book on the shelf fell down. (with prepositional phrase)
  • The man who lives next door is a doctor. (with relative clause)
  • All of my friends from college are coming. (complex phrase)

{% translate "3. Gerunds and Gerund Phrases" %}

[Verb+ing] (+ [Modifiers/Objects]) + [Predicate]

{% translate "Characteristics:" %}

  • {% translate "Verbal nouns ending in -ing" %}
  • {% translate "Can take objects and modifiers" %}
  • {% translate "Always singular in number" %}
  • {% translate "Common for activities and processes" %}

{% translate "Examples:" %}

  • Swimming is great exercise. (simple gerund)
  • Reading books improves vocabulary. (with object)
  • Quickly solving problems is his specialty. (with adverb)
  • Your constantly interrupting is annoying. (with possessive)

{% translate "4. Infinitives and Infinitive Phrases" %}

[to + Verb] (+ [Modifiers/Objects]) + [Predicate]

{% translate "Characteristics:" %}

  • {% translate "Less common than gerund subjects" %}
  • {% translate "Often abstract or philosophical" %}
  • {% translate "Can be modified and take objects" %}
  • {% translate "Often used with 'be' verbs" %}

{% translate "Examples:" %}

  • To err is human. (simple infinitive)
  • To learn a new language takes time. (with object)
  • To completely understand this concept requires study. (with adverb)
  • To be or not to be is the question. (compound)

{% translate "5. Other Subject Types" %}

{% translate "A. Noun Clauses" %}

  • What you said makes sense.
  • That he arrived late surprised everyone.

{% translate "B. Dummy 'It'" %}

  • It is raining.
  • It seems that we're late.

{% translate "C. Existential 'There'" %}

  • There are many reasons.
  • There appears to be a problem.

{% translate "D. Quotations" %}

  • 'I'm tired' was all she said.

{% translate "Special Considerations" %}

{% translate "1. Subject-Verb Agreement" %}

{% translate "Subjects determine verb form (singular/plural):" %}

  • The team is ready. (collective noun - singular)
  • Bread and butter are my favorites. (compound - plural)

{% translate "2. Delayed Subjects" %}

{% translate "When subjects appear after the verb:" %}

  • There is a problem.
  • Here come the children.

{% translate "3. Common Errors" %}

  • {% translate "Case errors with pronouns:" %}

    Wrong: Me and John went.
    Right: John and I went.

  • {% translate "Agreement errors:" %}

    Wrong: The list of items are long.
    Right: The list of items is long.
