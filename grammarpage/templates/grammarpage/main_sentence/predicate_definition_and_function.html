{% translate "Definition and Function of the Predicate" %}

{% translate "The predicate is the part of a sentence that contains the verb and provides information about the subject. It tells what the subject is doing, what is being done to it, or what state of being it is in." %}

{% translate "Core Function" %}

{% translate "The predicate completes the thought initiated by the subject. It must contain at least one verb but can also include:" %}

  • {% translate "Objects (direct and indirect)" %}
  • {% translate "Complements (subject and object complements)" %}
  • {% translate "Adverbial modifiers" %}
  • {% translate "Prepositional phrases" %}
[Subject] + [Verb] + [Optional Elements]

{% translate "Types of Predicates" %}

{% translate "1. Simple Predicate" %}

{% translate "Consists only of the main verb or verb phrase (without any modifiers or objects)." %}

{% translate "Examples:" %}

  • The baby cried.
  • She has been sleeping.
  • They will arrive.

{% translate "2. Complete Predicate" %}

{% translate "Includes the verb plus all words that modify or complete its meaning." %}

{% translate "Examples:" %}

  • The dog barked loudly at the mail carrier.
  • My sister gave me a wonderful present yesterday.
  • We have been studying English grammar for three months.

{% translate "3. Compound Predicate" %}

{% translate "Contains two or more verbs that share the same subject (connected by conjunctions)." %}

{% translate "Examples:" %}

  • She sings and dances beautifully.
  • The children played outside, got dirty, and came home hungry.
  • He will either call or text you later.

{% translate "Predicate Components" %}

{% translate "Component" %} {% translate "Function" %} {% translate "Example" %}
{% translate "Verb" %} {% translate "Core action or state of being" %} She writes novels.
{% translate "Direct Object" %} {% translate "Receives the action of the verb" %} She writes novels.
{% translate "Indirect Object" %} {% translate "Receives the direct object" %} She gave me a book.
{% translate "Subject Complement" %} {% translate "Renames or describes the subject" %} He is a teacher.
{% translate "Adverbial Modifier" %} {% translate "Adds information about time, place, manner" %} They arrived yesterday.

{% translate "Special Cases" %}

  • {% translate "Linking verbs: Connect the subject to its complement (be, seem, become)" %}

    She is happy.

  • {% translate "Verb phrases: Multiple words functioning as a single verb" %}

    They should have been studying.

  • {% translate "Elliptical constructions: Where part of the predicate is implied" %}

    She can swim faster than I (can swim).

  • {% translate "Imperatives: The subject (you) is omitted" %}

    Close the door.

{% translate "Common Mistakes" %}

  • {% translate "Sentence fragments: Missing predicate" %}

    Incorrect: The big red house on the corner.

    Correct: The big red house on the corner belongs to my uncle.

  • {% translate "Subject-verb agreement: Mismatch in number" %}

    Incorrect: The list of items are long.

    Correct: The list of items is long.

  • {% translate "Dangling modifiers: Unclear what the predicate refers to" %}

    Incorrect: Running down the street, the dog was seen.

    Correct: Running down the street, I saw the dog.
