{% translate "Countable/Uncountable Nouns" %}

{% translate "Nouns can be divided into two main categories based on whether they can be counted. This distinction affects the use of articles, determiners, and verb agreement." %}

{% translate "Countable Nouns" %}

{% translate "Countable nouns (or count nouns) are for things we can count using numbers. They have both a singular and a plural form." %}

{% translate "Singular" %} {% translate "Plural" %} {% translate "Sentence Example" %}
{% translate "a book" %} {% translate "two books" %} {% translate "I have a book. I have two books." %}
{% translate "an apple" %} {% translate "three apples" %} {% translate "She ate an apple. She ate three apples." %}
{% translate "a car" %} {% translate "many cars" %} {% translate "We bought a car. We have many cars." %}

{% translate "Key Rules for Countable Nouns" %}

  • {% translate "They can be used with numbers:" %} one apple, five apples.
  • {% translate "They can be used with indefinite articles a/an in the singular form:" %} a dog, an elephant.
  • {% translate "They can be used with quantifiers like many, a few, several." %}

{% translate "Uncountable Nouns" %}

{% translate "Uncountable nouns (or mass nouns) are for things that we cannot count. They are typically used in the singular form and do not have a plural." %}

{% translate "Category" %} {% translate "Example" %} {% translate "Sentence Example" %}
{% translate "Liquids" %} {% translate "water, milk, oil" %} {% translate "I need some water." %}
{% translate "Materials" %} {% translate "wood, paper, gold" %} {% translate "The table is made of wood." %}
{% translate "Abstract Ideas" %} {% translate "information, advice, happiness" %} {% translate "That is useful information." %}
{% translate "Fields of Study" %} {% translate "physics, history, economics" %} {% translate "I am studying economics." %}

{% translate "Key Rules for Uncountable Nouns" %}

  • {% translate "They cannot be used with numbers directly." %}
  • {% translate "They do not take the indefinite articles a/an." %}
  • {% translate "They are used with quantifiers like much, a little, some." %}
  • {% translate "To count them, we use phrases like a piece of, a glass of, a loaf of:" %} a glass of water, a slice of bread.

{% translate "Nouns that can be both" %}

{% translate "Some nouns can be either countable or uncountable, depending on the context and meaning." %}

  • {% translate "Countable:" %} {% translate "There are two hairs on the table." %}
  • {% translate "Uncountable:" %} {% translate "She has long hair." %}
  • {% translate "Countable:" %} {% translate "I want a coffee (a cup of coffee)." %}
  • {% translate "Uncountable:" %} {% translate "I love coffee (the drink in general)." %}
