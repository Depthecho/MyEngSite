{% translate "Demonstratives" %}

{% translate "Demonstratives are words that point to specific things. The four main demonstrative determiners are this, that, these, and those. They tell us whether the noun is singular or plural and whether it is close to or far from the speaker." %}

{% translate "Singular and Plural" %}

{% translate "Use this and that for singular nouns, and these and those for plural nouns." %}

{% translate "Number" %} {% translate "Close to the speaker" %} {% translate "Far from the speaker" %}
{% translate "Singular" %}thisthat
{% translate "Plural" %}thesethose

{% translate "Demonstratives as Determiners vs. Pronouns" %}

{% translate "Demonstratives can function as both determiners and pronouns. The distinction depends on whether they are followed by a noun." %}

{% translate "Function" %} {% translate "Example" %}
Determiner

{% translate "Followed by a noun." %}

 This book is old.
Pronoun

{% translate "Stands alone, replacing a noun." %}

 This is my book.

{% translate "Examples of Usage" %}

  • {% translate "Close and Singular:" %}
    I like this painting right here.
  • {% translate "Far and Singular:" %}
    Who is that person over there?
  • {% translate "Close and Plural:" %}
    Please pass me these papers on the table.
  • {% translate "Far and Plural:" %}
    Those clouds look like a storm is coming.
