{% translate "Concrete vs Abstract Nouns" %}

{% translate "Nouns can be classified based on whether they refer to tangible things that can be perceived with the five senses or intangible concepts. This distinction helps to better understand the nature of a noun." %}

{% translate "Concrete Nouns" %}

{% translate "Concrete nouns refer to things that you can experience with your five senses: sight, sound, smell, touch, or taste. They are physical, tangible objects." %}

{% translate "Sense" %} {% translate "Example" %} {% translate "Sentence" %}
{% translate "Sight" %} book, car, cloud He saw a large cloud in the sky.
{% translate "Sound" %} music, thunder, whisper The music was very loud.
{% translate "Smell" %} perfume, smoke, coffee I love the smell of fresh coffee.
{% translate "Touch" %} stone, blanket, chair She sat on the hard chair.
{% translate "Taste" %} sugar, salt, apple This apple is very sweet.

{% translate "Most common nouns and all proper nouns are concrete nouns." %}

{% translate "Abstract Nouns" %}

{% translate "Abstract nouns refer to intangible things such as ideas, concepts, emotions, qualities, or states of being. They cannot be perceived with the five senses." %}

{% translate "Type" %} {% translate "Example" %} {% translate "Sentence" %}
{% translate "Emotions" %} happiness, anger, fear Happiness is the most important thing.
{% translate "Qualities" %} bravery, honesty, wisdom His bravery saved the day.
{% translate "Concepts" %} freedom, justice, democracy They fought for freedom.
{% translate "States" %} childhood, poverty, rest His childhood was difficult.

{% translate "Some abstract nouns can be formed by adding a suffix to an adjective or verb (e.g., kind -> kindness, decide -> decision)." %}
