{% translate "Subject Complement" %}

{% translate "A subject complement is a noun, pronoun, or adjective that follows a linking verb (e.g., be, become, seem) and renames or describes the subject of the sentence. It does not receive the action of the verb; instead, it provides more information about the subject." %}

{% translate "Types of Subject Complements" %}

{% translate "Predicate Nominative (Noun)" %}

{% translate "A predicate nominative is a noun or noun phrase that renames the subject. It is always connected to the subject by a linking verb." %}

{% translate "Sentence" %} {% translate "Subject" %} {% translate "Linking Verb" %} {% translate "Predicate Nominative" %}
{% translate "John is a doctor." %}{% translate "John" %}{% translate "is" %}{% translate "a doctor" %}
{% translate "She became a great leader." %}{% translate "She" %}{% translate "became" %}{% translate "a great leader" %}

{% translate "Predicate Adjective (Adjective)" %}

{% translate "A predicate adjective is an adjective that describes the subject. It is also connected to the subject by a linking verb." %}

{% translate "Sentence" %} {% translate "Subject" %} {% translate "Linking Verb" %} {% translate "Predicate Adjective" %}
{% translate "The car is red." %}{% translate "The car" %}{% translate "is" %}{% translate "red" %}
{% translate "He seems tired." %}{% translate "He" %}{% translate "seems" %}{% translate "tired" %}
