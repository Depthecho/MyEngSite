{% translate "Zero Plurals" %}

{% translate "Some nouns in English have the same form for both the singular and the plural. These are often called zero plurals or unchanging nouns. The context of the sentence, including determiners like numbers or articles, tells us whether the noun is singular or plural." %}

{% translate "Common Zero Plural Nouns" %}

{% translate "For Animals" %}

{% translate "Many animals, especially those used for hunting or as food, have the same singular and plural form." %}

{% translate "Noun" %} {% translate "Singular Example" %} {% translate "Plural Example" %}
sheepOne sheep is in the field.Ten sheep are in the field.
deerI saw a deer.We saw many deer.
fishHe caught a big fish.He caught a lot of fish.
bisonA single bison crossed the road.A herd of bison roamed the plains.

{% translate "Other Zero Plural Nouns" %}

{% translate "This group includes many nouns related to specific fields, foreign words, and some common nouns." %}

{% translate "Noun" %} {% translate "Singular Example" %} {% translate "Plural Example" %}
seriesThis series is my favorite.These three series are great.
speciesA rare species of bird.Many species are endangered.
aircraftThe aircraft is ready for takeoff.All aircraft were grounded.
dataThe data is important.The data are not reliable.

{% translate "Important Note" %}

{% translate "While fish is a zero plural, the plural fishes can be used to refer to different types or species of fish." %}

  • We caught a lot of fish (quantity).
  • The aquarium has many tropical fishes (different species).
