{% translate "Object of Preposition" %}

{% translate "The object of a preposition is the noun or pronoun that follows a preposition (e.g., in, on, at, with, for) and completes its meaning. The preposition, along with its object and any modifiers, forms a prepositional phrase, which adds detail to a sentence." %}

{% translate "Structure of a Prepositional Phrase" %}

{% translate "A prepositional phrase always has the following structure: Preposition + Noun/Pronoun (Object)." %}

{% translate "Preposition" %} {% translate "Object of Preposition" %} {% translate "Example" %}
{% translate "in" %}{% translate "the box" %}{% translate "The keys are in the box." %}
{% translate "for" %}{% translate "my friend" %}{% translate "I bought a gift for my friend." %}

{% translate "Compound Prepositions" %}

{% translate "Some prepositions consist of more than one word. The noun or pronoun that follows is still considered the object of the preposition." %}
