{% translate "Proper vs Common Nouns" %}

{% translate "Nouns in English can be broadly categorized into two main groups: common nouns and proper nouns. Understanding the difference is essential for correct capitalization and usage." %}

{% translate "Common Nouns" %}

{% translate "Common nouns are general names for a person, place, thing, or idea. They are not capitalized unless they appear at the beginning of a sentence." %}

{% translate "Person" %} {% translate "Place" %} {% translate "Thing" %} {% translate "Idea" %}
{% translate "teacher, boy, mother" %} {% translate "city, country, school" %} {% translate "computer, book, car" %} {% translate "love, freedom, happiness" %}

{% translate "Example sentences:" %}

  • {% translate "The dog is playing in the garden." %}
  • {% translate "She is a good teacher." %}

{% translate "Proper Nouns" %}

{% translate "Proper nouns are specific names for a particular person, place, organization, or thing. They are always capitalized." %}

{% translate "Person" %} {% translate "Place" %} {% translate "Thing / Organization" %} {% translate "Date / Event" %}
{% translate "John, Maria, Mr. Smith" %} {% translate "Paris, Spain, Central High School" %} {% translate "Google, The Empire State Building, Toyota" %} {% translate "Monday, Christmas, World War II" %}

{% translate "Example sentences:" %}

  • {% translate "Maria is visiting Paris." %}
  • {% translate "I bought a new Toyota last Monday." %}

{% translate "Key Rules and Summary" %}

  • {% translate "Capitalization:" %}
    • {% translate "Common Nouns:" %} {% translate "Only capitalized at the beginning of a sentence." %}
    • {% translate "Proper Nouns:" %} {% translate "Always capitalized, regardless of their position in the sentence." %}
  • {% translate "Articles:" %}
    • {% translate "Proper nouns often do not require articles (e.g., I live in London, not in the London)." %}
    • {% translate "However, there are exceptions (e.g., The United States, The Sahara Desert)." %}
