{% translate "Subject of Verb" %}

{% translate "The subject of a verb is the noun or pronoun that performs the action described by the verb. It answers the question 'Who or what is performing the action?' The subject is a fundamental part of a sentence and determines the form of the verb (singular or plural)." %}

{% translate "Identifying the Subject" %}

{% translate "The subject typically appears at the beginning of a sentence before the verb. It can be a single noun or a noun phrase." %}

{% translate "Subject" %} {% translate "Verb" %} {% translate "Sentence" %}
{% translate "The teacher" %}{% translate "explained" %}{% translate "The teacher explained the lesson." %}
{% translate "My best friend" %}{% translate "called" %}{% translate "My best friend called me yesterday." %}

{% translate "Subject in Questions" %}

{% translate "In questions, the subject often comes after a helping verb (e.g., do, does, have, is) but still performs the main action." %}

{% translate "Question" %} {% translate "Subject" %} {% translate "Main Verb" %}

{% translate "Subject-Verb Agreement" %}

{% translate "The subject must agree with the verb in number. A singular subject takes a singular verb, and a plural subject takes a plural verb." %}
