{% translate "Always Plural Nouns" %}

{% translate "Some nouns in English exist only in the plural form and are always used with a plural verb. These nouns often refer to objects that have two parts or items that come in a set." %}

{% translate "Nouns for Two-Part Objects" %}

{% translate "This group includes items that are symmetrical and consist of two identical parts joined together." %}

{% translate "Noun" %} {% translate "Example" %}
scissorsThese scissors are dull.
trousersMy trousers are too short.
glassesHer glasses are new.
binocularsThose binoculars belong to me.

{% translate "Nouns for Clothing and Tools" %}

{% translate "Some articles of clothing and tools also fall into this category." %}

{% translate "Noun" %} {% translate "Example" %}
jeansMy jeans are in the wash.
shortsI need new shorts for the summer.
pajamasHis pajamas were soft.
pliersWhere are the pliers?

{% translate "Other Categories" %}

{% translate "Some other nouns, including abstract concepts and names of places, are always plural." %}

{% translate "Noun" %} {% translate "Example" %}
contentsThe contents of the bag were a mystery.
earningsMy weekly earnings are very low.
remainsThe remains of the old city were found.
outskirtsThey live on the outskirts of town.

{% translate "Using 'a pair of' with These Nouns" %}

{% translate "To refer to a single item from these categories, we use the phrase a pair of. This allows us to use a singular verb." %}

  • I need a new pair of glasses.
  • This pair of scissors is very sharp.
  • I have only one pair of jeans.
