{% translate "Nouns ending in -y/-o/-f" %}

{% translate "Forming the plural of nouns that end in -y, -o, and -f/-fe follows specific rules, often with a number of common exceptions. " %}

{% translate "Nouns ending in -y" %}

{% translate "The rule for nouns ending in -y depends on the letter that comes before it." %}

{% translate "Rule" %} {% translate "Singular" %} {% translate "Plural" %}
{% translate "If a noun ends in a consonant + -y, change -y to -ies." %} city cities
{% translate "If a noun ends in a vowel + -y, simply add -s." %} day days

{% translate "Nouns ending in -o" %}

{% translate "For nouns ending in -o, the plural form can be either -s or -es, and sometimes both." %}

{% translate "Rule" %} {% translate "Singular" %} {% translate "Plural" %}
{% translate "Add -es if the noun ends in a consonant + -o." %} hero heroes
{% translate "Add -s if the noun ends in a vowel + -o." %} radio radios
{% translate "Many nouns ending in -o have an irregular plural form, or both forms are acceptable." %} potato / piano potatoes / pianos

{% translate "Common Exceptions for -o" %}

{% translate "Nouns of foreign origin and some common nouns only add -s." %}

  • photo -> photos
  • piano -> pianos
  • zoo -> zoos

{% translate "Nouns ending in -f or -fe" %}

{% translate "For nouns ending in -f or -fe, the plural is usually formed by changing the ending to -ves." %}

{% translate "Singular" %} {% translate "Plural" %}
leafleaves
wifewives
wolfwolves
knifeknives

{% translate "Common Exceptions for -f" %}

{% translate "Some nouns ending in -f or -fe simply add -s to form the plural." %}

  • belief -> beliefs
  • chief -> chiefs
  • safe -> safes
  • roof -> roofs
