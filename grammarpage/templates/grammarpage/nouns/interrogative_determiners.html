{% translate "Interrogative Determiners" %}

{% translate "Interrogative determiners are words used to ask questions about nouns. They always come before a noun and act as an adjective, modifying the noun that follows. The main interrogative determiners are which, what, and whose." %}

{% translate "The Interrogative Determiners" %}

{% translate "Determiner" %} {% translate "Function" %} {% translate "Example" %}
{% translate "which" %} {% translate "Used when there is a limited or specific choice of items." %} {% translate "Which book do you want to read?" %}
{% translate "what" %} {% translate "Used when there is an unlimited or general choice." %} {% translate "What music do you like?" %}
{% translate "whose" %} {% translate "Used to ask about possession." %} {% translate "Whose car is this?" %}

{% translate "Detailed Explanations" %}

  • {% translate "Using 'Which':" %}

    {% translate "Which implies a selection from a known group. For example, if you are looking at a shelf with three books, you would ask, 'Which book do you want?'" %}

  • {% translate "Using 'What':" %}

    {% translate "What is used for a general or open-ended choice. For example, if you want to know a person's taste in music, you would ask, 'What music do you like?' because the number of possible answers is not limited." %}

  • {% translate "Using 'Whose':" %}

    {% translate "Whose is a possessive interrogative determiner. It always asks about who owns something." %}

    {% translate "Whose jacket is on the chair?" %}
