{% translate "BrE vs AmE Usage" %}

{% translate "One of the most notable differences between British English (BrE) and American English (AmE) is the way they handle verb agreement with collective nouns. While both can be considered correct, there are clear preferences in each dialect." %}

{% translate "British English" %}

{% translate "In British English, collective nouns can take either a singular or a plural verb. The choice depends on whether the speaker is thinking of the group as a single, unified entity or as a collection of individual members." %}

{% translate "Perspective" %} {% translate "Example" %} {% translate "Meaning" %}
{% translate "As a single unit" %} {% translate "The committee is meeting today." %} {% translate "The committee as one body is having a meeting." %}
{% translate "As individual members" %} {% translate "The committee are disagreeing on the proposal." %} {% translate "The members of the committee are in disagreement with each other." %}

{% translate "Key point for BrE" %}

{% translate "The use of a plural verb is very common in British English, especially when talking about groups of people like a team, government, or staff." %}

  • {% translate "The government are debating a new law." %}
  • {% translate "The staff were given a bonus." %}

{% translate "American English" %}

{% translate "In American English, the standard practice is to treat collective nouns as a single unit and use a singular verb, regardless of whether the members are acting together or individually. A plural verb is considered informal and is rarely used." %}

{% translate "Perspective" %} {% translate "Example" %} {% translate "Meaning" %}
{% translate "As a single unit" %} {% translate "The committee is meeting today." %} {% translate "The committee as one body is having a meeting." %}
{% translate "As individual members" %} {% translate "The committee is disagreeing on the proposal." %} {% translate "Even when individuals disagree, the group is still a single unit." %}

{% translate "Key point for AmE" %}

{% translate "To emphasize the individual members, Americans often rephrase the sentence rather than using a plural verb with the collective noun." %}

  • {% translate "Instead of: The team are putting on their shoes." %}
  • {% translate "It is more common to say: The members of the team are putting on their shoes." %}
