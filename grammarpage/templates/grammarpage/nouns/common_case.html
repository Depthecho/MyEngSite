{% translate "Common Case" %}

{% translate "The common case is the base form of a noun in English. It is the form you use when the noun is not showing possession. Most nouns in English are used in the common case, acting as either the subject or the object of a sentence without any change to their spelling." %}

{% translate "Nouns as Subjects" %}

{% translate "When a noun in the common case performs the action in a sentence, it is the subject." %}

{% translate "Subject Noun" %} {% translate "Example Sentence" %}
{% translate "The cat" %}{% translate "The cat slept on the bed." %}
{% translate "My friend" %}{% translate "My friend and I went to the movies." %}
{% translate "The books" %}{% translate "The books are on the shelf." %}

{% translate "Nouns as Objects" %}

{% translate "When a noun in the common case receives the action of the verb, it is the object." %}

{% translate "Object Noun" %} {% translate "Example Sentence" %}
{% translate "the letter" %}{% translate "She wrote the letter." %}
{% translate "a car" %}{% translate "He bought a car yesterday." %}
{% translate "coffee" %}{% translate "I prefer to drink coffee in the morning." %}

{% translate "It's important to note that the form of the noun (cat, friend, letter, etc.) does not change whether it is the subject or the object. This is a key feature of the common case in English." %}
