{% translate "Appositive" %}

{% translate "An appositive is a noun or noun phrase that renames or clarifies another noun right next to it. It adds extra information to a sentence without changing its core meaning. Appositives can be essential to a sentence or simply provide additional, non-essential details." %}

{% translate "Types of Appositives" %}

{% translate "Non-Restrictive Appositive" %}

{% translate "A non-restrictive appositive provides extra, non-essential information. The sentence would still make sense without it. These are usually set off by commas." %}

{% translate "Sentence" %} {% translate "Noun" %} {% translate "Appositive" %}
My brother, a talented musician, plays the piano. My brother a talented musician
London, the capital of England, is a famous city. London the capital of England

{% translate "Restrictive Appositive" %}

{% translate "A restrictive appositive is essential to the meaning of the sentence. It specifies or identifies the noun and cannot be removed without changing the sentence's meaning or creating confusion. Restrictive appositives are not set off by commas." %}

{% translate "Sentence" %} {% translate "Noun" %} {% translate "Appositive" %}
My friend Mary is coming over. My friend Mary
The film 'Titanic' is a classic. The film 'Titanic'

{% translate "Key Difference and Punctuation" %}

{% translate "The main difference between restrictive and non-restrictive appositives lies in the use of commas. If the appositive could be removed without making the sentence unclear, use commas. If the appositive is necessary to identify the noun, do not use commas." %}

  • {% translate "With commas (non-restrictive):" %}
    My dog, Bob, loves to play fetch.
    ({% translate "I only have one dog, and his name is Bob. The name is extra information." %})
  • {% translate "Without commas (restrictive):" %}
    My dog Bob loves to play fetch.
    ({% translate "I have multiple dogs, and I am specifying that I mean the one named Bob. The name is essential information." %})
