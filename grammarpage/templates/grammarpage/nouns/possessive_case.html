{% translate "Possessive Case" %}

{% translate "The possessive case of a noun shows ownership or a close relationship between a noun and another word in a sentence. It is most often formed by adding an apostrophe and an -s (’s) to the end of the noun." %}

{% translate "Rules for Forming the Possessive" %}

{% translate "Rule" %} {% translate "Example" %}
{% translate "For a singular noun, add ’s." %} {% translate "the boy’s toy, a woman’s hat, the dog’s tail" %}
{% translate "For a plural noun that ends in -s, add only an apostrophe (’)." %} {% translate "the students’ projects, the dogs’ bowls, the parents’ car" %}
{% translate "For an irregular plural noun (not ending in -s), add ’s." %} {% translate "the children’s books, the men’s suits, the geese’s feathers" %}
{% translate "For a singular noun ending in -s, you can add either ’s or just an apostrophe (’). Both are generally considered correct." %} {% translate "James’s book OR James’ book" %}

{% translate "Using 'of' for Possession" %}

{% translate "For inanimate objects (things that are not alive), it is more common to use the preposition of to show possession." %}

{% translate "Correct" %} {% translate "Less common" %}
{% translate "the roof of the house" %}{% translate "the house’s roof" %}
{% translate "the cover of the book" %}{% translate "the book’s cover" %}
{% translate "the wheels of the car" %}{% translate "the car’s wheels" %}

{% translate "However, using the ’s possessive with inanimate nouns is common when referring to time, value, or personified objects (e.g., a day’s work, the city’s streets)." %}

{% translate "Compound Nouns and Joint Possession" %}

  • {% translate "Compound Nouns:" %}

    {% translate "Add ’s to the end of the entire compound noun." %}

    {% translate "my mother-in-law’s car" %}
  • {% translate "Joint Possession:" %}

    {% translate "If two or more people own something together, add ’s only to the last person named." %}

    {% translate "John and Mary’s house" %}
