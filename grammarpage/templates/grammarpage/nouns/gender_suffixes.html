{% translate "Gender Suffixes" %}

{% translate "In some cases, the gender of a noun is indicated by a special suffix added to the root word. These suffixes typically turn a masculine noun into a feminine one." %}

{% translate "While these forms were common in the past, modern English tends to use gender-neutral nouns to avoid gender bias, making many of these suffixes less common today." %}
