{% translate "Verb Agreement Rules" %}

{% translate "Subject-verb agreement is the grammatical rule that states the verb in a sentence must agree in number (singular or plural) with its subject. This is a crucial rule for forming correct sentences in English." %}

{% translate "Basic Rules" %}

{% translate "The most fundamental rule is that a singular subject takes a singular verb, and a plural subject takes a plural verb." %}

{% translate "Subject Type" %} {% translate "Verb Type" %} {% translate "Example" %}
{% translate "Singular Subject" %}{% translate "Singular Verb" %}{% translate "The dog chases the cat." %}
{% translate "Plural Subject" %}{% translate "Plural Verb" %}{% translate "The dogs chase the cat." %}

{% translate "Special Cases and Exceptions" %}

{% translate "Collective Nouns" %}

{% translate "Collective nouns (e.g., team, family, government) can be tricky. The verb agreement depends on whether the group is acting as a single unit or as individual members." %}

{% translate "Usage" %} {% translate "Example" %}
{% translate "As a single unit (singular verb)" %} {% translate "The team is playing today." %}
{% translate "As individuals (plural verb)" %} {% translate "The team are putting on their shoes." %}

{% translate "Nouns joined by 'and'" %}

{% translate "When two or more subjects are joined by and, use a plural verb." %}

  • {% translate "Maria and Tom are going to the store." %}

{% translate "However, if the two nouns refer to a single person or idea, use a singular verb." %}

  • {% translate "Macaroni and cheese is my favorite dish." %}

{% translate "Nouns joined by 'or' or 'nor'" %}

{% translate "When subjects are joined by or or nor, the verb agrees with the subject closest to it." %}

  • {% translate "Neither the student nor the teachers know the answer." %}
  • {% translate "Neither the teachers nor the student knows the answer." %}
