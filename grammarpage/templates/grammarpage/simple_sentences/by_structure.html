{% translate "Structural Variations of Simple Sentences" %}

{% translate "While all simple sentences contain one independent clause, their internal structure can vary significantly in complexity." %}

{% translate "1. Simple Subject + Simple Predicate" %}

{% translate "The most basic sentence structure with one subject and one predicate." %}

[Subject] + [Verb]

{% translate "Examples:" %}

  • Birds fly.
  • The sun rose.
  • She laughed.

{% translate "Characteristics:" %}

  • {% translate "Minimal grammatical elements" %}
  • {% translate "Often used in instructions or commands" %}
  • {% translate "Common in children's early speech" %}

{% translate "2. Compound Subject + Simple Predicate" %}

{% translate "Multiple subjects performing the same action." %}

[Subject] + [Subject] + [Verb]

{% translate "Examples:" %}

  • Tom and Jerry argue.
  • The teacher and the students celebrated.
  • Coffee, tea, or milk is available.

{% translate "Usage Notes:" %}

  • {% translate "Use 'and' for additive relationships" %}
  • {% translate "Use 'or' for alternative relationships" %}
  • {% translate "Verb agrees with compound subject (plural usually)" %}

{% translate "3. Simple Subject + Compound Predicate" %}

{% translate "One subject performing multiple actions." %}

[Subject] + [Verb] + [Verb]

{% translate "Examples:" %}

  • She sings and dances.
  • The athlete trained hard but lost.
  • The baby cried, screamed, and kicked.

{% translate "Verb Forms:" %}

  • {% translate "Verbs must share the same subject" %}
  • {% translate "Can use coordinating conjunctions (and, but, or)" %}
  • {% translate "Verbs can be in different tenses if logical" %}

{% translate "4. Compound Subject + Compound Predicate" %}

{% translate "Multiple subjects performing multiple actions." %}

[Subject] + [Subject] + [Verb] + [Verb]

{% translate "Examples:" %}

  • My brother and I study and work in the city.
  • The wind and rain howled and battered the house.
  • The chef and her assistant prepared and served the meal.

{% translate "Complexity Considerations:" %}

  • {% translate "Longest possible simple sentence structure" %}
  • {% translate "Risk of becoming confusing if too extended" %}
  • {% translate "Often better divided into separate sentences" %}

{% translate "Structural Comparison" %}

{% translate "Type" %} {% translate "Subject" %} {% translate "Predicate" %} {% translate "Example" %}
{% translate "Basic" %} {% translate "Simple" %} {% translate "Simple" %} Dogs bark.
{% translate "Compound Subject" %} {% translate "Compound" %} {% translate "Simple" %} Dogs and cats play.
{% translate "Compound Predicate" %} {% translate "Simple" %} {% translate "Compound" %} Dogs bark and growl.
{% translate "Full Compound" %} {% translate "Compound" %} {% translate "Compound" %} Dogs and cats bark and play.

{% translate "Special Cases and Exceptions" %}

  • {% translate "Implied subjects: In commands, the subject (you) is often omitted" %}

    (You) Close the door.

  • {% translate "Inverted structures: Questions and exclamations may reverse normal order" %}

    Are you ready?

  • {% translate "Verb phrases: Single actions expressed with multiple words still count as simple predicates" %}

    She has been working.
