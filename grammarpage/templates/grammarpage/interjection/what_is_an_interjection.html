{% translate "What is an Interjection?" %}

{% translate "An interjection is a word or phrase that expresses a sudden strong feeling, emotion, or reaction. These unique elements of speech stand apart from grammatical sentence structure, serving as emotional outbursts or spontaneous reactions." %}

{% translate "Key Characteristics" %}

  • {% translate "Typically short and emotionally charged" %}
  • {% translate "Can stand alone or appear within sentences" %}
  • {% translate "No grammatical connection to other sentence parts" %}
  • {% translate "Often followed by exclamation marks or commas" %}
  • {% translate "May include sounds that aren't proper words" %}

{% translate "Examples:" %}

  • Wow! That's amazing!
  • Ouch! That hurt!
  • I, uh, don't know what to say.

{% translate "Types of Interjections" %}

{% translate "1. Primary Interjections" %}

{% translate "Words that exist solely as interjections with no other meanings:" %}

  • Wow, Ouch, Oops, Yikes

{% translate "2. Secondary Interjections" %}

{% translate "Regular words that can function as interjections:" %}

  • Great!, Hell, Well, Dear me

{% translate "3. Sound Interjections" %}

{% translate "Non-word sounds that express reactions:" %}

  • Phew, Ugh, Ahem, Brrr

{% translate "Interjections by Emotion" %}

{% translate "Emotion" %} {% translate "Interjections" %} {% translate "Example" %}
{% translate "Surprise" %} Wow, Oh, Whoa Whoa! That was unexpected!
{% translate "Pain" %} Ouch, Ow, Argh Ouch! I stubbed my toe.
{% translate "Joy" %} Yay, Hurray, Woo-hoo Yay! We won!
{% translate "Disgust" %} Ew, Yuck, Ugh Ew! That smells terrible.
{% translate "Hesitation" %} Um, Uh, Er I think, um, we should leave.

{% translate "Punctuation Rules" %}

  • {% translate "Strong interjections: Use exclamation mark" %}

    Help! I need assistance!

  • {% translate "Mild interjections: Use comma" %}

    Well, I'm not sure about that.

  • {% translate "Mid-sentence interjections: Set off with commas" %}

    That was, oh, about two years ago.

  • {% translate "Question interjections: Use question mark" %}

    Huh? What did you say?

{% translate "Common Mistakes" %}

  • {% translate "Overuse: Too many interjections weaken writing" %}

    Poor: Wow! Look at that! Oh! It's so big! Gee!

  • {% translate "Inappropriate punctuation: Using commas for strong emotions" %}

    Incorrect: Ouch, that really hurt.

    Correct: Ouch! That really hurt.

  • {% translate "Confusing with other parts: Some words can be interjections or other parts" %}

    As interjection: Well! That's surprising.

    As adverb: You did that very well.
