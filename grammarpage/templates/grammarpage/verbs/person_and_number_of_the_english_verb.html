{% translate "Person and Number of the English Verb" %}

{% translate "In English, verbs generally do not change their form significantly based on the person and number of the subject, unlike many other languages. The primary exception occurs in the Present Simple tense for the third person singular." %}

{% translate "Conjugation by Person and Number (Present Simple)" %}

{% translate "For most verbs, the form remains the same for all persons and numbers, except for the third person singular (he, she, it), where an -s or -es ending is added to the base form of the verb." %}

  • {% translate "First Person Singular:" %} {% translate "I" %}
    {% translate "Example:" %} I work, I read, I go
  • {% translate "Second Person Singular:" %} {% translate "You" %}
    {% translate "Example:" %} You work, You read, You go
  • {% translate "Third Person Singular:" %} {% translate "He, She, It" %}
    {% translate "Example:" %} He works, She reads, It goes
  • {% translate "First Person Plural:" %} {% translate "We" %}
    {% translate "Example:" %} We work, We read, We go
  • {% translate "Second Person Plural:" %} {% translate "You" %}
    {% translate "Example:" %} You work, You read, You go
  • {% translate "Third Person Plural:" %} {% translate "They" %}
    {% translate "Example:" %} They work, They read, They go

{% translate "Rules for Adding -s / -es in Third Person Singular (Present Simple)" %}

{% translate "The rules for adding -s or -es are similar to those for forming plural nouns:" %}

  • {% translate "Most verbs simply add -s." %}
    {% translate "Examples:" %} play – plays, eat – eats, sleep – sleeps
  • {% translate "Verbs ending in -s, -ss, -sh, -ch, -x, -o, -z (or -zz) add -es." %}
    {% translate "Examples:" %} pass – passes, wash – washes, teach – teaches, fix – fixes, go – goes, buzz – buzzes
  • {% translate "Verbs ending in a consonant + y, change y to i and add -es." %}
    {% translate "Examples:" %} study – studies, try – tries, carry – carries
  • {% translate "Verbs ending in a vowel + y, simply add -s (y does not change)." %}
    {% translate "Examples:" %} buy – buys, say – says, enjoy – enjoys

{% translate "In other tenses (e.g., Past Simple, Future Simple) and with modal verbs, English verbs typically do not change their form based on person or number, which simplifies conjugation considerably compared to many other languages." %}
