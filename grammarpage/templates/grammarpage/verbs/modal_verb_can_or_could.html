{% translate "Can / Could" %}

{% translate "Can and Could are modal auxiliary verbs in English. They are used to express various meanings, including ability, permission, possibility, and requests. Like all modal verbs, they do not change their form based on the subject (e.g., \"I can,\" \"she can\") and are always followed by the base form of the main verb." %}

{% translate "While \"can\" generally refers to the present or future, \"could\" can refer to the past, or be used for more polite or hypothetical situations in the present/future." %}

{% translate "Forms of Can / Could" %}

{% translate "The forms of \"can\" and \"could\" are simple, as they do not conjugate like regular verbs. They always precede the base form of the main verb." %}

{% translate "1. Can (Present / Future)" %}

{% translate "The verb \"can\" is used for present or future abilities, possibilities, or permissions." %}

{% translate "Form" %} {% translate "Structure" %} {% translate "Example" %}
{% translate "Affirmative" %} {% translate "Subject + can + Base Verb" %} {% translate "I" %} {% translate "can swim" %}.
{% translate "She" %} {% translate "can speak" %} {% translate "French." %}
{% translate "Negative" %} {% translate "Subject + cannot / can't + Base Verb" %} {% translate "He" %} {% translate "can't run" %} {% translate "very fast." %}
{% translate "We" %} {% translate "cannot come" %}.
{% translate "Interrogative" %} {% translate "Can + Subject + Base Verb?" %} {% translate "Can you" %} {% translate "help" %} {% translate "me?" %}
{% translate "Can she" %} {% translate "sing" %}?
{% translate "2. Could (Past / Polite / Possibility)" %}

{% translate "The verb \"could\" is used to talk about past abilities, polite requests, or possibilities in the present or future." %}

{% translate "Form" %} {% translate "Structure" %} {% translate "Example" %}
{% translate "Affirmative" %} {% translate "Subject + could + Base Verb" %} {% translate "I" %} {% translate "could play" %} {% translate "the piano when I was young." %}
{% translate "We" %} {% translate "could go" %} {% translate "to the cinema." %}
{% translate "Negative" %} {% translate "Subject + could not / couldn't + Base Verb" %} {% translate "He" %} {% translate "couldn't open" %} {% translate "the jar." %}
{% translate "They" %} {% translate "could not find" %} {% translate "it." %}
{% translate "Interrogative" %} {% translate "Could + Subject + Base Verb?" %} {% translate "Could you please" %} {% translate "close" %} {% translate "the window?" %}
{% translate "Could she" %} {% translate "be" %} {% translate "at home?" %}
{% translate "3. Could have + Past Participle (Unrealized Past Possibility)" %}

{% translate "This form is used to express possibilities that existed in the past but did not happen, or to express regret or criticism about a past action." %}

{% translate "Form" %} {% translate "Structure" %} {% translate "Example" %}
{% translate "Affirmative" %} {% translate "Subject + could have + Past Participle" %} {% translate "He" %} {% translate "could have gone" %} {% translate "to university." %}
{% translate "They" %} {% translate "could have arrived" %} {% translate "earlier." %}
{% translate "Negative" %} {% translate "Subject + could not have / couldn't have + Past Participle" %} {% translate "She" %} {% translate "couldn't have known" %}.
{% translate "I" %} {% translate "could not have done" %} {% translate "it without you." %}

{% translate "Use of Can / Could" %}

{% translate "\"Can\" and \"could\" are versatile modal verbs with distinct and overlapping uses depending on the context." %}

{% translate "1. Expressing Ability" %}
  • {% translate "Can (Present / General Ability):" %} {% translate "Indicates capability or skill in the present." %}
    • {% translate "I" %} {% translate "can run" %} {% translate "a marathon." %} ({% translate "Physical ability" %})
    • {% translate "She" %} {% translate "can solve" %} {% translate "complex math problems." %} ({% translate "Mental ability" %})
    • {% translate "Birds" %} {% translate "can fly" %}. ({% translate "General ability" %})
  • {% translate "Could (Past Ability):" %} {% translate "Refers to an ability someone had in the past." %}
    • {% translate "When I was five, I" %} {% translate "could speak" %} {% translate "a little French." %}
    • {% translate "He" %} {% translate "could lift" %} {% translate "heavy weights in his youth." %}
    • {% translate "Note:" %} {% translate "For a single, specific achievement in the past, \"was/were able to\" is often preferred. Example:" %} {% translate "He" %} {% translate "was able to finish" %} {% translate "the race despite his injury." %}
{% translate "2. Expressing Possibility" %}
  • {% translate "Can (General / Theoretical Possibility):" %} {% translate "Used for things that are generally possible or allowed by rules/nature." %}
    • {% translate "It" %} {% translate "can get" %} {% translate "very cold here in winter." %}
    • {% translate "Accidents" %} {% translate "can happen" %}.
  • {% translate "Could (Present / Future Possibility - less certain):" %} {% translate "Suggests a less certain or more remote possibility than \"can\" or \"may/might\". Also used for suggestions." %}
    • {% translate "It" %} {% translate "could rain" %} {% translate "later." %} ({% translate "It's possible, but not definite" %})
    • {% translate "She" %} {% translate "could be" %} {% translate "at the library right now." %} ({% translate "It's a possibility" %})
    • {% translate "We" %} {% translate "could try" %} {% translate "a different approach." %} ({% translate "Suggesting an option" %})
  • {% translate "Could have + Past Participle (Past Possibility / Unrealized Possibility):" %} {% translate "Describes something that was possible but did not happen. It can also imply regret or criticism." %}
    • {% translate "They" %} {% translate "could have won" %} {% translate "the game if they had played better." %} ({% translate "It was possible, but they didn't" %})
    • {% translate "He" %} {% translate "could have been" %} {% translate "hurt in the accident." %} ({% translate "It was possible for him to be hurt" %})
    • {% translate "You" %} {% translate "could have told" %} {% translate "me!" %} ({% translate "Implies criticism or regret" %})
{% translate "3. Giving / Asking for Permission" %}
  • {% translate "Can (Informal Permission):" %} {% translate "Commonly used for giving and asking for permission in casual situations." %}
    • {% translate "You" %} {% translate "can go" %} {% translate "now." %} ({% translate "Giving permission" %})
    • {% translate "Can I" %} {% translate "use" %} {% translate "your phone?" %} ({% translate "Asking for permission" %})
  • {% translate "Could (Polite Permission / Requests):" %} {% translate "More formal or polite than \"can\" for asking for permission or making requests." %}
    • {% translate "Could I" %} {% translate "borrow" %} {% translate "your pen for a moment?" %}
    • {% translate "Could you please" %} {% translate "pass" %} {% translate "the salt?" %}
{% translate "4. Making Requests / Offers / Suggestions" %}
  • {% translate "Can (Informal Request):" %}
    • {% translate "Can you" %} {% translate "help" %} {% translate "me with this box?" %}
  • {% translate "Could (Polite Request / Offering a suggestion):" %}
    • {% translate "Could you" %} {% translate "open" %} {% translate "the door for me, please?" %}
    • {% translate "We" %} {% translate "could order" %} {% translate "a pizza tonight." %} ({% translate "Suggestion" %})
    • {% translate "You" %} {% translate "could try" %} {% translate "talking to him about it." %} ({% translate "Suggestion/Advice" %})
