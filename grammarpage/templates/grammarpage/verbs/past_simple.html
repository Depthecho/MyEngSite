{% translate "The Past Simple" %}

{% translate "The Past Simple tense (also known as the Simple Past) is used to describe actions or states that started and finished at a specific time in the past. It's one of the most common tenses for narrating past events." %}

{% translate "Formation of the Past Simple" %}

{% translate "The formation of the Past Simple differs for regular and irregular verbs." %}

{% translate "1. Affirmative Sentences" %}
  • {% translate "Regular Verbs:" %} {% translate "Add" %} -ed {% translate "to the base form of the verb." %}
    • work → worked
    • play → played
    • visit → visited
  • {% translate "Irregular Verbs:" %} {% translate "These verbs have unique past simple forms that must be memorized." %}
    • go → went
    • see → saw
    • eat → ate
    • be → was / were
{% translate "Person / Number" %} {% translate "Regular Verb" %} (to work) {% translate "Irregular Verb" %} (to go) {% translate "Verb" %} (to be)
Iworkedwentwas
Youworkedwentwere
He / She / Itworkedwentwas
Weworkedwentwere
Youworkedwentwere
Theyworkedwentwere

{% translate "Spelling Rules for Adding -ed to Regular Verbs:" %}

  • {% translate "Most verbs simply add" %} -ed:
    • open → opened
    • start → started
  • {% translate "Verbs ending in" %} -e, {% translate "just add" %} -d:
    • live → lived
    • love → loved
  • {% translate "Verbs ending in a consonant + y, change" %} y {% translate "to" %} i {% translate "and add" %} -ed:
    • study → studied
    • try → tried
  • {% translate "Verbs ending in a vowel + y, simply add" %} -ed ({% translate "the" %} y {% translate "does not change" %}):
    • play → played
    • enjoy → enjoyed
  • {% translate "Verbs ending in CVC (consonant-vowel-consonant) and stressed on the last syllable (or one-syllable verbs), double the final consonant and add" %} -ed:
    • stop → stopped
    • plan → planned
    • refer → referred

    {% translate "Note:" %} {% translate "In British English, verbs ending in a single -l after a single vowel usually double the l (e.g.," %} travel → travelled). {% translate "In American English, the" %} l {% translate "is not doubled unless the last syllable is stressed." %}

{% translate "2. Negative Sentences" %}

{% translate "To form negative sentences, we use the auxiliary verb 'did not' (didn't) before the base form of the main verb." %}

{% translate "Person / Number" %} {% translate "Full Form" %} {% translate "Contracted Form" %}
Idid not workdidn't work
Youdid not workdidn't work
He / She / Itdid not workdidn't work
Wedid not workdidn't work
Youdid not workdidn't work
Theydid not workdidn't work
  • {% translate "Example:" %} I didn't like the movie.
  • {% translate "Example:" %} She didn't go to the party last night.
  • {% translate "Example:" %} They did not finish their project.

{% translate "Note:" %} {% translate "The verb" %} 'to be' {% translate "does not use" %} 'did not' {% translate "for negatives." %}

  • I was not hungry. / I wasn't hungry. ({% translate "NOT" %} I didn't be hungry.)
  • They were not at home. / They weren't at home.
{% translate "3. Interrogative Sentences (Questions)" %}

{% translate "To form questions, we use the auxiliary verb 'did' before the subject, followed by the base form of the main verb." %}

{% translate "Auxiliary Verb" %} (did) {% translate "Subject" %} {% translate "Main Verb (Base Form)" %}
DidIwork?
Didyouwork?
Didhe / she / itwork?
Didwework?
Didyouwork?
Didtheywork?
  • {% translate "Example:" %} Did you play sports yesterday?
  • {% translate "Example:" %} Did she call you?
  • {% translate "Example:" %} Where did they go last weekend? ({% translate "for Wh-questions" %})

{% translate "Note:" %} {% translate "Again," %} 'to be' {% translate "forms questions differently." %}

  • Was he here? ({% translate "NOT" %} Did he be here?)
  • Were they tired?

{% translate "Uses of the Past Simple" %}

{% translate "The Past Simple is used to express:" %}

  1. {% translate "Completed Actions in the Past at a Specific Time:" %} {% translate "The action began and ended in the past, and the time is usually mentioned or implied." %}
    • I visited London last year.
    • She finished her homework an hour ago.
    • They bought a new car yesterday.
  2. {% translate "A Series of Completed Actions in the Past:" %} {% translate "Used to describe events in chronological order, often in storytelling." %}
    • She woke up, had breakfast, and left for work.
    • He opened the door, walked in, and sat down.
  3. {% translate "Past Habits or Routines:" %} {% translate "Actions that happened regularly in the past but don't happen anymore. Often used with adverbs like" %} always, often, usually, every day ({% translate "in the past" %}).
    • When I was a child, I often played in the park.
    • He smoked for 20 years, but he stopped last year.

    {% translate "Note:" %} Used to {% translate "is also common for past habits: " %} I used to play in the park.

  4. {% translate "Facts or Generalizations About the Past:" %} {% translate "States that were true in the past but may not be true now." %}
    • She was a very talented student.
    • The Roman Empire existed for centuries.
  5. {% translate "In Type 2 Conditional Sentences:" %}
    • If I had more money, I would travel the world.

{% translate "Time Expressions Used with the Past Simple" %}

{% translate "Common time expressions that indicate a specific past time and are typically used with the Past Simple include:" %}

  • yesterday
  • last night / week / month / year
  • ago (two days ago, a week ago)
  • in 1990 ({% translate "or any specific year" %})
  • on Monday ({% translate "or any specific day" %})
  • when I was young ({% translate "or any specific past period" %})

{% translate "The Past Simple is fundamental for talking about history, personal anecdotes, and any events that have concluded in the past." %}
