{% translate "The Participle" %}

{% translate "The participle is a non-finite form of a verb that can function as part of a verb phrase (to form tenses or the passive voice) or as an adjective. English has two main types of participles: the Present Participle and the Past Participle." %}

{% translate "1. The Present Participle (Причастие настоящего времени)" %}

{% translate "The Present Participle is formed by adding -ing to the base form of the verb (e.g., walking, studying, doing). It looks identical to the gerund, but its function in a sentence is different." %}

{% translate "Uses of the Present Participle:" %}
  1. {% translate "To form Continuous (Progressive) Tenses:" %}

    {% translate "Used with the auxiliary verb \"to be\" to indicate ongoing actions." %}

    • {% translate "Example:" %} I am reading a book. ({% translate "Present Continuous" %})
    • {% translate "Example:" %} They were playing football. ({% translate "Past Continuous" %})
    • {% translate "Example:" %} She will be waiting for you. ({% translate "Future Continuous" %})
  2. {% translate "As an Adjective:" %}

    {% translate "Describes a noun or pronoun, often indicating something that is performing the action or causing the feeling." %}

    • {% translate "Example:" %} The running water was cold. ({% translate "Water that is running" %})
    • {% translate "Example:" %} It was an exciting game. ({% translate "The game caused excitement" %})
    • {% translate "Example:" %} A charming smile. ({% translate "A smile that charms" %})
  3. {% translate "After Verbs of Perception (see, hear, feel, watch, notice) + Object:" %}

    {% translate "To indicate an ongoing action that is perceived." %}

    • {% translate "Example:" %} I saw him crossing the street.
    • {% translate "Example:" %} She heard someone singing.
  4. {% translate "To replace a subordinate clause (Participle Clauses):" %}

    {% translate "Used to condense sentences, especially when two actions happen at the same time or one after another, or to give a reason." %}

    • {% translate "Example:" %} Walking down the street, I saw an old friend. ({% translate "While I was walking..." %})
    • {% translate "Example:" %} Feeling tired, she went to bed. ({% translate "Because she felt tired..." %})

{% translate "2. The Past Participle (Причастие прошедшего времени)" %}

{% translate "The Past Participle is the third form of a verb (V3). For regular verbs, it ends in -ed (e.g., worked, finished, played). For irregular verbs, its form varies (e.g., done, seen, written, broken)." %}

{% translate "Uses of the Past Participle:" %}
  1. {% translate "To form Perfect Tenses:" %}

    {% translate "Used with the auxiliary verb \"to have\" to indicate completed actions relative to a point in time." %}

    • {% translate "Example:" %} I have finished my homework. ({% translate "Present Perfect" %})
    • {% translate "Example:" %} She had already left when I arrived. ({% translate "Past Perfect" %})
    • {% translate "Example:" %} They will have completed the project by June. ({% translate "Future Perfect" %})
  2. {% translate "To form the Passive Voice:" %}

    {% translate "Used with the auxiliary verb \"to be\" to show that the subject receives the action." %}

    • {% translate "Example:" %} The car was stolen. ({% translate "Past Simple Passive" %})
    • {% translate "Example:" %} The book is written in English. ({% translate "Present Simple Passive" %})
    • {% translate "Example:" %} The house has been built. ({% translate "Present Perfect Passive" %})
  3. {% translate "As an Adjective:" %}

    {% translate "Describes a noun or pronoun, often indicating something that has undergone the action or is affected by it." %}

    • {% translate "Example:" %} The broken window. ({% translate "The window was broken" %})
    • {% translate "Example:" %} I ate a fried egg. ({% translate "The egg was fried" %})
    • {% translate "Example:" %} He was very tired. ({% translate "He was made tired" %})
  4. {% translate "To replace a subordinate clause (Participle Clauses):" %}

    {% translate "Similar to present participle clauses, used to condense sentences, often indicating a completed action that precedes another, or a cause/reason." %}

    • {% translate "Example:" %} Written in haste, the report contained many errors. ({% translate "Because it was written..." %})
    • {% translate "Example:" %} Having finished her work, she went home. ({% translate "After she had finished..." %})

{% translate "Understanding participles is key to forming complex tenses, passive constructions, and more concise, sophisticated sentences in English." %}
