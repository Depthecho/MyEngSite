{% translate "The Gerund" %}

{% translate "The gerund is a non-finite verb form in English that ends in -ing and functions as a noun. While it looks identical to the present participle, its grammatical role in a sentence is different. It refers to an action or a state, but treats that action/state as a thing or concept." %}

{% translate "Formation of the Gerund" %}

{% translate "The gerund is formed by adding -ing to the base form of the verb. The spelling rules are the same as for the present participle (V-ing form)." %}

  • {% translate "Example:" %} {% translate "Work" %} → working
  • {% translate "Example:" %} {% translate "Read" %} → reading
  • {% translate "Example:" %} {% translate "Swim" %} → swimming ({% translate "doubling consonant" %})
  • {% translate "Example:" %} {% translate "Live" %} → living ({% translate "dropping 'e'" %})

{% translate "Uses of the Gerund" %}

{% translate "As a verb-noun, the gerund can take on various roles in a sentence:" %}

  1. {% translate "As the Subject of a Sentence:" %}

    {% translate "The gerund can perform the action of the main verb." %}

    • {% translate "Example:" %} Swimming {% translate "is good for your health." %}
    • {% translate "Example:" %} Reading {% translate "helps to expand your vocabulary." %}
  2. {% translate "As the Object of a Verb:" %}

    {% translate "Many verbs are typically followed by a gerund. There's no simple rule; these verbs usually need to be learned." %}
    {% translate "Common verbs followed by gerunds:" %} enjoy, finish, mind, stop, avoid, admit, consider, deny, risk, suggest, imagine, practice, postpone, involve, keep, {% translate "etc." %}

    • {% translate "Example:" %} {% translate "He enjoys" %} playing {% translate "football." %}
    • {% translate "Example:" %} {% translate "Have you finished" %} writing {% translate "your report?" %}
    • {% translate "Example:" %} {% translate "I don't mind" %} waiting.
  3. {% translate "As the Object of a Preposition:" %}

    {% translate "A gerund must be used after prepositions." %}

    • {% translate "Example:" %} {% translate "I'm good at" %} drawing.
    • {% translate "Example:" %} {% translate "Thank you for" %} helping {% translate "me." %}
    • {% translate "Example:" %} {% translate "She left without" %} saying {% translate "goodbye." %}
    • {% translate "Example:" %} {% translate "He's interested in" %} learning {% translate "English." %}
  4. {% translate "After Certain Expressions:" %}

    {% translate "Some common expressions are followed by a gerund." %}

    • {% translate "It's no use / It's no good" %}: {% translate "It's no use" %} crying {% translate "over spilled milk." %}
    • {% translate "There's no point in" %}: {% translate "There's no point in" %} arguing {% translate "with him." %}
    • {% translate "Can't stand / Can't help" %}: {% translate "I can't stand" %} waiting. / {% translate "I can't help" %} laughing.
    • {% translate "To be busy" %}: {% translate "I'm busy" %} preparing {% translate "for the exam." %}
    • {% translate "To spend time" %}: {% translate "We spent all day" %} exploring {% translate "the city." %}
    • {% translate "To look forward to" %}: {% translate "I'm looking forward to" %} seeing {% translate "you." %} ({% translate "Note: 'to' here is a preposition, not part of an infinitive" %})
    • {% translate "To be used to / To get used to" %}: {% translate "She's used to" %} waking {% translate "up early." %}
    • {% translate "Worth" %}: {% translate "This book is worth" %} reading.
  5. {% translate "After Phrasal Verbs:" %}

    {% translate "If a phrasal verb's preposition/adverb is followed by another verb, that verb takes the gerund form." %}

    • {% translate "Example:" %} {% translate "Give up" %} smoking!
    • {% translate "Example:" %} {% translate "She carried on" %} talking.

{% translate "Gerund vs. Infinitive" %}

{% translate "A common point of confusion is when to use a gerund versus an infinitive. The choice often depends on the preceding verb or expression. Some verbs can be followed by both with a change in meaning, while others allow both without a significant change." %}

  • {% translate "Verbs followed by Gerund or Infinitive with different meanings:" %}
    • {% translate "Stop" %}: {% translate "He stopped" %} smoking. ({% translate "He quit the habit" %}) vs. {% translate "He stopped" %} to smoke. ({% translate "He paused to have a cigarette" %})
    • {% translate "Remember" %}: {% translate "I remember" %} seeing {% translate "him before." %} ({% translate "Recalling a past event" %}) vs. {% translate "Remember" %} to lock {% translate "the door." %} ({% translate "Don't forget to do it" %})
  • {% translate "Verbs followed by Gerund or Infinitive with similar meanings:" %}
    • {% translate "Start / Begin" %}: {% translate "It started" %} raining. / {% translate "It started" %} to rain.
    • {% translate "Like / Love / Hate / Prefer" %}: {% translate "I love" %} swimming. / {% translate "I love" %} to swim.
