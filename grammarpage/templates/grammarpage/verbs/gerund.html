{% translate "The Gerund" %}

{% translate "The gerund is a non-finite verb form in English that ends in -ing and functions as a noun. While it looks identical to the present participle, its grammatical role in a sentence is different. It refers to an action or a state, but treats that action/state as a thing or concept." %}

{% translate "Formation of the Gerund" %}

{% translate "The gerund is formed by adding -ing to the base form of the verb. The spelling rules are the same as for the present participle (V-ing form)." %}

  • {% translate "Example:" %} Work → working
  • {% translate "Example:" %} Read → reading
  • {% translate "Example:" %} Swim → swimming ({% translate "doubling consonant" %})
  • {% translate "Example:" %} Live → living ({% translate "dropping 'e'" %})

{% translate "Uses of the Gerund" %}

{% translate "As a verb-noun, the gerund can take on various roles in a sentence:" %}

  1. {% translate "As the Subject of a Sentence:" %}

    {% translate "The gerund can perform the action of the main verb." %}

    • {% translate "Example:" %} Swimming is good for your health.
    • {% translate "Example:" %} Reading helps to expand your vocabulary.
  2. {% translate "As the Object of a Verb:" %}

    {% translate "Many verbs are typically followed by a gerund. There's no simple rule; these verbs usually need to be learned." %}
    {% translate "Common verbs followed by gerunds:" %} enjoy, finish, mind, stop, avoid, admit, consider, deny, risk, suggest, imagine, practice, postpone, involve, keep, {% translate "etc." %}

    • {% translate "Example:" %} He enjoys playing football.
    • {% translate "Example:" %} Have you finished writing your report?
    • {% translate "Example:" %} I don't mind waiting.
  3. {% translate "As the Object of a Preposition:" %}

    {% translate "A gerund must be used after prepositions." %}

    • {% translate "Example:" %} I'm good at drawing.
    • {% translate "Example:" %} Thank you for helping me.
    • {% translate "Example:" %} She left without saying goodbye.
    • {% translate "Example:" %} He's interested in learning English.
  4. {% translate "After Certain Expressions:" %}

    {% translate "Some common expressions are followed by a gerund." %}

    • It's no use / It's no good: It's no use crying over spilled milk.
    • There's no point in: There's no point in arguing with him.
    • Can't stand / Can't help: I can't stand waiting. / I can't help laughing.
    • To be busy: I'm busy preparing for the exam.
    • To spend time: We spent all day exploring the city.
    • To look forward to: I'm looking forward to seeing you. ({% translate "Note: 'to' here is a preposition, not part of an infinitive" %})
    • To be used to / To get used to: She's used to waking up early.
    • Worth: This book is worth reading.
  5. {% translate "After Phrasal Verbs:" %}

    {% translate "If a phrasal verb's preposition/adverb is followed by another verb, that verb takes the gerund form." %}

    • {% translate "Example:" %} Give up smoking!
    • {% translate "Example:" %} She carried on talking.

{% translate "Gerund vs. Infinitive" %}

{% translate "A common point of confusion is when to use a gerund versus an infinitive. The choice often depends on the preceding verb or expression. Some verbs can be followed by both with a change in meaning, while others allow both without a significant change." %}

  • {% translate "Verbs followed by Gerund or Infinitive with different meanings:" %}
    • Stop: He stopped smoking. ({% translate "He quit the habit" %}) vs. He stopped to smoke. ({% translate "He paused to have a cigarette" %})
    • Remember: I remember seeing him before. ({% translate "Recalling a past event" %}) vs. Remember to lock the door. ({% translate "Don't forget to do it" %})
  • {% translate "Verbs followed by Gerund or Infinitive with similar meanings:" %}
    • Start / Begin: It started raining. / It started to rain.
    • Like / Love / Hate / Prefer: I love swimming. / I love to swim.
