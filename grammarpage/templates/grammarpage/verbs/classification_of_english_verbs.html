{% translate "Classification of English Verbs" %}

{% translate "English verbs are one of the most complex and diverse parts of speech, essential for constructing any sentence. They can be classified according to various criteria, which helps to better understand their functions and usage." %}

{% translate "By Meaning and Function:" %}

  • {% translate "Notional (Main) Verbs:" %} {% translate "Have independent lexical meaning and denote a specific action, state, process, or relation." %}
    {% translate "Examples:" %} to work, to read, to live, to speak.
  • {% translate "Auxiliary (Helping) Verbs:" %} {% translate "Are used to form complex verb constructions (tenses, voices, moods) and do not have independent lexical meaning." %}
    {% translate "Examples:" %} to be, to have, to do. {% translate "(e.g., in 'I am reading', 'He has finished', 'Do you like?')."%}
  • {% translate "Link-Verbs (Copular Verbs):" %} {% translate "Connect the subject with the predicative part of the predicate (noun, adjective, pronoun, etc.), indicating the subject's state or characteristic. They partially or completely lose their lexical meaning." %}
    {% translate "Examples:" %} to be, to become, to get, to grow, to seem, to appear.
  • {% translate "Modal Verbs:" %} {% translate "Express the speaker's attitude towards an action (possibility, necessity, obligation, permission, etc.), but not the action itself. They do not change form for person or number." %}
    {% translate "Examples:" %} can, may, must, should, will, would.

{% translate "By the Presence of a Direct Object:" %}

  • {% translate "Transitive Verbs:" %} {% translate "Require a direct object, which the action is directed upon." %}
    {% translate "Examples:" %} to give (something), to buy (something), to write (something).
  • {% translate "Intransitive Verbs:" %} {% translate "Do not require a direct object. The action is performed by the subject itself and does not pass to another object." %}
    {% translate "Examples:" %} to sleep, to walk, to arrive.

{% translate "Some verbs can be both transitive and intransitive depending on the context." %}
