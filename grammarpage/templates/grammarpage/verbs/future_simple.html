{% translate "The Future Simple (Will)" %}

{% translate "The Future Simple tense, formed with \"will\" + the base form of the verb, is used to talk about future events that are predictions, spontaneous decisions, promises, or facts. It often expresses a decision made at the moment of speaking." %}

{% translate "Formation of the Future Simple" %}

{% translate "The Future Simple is formed using the auxiliary verb \"will\" and the base form (infinitive without 'to') of the main verb. \"Will\" is used for all subjects." %}

{% translate "1. Affirmative Sentences" %}
{% translate "Person / Number" %} {% translate "Subject" %} {% translate "Auxiliary" %} (will) {% translate "Main Verb" %} ({% translate "Base Form" %}) {% translate "Example" %}
IIwillwork{% translate "I will work." %}
YouYouwillwork{% translate "You will work." %}
He / She / ItHewillwork{% translate "He will work." %}
WeWewillwork{% translate "We will work." %}
YouYouwillwork{% translate "You will work." %}
TheyTheywillwork{% translate "They will work." %}
  • {% translate "Contractions:" %} {% translate "In everyday speech and informal writing, contractions are very common:" %}
    • {% translate "I'll work" %}
    • {% translate "You'll work" %}
    • {% translate "He'll / She'll / It'll work" %}
    • {% translate "We'll work" %}
    • {% translate "They'll work" %}
{% translate "2. Negative Sentences" %}

{% translate "To form negative sentences, we place 'not' after the auxiliary verb \"will\"." %}

{% translate "Person / Number" %} {% translate "Subject" %} {% translate "Auxiliary" %} (will) + not {% translate "Main Verb" %} ({% translate "Base Form" %}) {% translate "Example" %}
IIwill not / won'twork{% translate "I will not work." %} / {% translate "I won't work." %}
YouYouwill not / won'twork{% translate "You will not work." %} / {% translate "You won't work." %}
He / She / ItHewill not / won'twork{% translate "He will not work." %} / {% translate "He won't work." %}
WeWewill not / won'twork{% translate "We will not work." %} / {% translate "We won't work." %}
YouYouwill not / won'twork{% translate "You will not work." %} / {% translate "You won't work." %}
TheyTheywill not / won'twork{% translate "They will not work." %} / {% translate "They won't work." %}
  • {% translate "Example:" %} {% translate "It won't rain tomorrow, I think." %}
  • {% translate "Example:" %} {% translate "I will not tell anyone your secret." %}
{% translate "3. Interrogative Sentences (Questions)" %}

{% translate "To form questions, we invert the subject and the auxiliary verb \"will\"." %}

{% translate "Auxiliary" %} (will) {% translate "Subject" %} {% translate "Main Verb" %} ({% translate "Base Form" %}) {% translate "Example" %}
WillIwork?{% translate "Will I work?" %}
Willyouwork?{% translate "Will you work?" %}
Willhework?{% translate "Will he work?" %}
Willwework?{% translate "Will we work?" %}
Willyouwork?{% translate "Will you work?" %}
Willtheywork?{% translate "Will they work?" %}
  • {% translate "Example:" %} {% translate "Will you help me with my homework?" %}
  • {% translate "Example:" %} {% translate "Where will you go on vacation?" %} ({% translate "for Wh-questions" %})

{% translate "Uses of the Future Simple" %}

{% translate "The Future Simple is used for various purposes related to the future:" %}

  1. {% translate "Predictions / Opinions about the Future (without strong evidence):" %} {% translate "Based on personal opinion, belief, or general knowledge. Often used with verbs like" %} think, believe, expect, be sure, probably.
    • {% translate "I" %} think {% translate "it" %} will rain {% translate "tomorrow." %}
    • {% translate "I" %} believe {% translate "she" %} will pass {% translate "the exam." %}
    • {% translate "He'll" %} probably be {% translate "late." %}
  2. {% translate "Spontaneous Decisions / Decisions Made at the Moment of Speaking:" %} {% translate "For actions decided upon at the time of speaking, not pre-planned." %}
    • {% translate "Oh, the phone is ringing! I" %} 'll answer {% translate "it." %}
    • {% translate "I'm thirsty. I" %} 'll get {% translate "a glass of water." %}
    • {% translate "That's a good idea! I" %} 'll do {% translate "that." %}
  3. {% translate "Promises, Offers, Requests, Threats, Warnings:" %}
    • {% translate "Promises:" %} {% translate "I" %} will call {% translate "you tonight." %} / {% translate "I" %} won't tell {% translate "anyone." %}
    • {% translate "Offers:" %} {% translate "I" %} 'll help {% translate "you with your bags." %}
    • {% translate "Requests:" %} {% translate "Will you please" %} {% translate "close the door?" %}
    • {% translate "Threats:" %} {% translate "If you do that again, I" %} will punish {% translate "you." %}
    • {% translate "Warnings:" %} {% translate "Be careful, or you" %} will fall!
  4. {% translate "Future Facts:" %} {% translate "Things that are certain to happen in the future." %}
    • {% translate "The sun" %} will rise {% translate "at 6:00 AM tomorrow." %}
    • {% translate "Christmas" %} will be {% translate "on a Thursday this year." %}
  5. {% translate "In Type 1 Conditional Sentences (Real Conditionals):" %} {% translate "To describe a likely future result if a present condition is met." %}
    • {% translate "If it rains, we" %} will stay {% translate "at home." %}
    • {% translate "If you study hard, you" %} will pass {% translate "the exam." %}

{% translate "Future Simple vs. \"Be Going To\"" %}

{% translate "While both can refer to the future, they have distinct uses:" %}

  • {% translate "Future Simple (will):" %}
    • {% translate "Spontaneous decisions" %}
    • {% translate "Predictions (often based on opinion, less on evidence)" %}
    • {% translate "Promises, offers, requests, threats, facts" %}
    • {% translate "Less certain or immediate plans" %}
  • {% translate "Be Going To:" %}
    • {% translate "Plans or intentions decided before the moment of speaking." %}
    • {% translate "Predictions based on strong present evidence." %}
      • {% translate "Look at those black clouds. It's" %} going to rain. ({% translate "Clear evidence" %})
  • {% translate "Compare:" %}
    • {% translate "I think I" %} will buy {% translate "a new car." %} ({% translate "A thought, maybe a loose plan." %})
    • {% translate "I" %} am going to buy {% translate "a new car next month." %} ({% translate "A definite plan, already decided." %})

{% translate "The Future Simple is a fundamental tense for communicating a wide range of future scenarios and intentions." %}
