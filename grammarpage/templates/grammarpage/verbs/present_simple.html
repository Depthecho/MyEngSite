{% translate "The Present Simple" %}

{% translate "The Present Simple tense (also known as the Simple Present) is one of the most fundamental and frequently used tenses in English. It's used to describe facts, habits, routines, and scheduled events." %}

{% translate "Formation of the Present Simple" %}

{% translate "The formation of the Present Simple depends on the subject of the sentence." %}

{% translate "1. Affirmative Sentences" %}

{% translate "For most verbs, you use the base form (infinitive without 'to'). However, for the third person singular (he, she, it), you add an '-s' or '-es' to the base form." %}

{% translate "Spelling Rules for Third Person Singular (-s / -es):" %}

  • {% translate "Most verbs simply add" %} -s:
    • play → plays
    • eat → eats
    • sleep → sleeps
  • {% translate "Verbs ending in" %} -s, -ss, -sh, -ch, -x, -o, -z ({% translate "or -zz" %}) {% translate "add" %} -es:
    • pass → passes
    • wash → washes
    • teach → teaches
    • fix → fixes
    • go → goes
    • buzz → buzzes
  • {% translate "Verbs ending in a" %} {% translate "consonant + y" %}, {% translate "change" %} y {% translate "to" %} i {% translate "and add" %} -es:
    • study → studies
    • try → tries
    • carry → carries
  • {% translate "Verbs ending in a" %} {% translate "vowel + y" %}, {% translate "simply add" %} -s ({% translate "the" %} y {% translate "does not change" %}):
    • buy → buys
    • say → says
    • enjoy → enjoys

{% translate "3. Interrogative Sentences (Questions)" %}

{% translate "To form questions, we use the auxiliary verbs 'do' or 'does' before the subject, followed by the base form of the main verb." %}

{% translate "Question Form" %}
Do I work?
Do you work?
Does he/she/it work?
Do we work?
Do you work?
Do they work?
  • {% translate "Example:" %} {% translate "Do you play sports?" %}
  • {% translate "Example:" %} {% translate "Does she like chocolate?" %}
  • {% translate "Example:" %} {% translate "Where do they live?" %} ({% translate "for Wh-questions" %})

{% translate "Note:" %} {% translate "Again, 'to be' and modal verbs form questions differently." %}

  • {% translate "Are you ready?" %} ({% translate "NOT" %} {% translate "Do you be ready?" %})
  • {% translate "Can he come?" %} ({% translate "NOT" %} {% translate "Does he can come?" %})

{% translate "Uses of the Present Simple" %}

{% translate "The Present Simple is used to express:" %}

  1. {% translate "Habitual Actions or Daily Routines:" %} {% translate "Things that happen regularly or repeatedly." %}
    • {% translate "I" %} wake up {% translate "early every day." %}
    • {% translate "She" %} drinks {% translate "coffee in the mornings." %}
    • {% translate "They often" %} visit {% translate "their grandparents." %}
  2. {% translate "Facts and Generalizations:" %} {% translate "Statements that are always true or widely accepted facts." %}
    • {% translate "The sun" %} rises {% translate "in the east." %}
    • {% translate "Water" %} boils {% translate "at 100 degrees Celsius." %}
    • {% translate "Doctors" %} help {% translate "sick people." %}
  3. {% translate "Schedules and Timetables:" %} {% translate "Fixed future events, especially for public transport, cinema, classes, etc." %}
    • {% translate "The train" %} leaves {% translate "at 7 PM." %}
    • {% translate "The movie" %} starts {% translate "at 8:30." %}
    • {% translate "Our class" %} finishes {% translate "at five o'clock." %}
  4. {% translate "Instructions or Directions:" %} {% translate "Used to give commands or explain how to do something." %}
    • {% translate "First, you" %} take {% translate "the flour, then you" %} add {% translate "the milk." %}
    • {% translate "You" %} turn {% translate "right at the corner." %}
  5. {% translate "With Verbs of State (Stative Verbs):" %} {% translate "Verbs that describe states rather than actions (e.g., emotions, possession, senses, opinions). These verbs are generally not used in continuous tenses." %}
    • {% translate "I" %} love {% translate "pizza." %}
    • {% translate "She" %} knows {% translate "the answer." %}
    • {% translate "He" %} has {% translate "a big house." %}
    • {% translate "This soup" %} tastes {% translate "delicious." %}
  6. {% translate "In Zero and First Conditional Sentences:" %}
    • {% translate "Zero Conditional (general truths):" %} {% translate "If you" %} heat {% translate "water, it" %} boils ({% translate "at 100°C" %}).
    • {% translate "First Conditional (future possibility):" %} {% translate "If it" %} rains {% translate "tomorrow, I" %} won't go {% translate "out." %}

{% translate "Additional Rules and Nuances" %}

  • {% translate "Position of Adverbs of Frequency:" %}

    {% translate "Adverbs like always, often, usually, sometimes, rarely, never are commonly used with the Present Simple. Their position depends on the type of verb." %}

    • {% translate "Before the main verb:" %} {% translate "I" %} always {% translate "drink coffee." %}
    • {% translate "After the verb 'to be':" %} {% translate "He" %} is often {% translate "late." %}
  • {% translate "Verbs of State vs. Verbs of Action:" %}

    {% translate "The Present Simple is typically used for Stative Verbs (verbs describing states, not actions), even if the state is happening now. Action Verbs can be used in both simple and continuous forms." %}

    • {% translate "Stative Verb (opinion):" %} {% translate "I" %} think {% translate "he is a good person." %}
    • {% translate "Action Verb (current thought process):" %} {% translate "I am" %} thinking {% translate "about my vacation." %}
  • {% translate "For Dynamic Narrative:" %}

    {% translate "This tense is used in sports commentary and news headlines to make events seem immediate and dramatic." %}

    • {% translate "Sports Commentary:" %} {% translate "The striker" %} takes {% translate "the shot and" %} scores!
    • {% translate "News Headline:" %} {% translate "Prime Minister" %} attends {% translate "summit." %}
