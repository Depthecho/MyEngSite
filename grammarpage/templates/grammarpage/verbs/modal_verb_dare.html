{% translate "Dare" %}

{% translate "The verb Dare is unique in English because it can function both as a main verb (lexical verb) and as a modal verb. This dual nature means its grammatical behavior changes depending on how it's used, affecting its conjugation, negation, and question formation." %}

{% translate "The core meaning of \"dare\" is \"to be brave enough to do something,\" \"to challenge,\" or \"to provoke.\"" %}

{% translate "Forms of Dare" %}

{% translate "1. \"Dare\" as a Main Verb (Lexical Verb)" %}

{% translate "When \"dare\" acts as a main verb, it means \"to challenge,\" \"to have the courage to do something,\" or \"to risk.\" It behaves like a regular verb:" %}

  • {% translate "It changes form for the third person singular present (dares)." %}
  • {% translate "It uses \"do/does/did\" for questions and negatives." %}
  • {% translate "It is followed by a \"to-infinitive.\"" %}
{% translate "Form" %} {% translate "Structure" %} {% translate "Example" %}
{% translate "Affirmative" %} {% translate "Subject + dare(s) + to + Base Verb" %} {% translate "He" %} {% translate "dares to speak" %} {% translate "his mind." %}
{% translate "She" %} {% translate "dares to challenge" %} {% translate "the decision." %}
{% translate "Negative" %} {% translate "Subject + do/does/did not dare + to + Base Verb" %} {% translate "I" %} {% translate "don't dare to ask" %} {% translate "him." %}
{% translate "He" %} {% translate "didn't dare to look" %} {% translate "back." %}
{% translate "Interrogative" %} {% translate "Do/Does/Did + Subject + dare + to + Base Verb?" %} {% translate "Do you dare to jump" %}?
{% translate "Did she dare to contradict" %} {% translate "him?" %}
{% translate "2. \"Dare\" as a Modal Verb" %}

{% translate "When \"dare\" functions as a modal verb, it is typically used in negative or interrogative sentences and means \"to be brave enough.\" It behaves like a true modal verb:" %}

  • {% translate "It does not change form (no \"-s\" for third person singular)." %}
  • {% translate "It does not use \"do/does/did\" for questions and negatives." %}
  • {% translate "It is followed directly by the base form of the main verb (without \"to\")." %}
  • {% translate "This usage is more formal or literary, often found in fixed expressions." %}
{% translate "Form" %} {% translate "Structure" %} {% translate "Example" %}
{% translate "Negative" %} {% translate "Subject + dare not / daren't + Base Verb" %} {% translate "I" %} {% translate "daren't tell" %} {% translate "her the truth." %}
{% translate "He" %} {% translate "dare not speak" %}.
{% translate "Interrogative" %} {% translate "Dare + Subject + Base Verb?" %} {% translate "Dare you challenge" %} {% translate "him?" %}
{% translate "How" %} {% translate "dare he speak" %} {% translate "to me like that?" %}

{% translate "Use of Dare" %}

{% translate "The usage of \"dare\" revolves around the concept of courage, challenge, or provocation, with nuances depending on its main or modal function." %}

{% translate "1. Expressing Courage / Bravery (Main Verb & Modal)" %}

{% translate "Both forms can express having the courage to do something, or the lack thereof." %}

  • {% translate "Main Verb:" %} {% translate "Focus on the act of daring or having the courage." %}
    • {% translate "She" %} {% translate "dares to speak" %} {% translate "her mind in public." %}
    • {% translate "He" %} {% translate "doesn't dare to argue" %} {% translate "with his boss." %}
  • {% translate "Modal Verb:" %} {% translate "More formal or emphatic, particularly in negatives." %}
    • {% translate "I" %} {% translate "daren't look" %} {% translate "down from the cliff." %} ({% translate "I don't have the courage" %})
    • {% translate "He" %} {% translate "dare not confront" %} {% translate "her." %}
{% translate "2. Issuing a Challenge (Main Verb)" %}

{% translate "When \"dare\" means \"to challenge someone to do something,\" it usually functions as a main verb." %}

  • {% translate "I" %} {% translate "dare you to jump" %} {% translate "over that fence." %}
  • {% translate "She" %} {% translate "dared him to tell" %} {% translate "the truth." %}
{% translate "3. Expressing Indignation / Strong Disapproval (Modal \"How dare...\")" %}

{% translate "The fixed phrase \"How dare...\" is a common modal use to express strong indignation or shock at someone's impudence or lack of respect." %}

  • {% translate "How dare you speak" %} {% translate "to me like that!" %}
  • {% translate "How dare she accuse" %} {% translate "me of lying!" %}
{% translate "4. Conditional Sentences (Often Modal)" %}

{% translate "\"Dare\" can be used in conditional sentences to imply courage or the risk involved." %}

  • {% translate "If you" %} {% translate "dare to try" %}, {% translate "you might succeed." %} ({% translate "Main verb" %})
  • {% translate "If he" %} {% translate "dare go" %}, {% translate "he will face consequences." %} ({% translate "Modal, formal" %})
{% translate "5. Fixed Phrases (Modal \"Dare say\")" %}

{% translate "The phrase \"I dare say\" (or \"I daresay\") is a somewhat archaic modal use meaning \"I suppose\" or \"I imagine it is probable.\"" %}

  • {% translate "I" %} {% translate "dare say" %} {% translate "it will rain later." %}
  • {% translate "I" %} {% translate "daresay" %} {% translate "you're right." %}
{% translate "Key Differences in Usage:" %}
  • {% translate "Affirmative:" %} {% translate "\"Dare\" as a main verb is common (e.g., He dares to dream big.). \"Dare\" as a modal is very rare in the affirmative (e.g., *He dare dream big* is incorrect)." %}
  • {% translate "Negatives/Questions:" %} {% translate "Both main verb (\"don't dare to\") and modal (\"daren't\") forms are used, but the modal form is more common in formal or fixed expressions, and often conveys a stronger sense of not having the courage." %}
  • {% translate "Following Verb:" %} {% translate "Main verb \"dare\" uses \"to-infinitive\" (dare to do). Modal \"dare\" uses bare infinitive (dare do)." %}
    • {% translate "You" %} {% translate "don't dare to oppose" %} {% translate "him." %} ({% translate "Main verb" %})
    • {% translate "You" %} {% translate "daren't oppose" %} {% translate "him." %} ({% translate "Modal verb" %})
