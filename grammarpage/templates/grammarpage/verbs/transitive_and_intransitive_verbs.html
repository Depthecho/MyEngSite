{% translate "Transitive and Intransitive Verbs" %}

{% translate "English verbs can be classified into two main categories based on whether they require a direct object to complete their meaning: transitive and intransitive verbs." %}

{% translate "Transitive Verbs (Переходные глаголы)" %}

{% translate "Transitive verbs express an action that passes from the subject to a direct object. This object receives the action of the verb. If a transitive verb is used without a direct object, its meaning often seems incomplete or ambiguous." %}

{% translate "To identify a transitive verb, ask 'what?' or 'whom?' after the verb. If there's an answer, the verb is transitive." %}

  • {% translate "Example:" %} She reads a book. ({% translate "Reads what? – a book" %})
  • {% translate "Example:" %} He wrote a letter. ({% translate "Wrote what? – a letter" %})
  • {% translate "Example:" %} They built a house. ({% translate "Built what? – a house" %})
  • {% translate "Example:" %} I saw John. ({% translate "Saw whom? – John" %})

{% translate "The direct object is typically a noun or pronoun that directly receives the action." %}

{% translate "Intransitive Verbs (Непереходные глаголы)" %}

{% translate "Intransitive verbs express an action that does not pass to an object. The action is complete within itself, performed by the subject without affecting something or someone else directly." %}

{% translate "Intransitive verbs cannot have a direct object. Asking 'what?' or 'whom?' after an intransitive verb will not yield a direct answer about an object receiving the action." %}

  • {% translate "Example:" %} The baby sleeps. ({% translate "Sleeps what? – no answer" %})
  • {% translate "Example:" %} Birds sing. ({% translate "Sing what? – no answer" %})
  • {% translate "Example:" %} He laughed loudly. ({% translate "Laughed what? – no answer" %})
  • {% translate "Example:" %} She arrived late. ({% translate "Arrived what? – no answer" %})

{% translate "Intransitive verbs can be followed by adverbs or prepositional phrases, but these are not direct objects." %}

{% translate "Verbs that can be Both Transitive and Intransitive" %}

{% translate "Some verbs can function as both transitive and intransitive depending on the context in which they are used. The key is whether a direct object is present." %}

  • {% translate "Run:" %}
    • {% translate "Transitive:" %} {% translate "Example:" %} He runs a business. ({% translate "Runs what? – a business" %})
    • {% translate "Intransitive:" %} {% translate "Example:" %} He runs every morning. ({% translate "Runs what? – no answer" %})
  • {% translate "Eat:" %}
    • {% translate "Transitive:" %} {% translate "Example:" %} She eats an apple. ({% translate "Eats what? – an apple" %})
    • {% translate "Intransitive:" %} {% translate "Example:" %} She eats quickly. ({% translate "Eats what? – no answer" %})
  • {% translate "Read:" %}
    • {% translate "Transitive:" %} {% translate "Example:" %} I read a book. ({% translate "Read what? – a book" %})
    • {% translate "Intransitive:" %} {% translate "Example:" %} I read before bed. ({% translate "Read what? – no answer" %})

{% translate "Understanding the distinction between transitive and intransitive verbs helps in constructing grammatically correct sentences and correctly identifying sentence structure." %}
