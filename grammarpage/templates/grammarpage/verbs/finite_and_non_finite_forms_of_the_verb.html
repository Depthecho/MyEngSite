{% translate "Finite and Non-Finite Forms of the Verb" %}

{% translate "English verbs can appear in two main categories of forms: finite verbs and non-finite verbs. Understanding this distinction is crucial for constructing grammatically correct and complex sentences." %}

{% translate "Finite Verbs (Личные формы глагола)" %}

{% translate "Finite verbs are forms of a verb that show tense (e.g., past, present, future), person (e.g., first, second, third), and number (singular or plural) relative to their subject. They can function as the main verb of an independent or dependent clause, forming the predicate of a sentence." %}

{% translate "Key characteristics of finite verbs:" %}

  • {% translate "They agree with their subject in person and number (e.g., He walks vs. They walk)." %}
  • {% translate "They indicate a specific tense (e.g., He walks (Present), He walked (Past))." %}
  • {% translate "They can stand alone as the main verb of a sentence or clause." %}
  • {% translate "They typically form questions and negatives using auxiliary verbs (like \"do,\" \"be,\" \"have\")." %}

{% translate "Examples:" %}

  • {% translate "Example:" %} {% translate "She" %} sings {% translate "beautifully." %} ({% translate "Present Simple, 3rd person singular" %})
  • {% translate "Example:" %} {% translate "They" %} were studying {% translate "when I called." %} ({% translate "Past Continuous, 3rd person plural" %})
  • {% translate "Example:" %} {% translate "He" %} will arrive {% translate "tomorrow." %} ({% translate "Future Simple, 3rd person singular" %})
  • {% translate "Example:" %} {% translate "I" %} have seen {% translate "that movie." %} ({% translate "Present Perfect, 1st person singular" %})

{% translate "Every complete sentence in English must contain at least one finite verb." %}

{% translate "Non-Finite Verbs (Неличные формы глагола)" %}

{% translate "Non-finite verbs do not show tense, person, or number. They cannot function as the main verb of a clause on their own; instead, they act as nouns, adjectives, or adverbs. Non-finite verbs require a finite verb to form a complete sentence." %}

{% translate "The three main types of non-finite verbs are:" %}

  • {% translate "Infinitives:" %} {% translate "The base form of the verb, often preceded by \"to\" (to-infinitive) but sometimes used without \"to\" (bare infinitive)." %}
    • {% translate "As a noun:" %} {% translate "Example:" %} To learn {% translate "is important." %}
    • {% translate "As an adjective:" %} {% translate "Example:" %} {% translate "I have a book" %} to read.
    • {% translate "As an adverb:" %} {% translate "Example:" %} {% translate "He came" %} to help.
    • {% translate "Bare infinitive after modal verbs:" %} {% translate "Example:" %} {% translate "She can" %} swim.
  • {% translate "Gerunds:" %} {% translate "The -ing form of a verb used as a noun." %}
    • {% translate "Example:" %} Swimming {% translate "is my hobby." %} ({% translate "Subject" %})
    • {% translate "Example:" %} {% translate "He enjoys" %} reading. ({% translate "Object" %})
  • {% translate "Participles:" %}
    • {% translate "Present Participle (-ing form):" %} {% translate "Used as part of continuous tenses or as an adjective." %}
      • {% translate "Example:" %} {% translate "They are" %} playing. ({% translate "Part of verb phrase" %})
      • {% translate "Example:" %} {% translate "The" %} running {% translate "water was cold." %} ({% translate "Adjective" %})
    • {% translate "Past Participle (-ed, -en, etc. form):" %} {% translate "Used as part of perfect tenses, in the passive voice, or as an adjective." %}
      • {% translate "Example:" %} {% translate "I have" %} finished {% translate "my work." %} ({% translate "Part of verb phrase" %})
      • {% translate "Example:" %} {% translate "The car was" %} stolen. ({% translate "Part of passive voice" %})
      • {% translate "Example:" %} {% translate "The" %} broken {% translate "glass was sharp." %} ({% translate "Adjective" %})

{% translate "Understanding the distinction between finite and non-finite verbs is fundamental for complex sentence construction and grammatical accuracy in English." %}
