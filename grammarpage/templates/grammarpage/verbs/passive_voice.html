{% translate "The Passive Voice" %}

{% translate "The Passive Voice is a grammatical construction where the subject of the sentence receives the action of the verb, rather than performing it. This contrasts with the active voice, where the subject performs the action. The passive voice focuses attention on the action itself or the recipient of the action, often when the doer is unknown, unimportant, or obvious from the context." %}

{% translate "Formation of the Passive Voice" %}

{% translate "The passive voice is consistently formed using a specific structure: a form of the verb" %} "to be" {% translate "followed by the" %} "past participle" {% translate "of the main verb. Optionally, the doer of the action (the \"agent\") can be included using" %} "by + agent".

{% translate "General Formula:" %}

{% translate "Subject + Form of \"to be\" + Past Participle (+ by + Agent)" %}

  • {% translate "The" %} {% translate "subject" %} {% translate "in a passive sentence is the object of the active sentence." %}
  • {% translate "The" %} {% translate "form of \"to be\"" %} {% translate "changes according to the tense." %}
  • {% translate "The" %} {% translate "past participle" %} {% translate "of the main verb always remains the same for a given verb." %}
  • {% translate "The" %} {% translate "agent" %} ({% translate "the person or thing performing the action in the active voice" %}) {% translate "is often omitted but can be included with \"by\"." %}

{% translate "Passive Voice Across Different Tenses:" %}

{% translate "1. Simple Tenses" %}
  • {% translate "Present Simple Passive:" %} {% translate "is / are + Past Participle" %}
    • {% translate "Active:" %} {% translate "Someone" %} {% translate "writes" %} {% translate "the letter." %}
    • {% translate "Passive:" %} {% translate "The letter" %} {% translate "is written" %}.
    • {% translate "Passive (with agent):" %} {% translate "The letter" %} {% translate "is written by" %} {% translate "John." %}
  • {% translate "Past Simple Passive:" %} {% translate "was / were + Past Participle" %}
    • {% translate "Active:" %} {% translate "Someone" %} {% translate "broke" %} {% translate "the window." %}
    • {% translate "Passive:" %} {% translate "The window" %} {% translate "was broken" %}.
    • {% translate "Passive (with agent):" %} {% translate "The window" %} {% translate "was broken by" %} {% translate "the children." %}

    • {% translate "Passive:" %} {% translate "The package" %} {% translate "will be delivered" %} {% translate "tomorrow." %}

{% translate "Use of the Passive Voice" %}

{% translate "While the active voice is generally preferred for clarity and directness, the passive voice is essential in certain situations to shift focus, emphasize different elements, or when specific information is unavailable or unimportant." %}

{% translate "1. When the Doer of the Action (Agent) is Unknown or Unimportant" %}

{% translate "This is the most common reason to use the passive voice. When we don't know who performed an action, or it's irrelevant to the message, the passive voice is ideal." %}

  • {% translate "Examples:" %}
    • {% translate "The window" %} {% translate "was broken" %}. ({% translate "We don't know who broke it." %})
    • {% translate "My car" %} {% translate "was stolen" %} {% translate "last night." %} ({% translate "The identity of the thief isn't the main point." %})
    • {% translate "English" %} {% translate "is spoken" %} {% translate "all over the world." %} ({% translate "The speakers are too numerous or obvious to mention." %})

{% translate "2. When the Action Itself or the Recipient of the Action is More Important than the Doer" %}

{% translate "In scientific, academic, or formal contexts, the process or outcome is often more significant than who performed it. This leads to frequent use of the passive voice." %}

  • {% translate "Examples:" %}
    • {% translate "Many experiments" %} {% translate "are conducted" %} {% translate "daily in this lab." %} ({% translate "Focus on the experiments, not the scientists." %})
    • {% translate "The new policy" %} {% translate "was announced" %} {% translate "yesterday." %} ({% translate "Focus on the policy, not who announced it." %})
    • {% translate "The theory of relativity" %} {% translate "was developed" %} {% translate "by Albert Einstein." %} ({% translate "While the doer is known, the focus is on the theory's development." %})

{% translate "3. To Maintain Objectivity or Formality" %}

{% translate "The passive voice can create a more impersonal and objective tone, which is often desirable in academic writing, reports, and news articles." %}

  • {% translate "Examples:" %}
    • {% translate "Mistakes" %} {% translate "were made" %}. ({% translate "Avoids blaming a specific person." %})
    • {% translate "Research" %} {% translate "has been conducted" %} {% translate "to prove this hypothesis." %} ({% translate "More formal than \"We have conducted research...\"" %})

{% translate "4. To Vary Sentence Structure and Flow" %}

{% translate "Occasionally, using the passive voice can help improve the flow and readability of a paragraph by avoiding repetitive sentence structures or by placing emphasis on different parts of the sentence." %}

  • {% translate "Example:" %}
    • {% translate "The artist painted a beautiful landscape. Then, a new exhibition" %} {% translate "was opened" %} {% translate "by the gallery." %} ({% translate "Connects the two sentences smoothly by focusing on the exhibition." %})

{% translate "When to Avoid the Passive Voice:" %}

{% translate "While useful, overuse of the passive voice can make writing seem:" %}

  • {% translate "Vague:" %} {% translate "Hiding the agent can lead to ambiguity." %}
  • {% translate "Wordy:" %} {% translate "Passive constructions often use more words than active ones." %}
  • {% translate "Less Direct:" %} {% translate "It can sometimes sound evasive or overly formal where directness is needed." %}
  • {% translate "Weak:" %} {% translate "The active voice is generally more dynamic and engaging." %}

{% translate "Always consider whether the passive voice serves a specific purpose in your writing. If the doer is important and known, or if you want clear, direct communication, the active voice is usually the better choice." %}
