{% translate "Conditional Sentences: Zero and First Types" %}

{% translate "Conditional sentences express possible or impossible situations and their consequences. The zero conditional describes universal truths, while the first conditional deals with real future possibilities." %}

{% translate "Zero Conditional" %}

{% translate "Used for general truths, scientific facts, and situations that are always true." %}

{% translate "If + Present Simple" %}, {% translate "Present Simple" %}
{% translate "Examples:" %}
  • {% translate "If you heat ice, it melts." %}
  • {% translate "If people don't eat, they get hungry." %}
  • {% translate "When you mix blue and yellow, you get green." %}
{% translate "Key Features:" %}
  • {% translate "Both clauses use present simple tense" %}
  • {% translate "Can use 'when' instead of 'if' with no change in meaning" %}
  • {% translate "Often used for scientific facts, general truths, and routines" %}
  • {% translate "The condition always leads to the result (100% certainty)" %}

{% translate "First Conditional" %}

{% translate "Used for real and possible situations in the future, often with predictions, promises, warnings, or threats." %}

{% translate "If + Present Simple" %}, {% translate "will + base verb" %}
{% translate "Examples:" %}
  • {% translate "If it rains tomorrow, we will cancel the picnic." %}
  • {% translate "If you study hard, you will pass the exam." %}
  • {% translate "Unless you hurry, you will miss the bus." %} ({% translate "unless = if not" %})
{% translate "Variations:" %}
  • {% translate "Modal verbs: can, may, might, should instead of will" %}

    {% translate "If you finish early, you can leave." %}

  • {% translate "Imperative: For instructions or suggestions" %}

    {% translate "If you feel sick, call a doctor." %}

  • {% translate "Going to future: For existing plans" %}

    {% translate "If it snows, we're going to build a snowman." %}

{% translate "Comparing Zero and First Conditionals" %}

{% translate "Feature" %} {% translate "Zero Conditional" %} {% translate "First Conditional" %}
{% translate "Usage" %} {% translate "General truths" %} {% translate "Future possibilities" %}
{% translate "Condition Tense" %} {% translate "Present Simple" %} {% translate "Present Simple" %}
{% translate "Result Tense" %} {% translate "Present Simple" %} {% translate "will + verb" %}
{% translate "Certainty" %} {% translate "100% (always true)" %} {% translate "Probable (likely to happen)" %}

{% translate "Common Mistakes" %}

  • {% translate "Using future tense in the if-clause" %}
    • {% translate "Incorrect:" %} {% translate "If it will rain, we will stay home." %}
    • {% translate "Correct:" %} {% translate "If it rains, we will stay home." %}
  • {% translate "Confusing zero and first conditionals" %}
    • {% translate "Incorrect:" %} {% translate "If water reaches 100°C, it will boil." %}
    • {% translate "Correct:" %} {% translate "If water reaches 100°C, it boils." %}
  • {% translate "Misplacing commas" %}
    • {% translate "Incorrect:" %} {% translate "You'll succeed if you work hard." %}
    • {% translate "Correct:" %} {% translate "If you work hard, you'll succeed." %}
