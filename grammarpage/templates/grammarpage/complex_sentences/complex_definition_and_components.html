{% translate "Complex Sentences: Structure and Function" %}

{% translate "A complex sentence contains one independent clause and at least one dependent (subordinate) clause. These sentences allow us to express more nuanced relationships between ideas than simple sentences." %}

{% translate "Key Components" %}

{% translate "Independent Clause" %}

{% translate "A complete thought that can stand alone as a sentence. Contains a subject and predicate." %}

{% translate "She went to work" %}

{% translate "The students passed the exam" %}

{% translate "Dependent Clause" %}

{% translate "Cannot stand alone; begins with a subordinating conjunction or relative pronoun. Contains a subject and verb but doesn't express a complete thought." %}

{% translate "although it was raining" %}

{% translate "who lives next door" %}

{% translate "because she studied hard" %}

{% translate "Types of Dependent Clauses" %}

{% translate "1. Adverbial Clauses" %}

{% translate "Function as adverbs; answer questions like when, where, why, how. Begin with subordinating conjunctions:" %}

{% translate "when, while, after, before, because, since, although, if, unless, whereas" %}

{% translate "When the bell rings, we'll leave." %}

{% translate "She succeeded because she worked hard." %}

{% translate "2. Relative (Adjective) Clauses" %}

{% translate "Function as adjectives; describe nouns. Begin with relative pronouns:" %}

{% translate "who, whom, whose, which, that" %}

{% translate "The book that I borrowed is fascinating." %}

{% translate "The teacher who taught us is retiring." %}

{% translate "3. Noun Clauses" %}

{% translate "Function as nouns; can be subjects, objects, or complements. Begin with:" %}

{% translate "that, whether, if, what, who, why, how" %}

{% translate "What you say matters." %}

{% translate "I wonder whether she'll come." %}

{% translate "Sentence Patterns" %}

  • {% translate "Dependent clause first: Use a comma after the dependent clause" %}

    {% translate "Although it was late, we decided to stay." %}

  • {% translate "Independent clause first: Usually no comma needed" %}

    {% translate "We decided to stay although it was late." %}

  • {% translate "Embedded clause: Relative clauses typically appear in the middle" %}

    {% translate "The car that I bought last week has broken down." %}

{% translate "Common Mistakes" %}

  • {% translate "Fragment error: Using a dependent clause as a complete sentence" %}
    • {% translate "Incorrect:" %} {% translate "Because I was tired." %}
    • {% translate "Correct:" %} {% translate "I went to bed because I was tired." %}
  • {% translate "Comma misuse: Forgetting the comma when the dependent clause comes first" %}
    • {% translate "Incorrect:" %} {% translate "While I was cooking the baby cried." %}
    • {% translate "Correct:" %} {% translate "While I was cooking, the baby cried." %}
  • {% translate "Wrong conjunction: Using coordinating conjunctions (FANBOYS) to begin dependent clauses" %}
    • {% translate "Incorrect:" %} {% translate "And I was hungry, I made a sandwich." %}
    • {% translate "Correct:" %} {% translate "Because I was hungry, I made a sandwich." %}

{% translate "Why Use Complex Sentences?" %}

  • {% translate "Show relationships between ideas (cause/effect, time, contrast)" %}
  • {% translate "Add variety to your writing style" %}
  • {% translate "Express more sophisticated thoughts" %}
  • {% translate "Combine information efficiently" %}
  • {% translate "Create emphasis by positioning key information" %}
