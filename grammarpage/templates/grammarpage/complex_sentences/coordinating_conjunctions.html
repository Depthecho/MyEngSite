{% translate "Coordinating Conjunctions (FANBOYS)" %}

{% translate "Coordinating conjunctions (FANBOYS) are used to connect words, phrases, or independent clauses of equal grammatical importance in a sentence. They are essential for creating compound sentences." %}

{% translate "The FANBOYS Acronym" %}

F {% translate "For" %}
A {% translate "And" %}
N {% translate "Nor" %}
B {% translate "But" %}
O {% translate "Or" %}
Y {% translate "Yet" %}
S {% translate "So" %}

{% translate "Usage Rules" %}

  • {% translate "Use a comma before the conjunction when connecting two independent clauses" %}
    • {% translate "She wanted to go to the party, but she had too much work." %}
  • {% translate "No comma is needed when connecting words or phrases" %}
    • {% translate "I like tea and coffee." %}
  • {% translate "Each conjunction expresses a specific relationship between ideas:" %}
    • {% translate "For" %}: {% translate "indicates reason or cause (similar to 'because')" %}
    • {% translate "And" %}: {% translate "adds information" %}
    • {% translate "Nor" %}: {% translate "presents a negative alternative" %}
    • {% translate "But" %}: {% translate "shows contrast" %}
    • {% translate "Or" %}: {% translate "presents alternatives" %}
    • {% translate "Yet" %}: {% translate "shows unexpected contrast" %}
    • {% translate "So" %}: {% translate "indicates result or consequence" %}

{% translate "Detailed Usage with Examples" %}

{% translate "For" %}

{% translate "Indicates reason or cause (similar to 'because'). More formal and less common in spoken English." %}

{% translate "I stayed home, for I was feeling ill." %}

{% translate "And" %}

{% translate "Adds information or connects similar ideas." %}

{% translate "She opened the door and walked inside." %}

{% translate "He likes swimming, and she prefers running." %}

{% translate "Nor" %}

{% translate "Connects two negative alternatives. Note that verb comes before subject after 'nor'." %}

{% translate "He didn't call, nor did he send a message." %}

{% translate "I don't like coffee, nor do I enjoy tea." %}

{% translate "But" %}

{% translate "Shows contrast between two different ideas." %}

{% translate "I wanted to go to the beach, but it started raining." %}

{% translate "Or" %}

{% translate "Presents alternatives or possibilities." %}

{% translate "We can go by train, or we can drive." %}

{% translate "Yet" %}

{% translate "Shows unexpected contrast (similar to 'but')." %}

{% translate "She studied hard, yet she failed the exam." %}

{% translate "So" %}

{% translate "Indicates result or consequence." %}

{% translate "It was getting late, so we decided to leave." %}

{% translate "Common Mistakes" %}

  • {% translate "Comma splice: Using just a comma without a conjunction" %}
    • {% translate "Incorrect:" %} {% translate "I love reading, it relaxes me." %}
    • {% translate "Correct:" %} {% translate "I love reading, for it relaxes me." %}
  • {% translate "Missing comma: Forgetting the comma before the conjunction" %}
    • {% translate "Incorrect:" %} {% translate "I wanted to go but I was too tired." %}
    • {% translate "Correct:" %} {% translate "I wanted to go, but I was too tired." %}
  • {% translate "Overuse of 'and': Creating run-on sentences" %}
    • {% translate "Poor style:" %} {% translate "I woke up and I brushed my teeth and I had breakfast and I went to work." %}
    • {% translate "Better:" %} {% translate "I woke up, brushed my teeth, had breakfast, and went to work." %}

{% translate "Practice Tips" %}

  • {% translate "Try rewriting simple sentences using different FANBOYS conjunctions" %}
  • {% translate "When proofreading, check for comma splices and add appropriate conjunctions" %}
  • {% translate "Vary your conjunctions to express different relationships between ideas" %}
  • {% translate "Read aloud to check if the conjunctions create natural-sounding connections" %}
