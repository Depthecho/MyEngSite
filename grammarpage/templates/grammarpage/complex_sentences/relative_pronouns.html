{% translate "Relative Pronouns: Connecting Clauses" %}

{% translate "Relative pronouns introduce relative clauses that describe nouns in the main clause, adding essential or extra information about people, things, or places." %}

{% translate "Common Relative Pronouns" %}

{% translate "Pronoun" %} {% translate "Usage" %} {% translate "Example" %}
who {% translate "Subject for people" %} The woman who called is my boss.
whom {% translate "Object for people (formal)" %} The man whom we met was friendly.
whose {% translate "Possession for people/things" %} The student whose book was lost.
which {% translate "Subject/object for things" %} The car, which is red, belongs to John.
that {% translate "Subject/object for people/things" %} The book that I read was fascinating.
where {% translate "Places" %} The house where I grew up.
when {% translate "Times" %} The day when we met.

{% translate "Types of Relative Clauses" %}

{% translate "Defining (Restrictive) Clauses" %}

{% translate "Essential information that defines which person/thing we mean. No commas used." %}

The book that you lent me was excellent.

People who work hard succeed.

{% translate "Non-defining (Non-restrictive) Clauses" %}

{% translate "Extra information about a specific person/thing. Use commas." %}

My brother, who lives in Paris, is visiting us.

The Eiffel Tower, which was built in 1889, is iconic.

{% translate "Note: Cannot use 'that' in non-defining clauses." %}

{% translate "Special Cases" %}

  • {% translate "Omitting relative pronouns: Can omit when it's the object of the clause" %}

    The movie (that) we saw was great.

  • {% translate "Prepositions with relative pronouns: Formal vs. informal usage" %}

    Formal: The man to whom I spoke

    Informal: The man (who) I spoke to

  • {% translate "'What' as relative pronoun: Means 'the thing(s) that'" %}

    I'll show you what I found.

{% translate "Common Mistakes" %}

  • {% translate "Using 'which' instead of 'that' in defining clauses" %}
    • Less natural: The book which I read was interesting.
    • Better: The book that I read was interesting.
  • {% translate "Omitting pronouns when they're subjects" %}
    • Incorrect: The man lives next door is a doctor.
    • Correct: The man who lives next door is a doctor.
  • {% translate "Misusing commas in defining clauses" %}
    • Incorrect: The car, that was parked outside, was stolen.
    • Correct: The car that was parked outside was stolen.
