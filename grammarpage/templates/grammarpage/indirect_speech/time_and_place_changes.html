{% translate "Time and Place Expression Changes in Reported Speech" %}

{% translate "When converting direct speech to indirect speech, expressions of time and place must often be adjusted to maintain the correct temporal and spatial relationships in the new context." %}

{% translate "1. Time Expression Transformations" %}

{% translate "Time references typically shift according to when the reporting occurs relative to the original statement:" %}

{% translate "Direct Speech" %} {% translate "Reported Speech" %} {% translate "Example" %}
today that day "I'll do it today" → She said she'd do it that day
yesterday the day before/the previous day "I saw her yesterday" → He said he'd seen her the day before
tomorrow the next day/the following day "I'll call tomorrow" → She said she'd call the following day
now then "I'm busy now" → He said he was busy then
last week the week before/the previous week "We met last week" → They said they'd met the previous week
next month the following month "It opens next month" → She said it would open the following month

{% translate "2. Place Expression Transformations" %}

{% translate "References to location shift according to the reporter's perspective:" %}

{% translate "Direct Speech" %} {% translate "Reported Speech" %} {% translate "Example" %}
here there "Put it here" → He told me to put it there
this place that place "I love this place" → She said she loved that place
over here over there "Come over here" → She told me to come over there

{% translate "3. Demonstrative Pronoun Changes" %}

{% translate "Demonstratives typically shift to maintain proper reference in the new context:" %}

{% translate "Direct Speech" %} {% translate "Reported Speech" %} {% translate "Example" %}
this that "I want this book" → He said he wanted that book
these those "Bring these files" → She told me to bring those files

{% translate "4. Special Cases and Exceptions" %}

{% translate "When Reporting Immediately" %}

  • "I'm leaving now" (said just now) → She says she's leaving now

{% translate "When Location Hasn't Changed" %}

  • "Put it here" (same location) → He told me to put it here

{% translate "With Universal Time References" %}

  • "The sun rises in the east" → He said the sun rises in the east (no change)
{% translate "Complete Transformation Reference" %}
{% translate "Category" %} {% translate "Direct Speech" %} {% translate "Reported Speech" %}
{% translate "Days" %} today, yesterday, tomorrow that day, the day before, the next day
{% translate "Weeks/Months" %} last week, next month the previous week, the following month
{% translate "General Time" %} now, ago, tonight then, before, that night
{% translate "Place" %} here, this place there, that place
{% translate "Demonstratives" %} this, these that, those

{% translate "Practical Tips" %}

  • {% translate "Consider whether the time/place reference would still make sense from the reporter's perspective" %}
  • {% translate "When reporting in the same time/place context, changes may not be necessary" %}
  • {% translate "Universal truths and permanent situations don't require time reference changes" %}
  • {% translate "Practice with real conversations to develop intuition for these transformations" %}
