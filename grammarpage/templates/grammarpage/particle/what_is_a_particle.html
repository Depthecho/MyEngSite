{% translate "What is a Particle?" %}

{% translate "A particle is a small, uninflected word that serves important grammatical functions but doesn't fit neatly into traditional parts of speech categories. These versatile words modify meaning, indicate relationships, or add nuance to sentences." %}

{% translate "Key Characteristics" %}

  • {% translate "Uninflected:" %} {% translate "Doesn't change form (no conjugation or declension)" %}
  • {% translate "Grammatical function:" %} {% translate "Modifies or connects other words" %}
  • {% translate "Small size:" %} {% translate "Typically one or two syllables" %}
  • {% translate "Essential meaning:" %} {% translate "Often crucial to sentence structure" %}
  • {% translate "Multi-functional:" %} {% translate "Can serve different purposes in different contexts" %}

{% translate "Common Types of Particles" %}

{% translate "1. Infinitive Marker" %}

{% translate "The word to used before base verbs:" %}

  • I want to eat.
  • She likes to swim.

{% translate "2. Negative Particle" %}

{% translate "The word not used for negation:" %}

  • I do not know.
  • He is not coming.

{% translate "3. Phrasal Verb Particles" %}

{% translate "Words that combine with verbs to create new meanings:" %}

  • She gave up smoking. (up)
  • Please turn on the light. (on)

{% translate "4. Discourse Particles" %}

{% translate "Words that organize conversation or add emphasis:" %}

  • Well, I'm not sure.
  • Oh, I see now!

{% translate "Why Particles Matter" %}

  • {% translate "Essential for phrasal verbs:" %} {% translate "Often change a verb's meaning completely" %}

    look (see) vs. look up (research)

  • {% translate "Create infinitives:" %} {% translate "Necessary for many verb constructions" %}

    I need to go.

  • {% translate "Form negations:" %} {% translate "Fundamental for negative statements" %}

    They are not here.

  • {% translate "Add nuance:" %} {% translate "Convey subtle shades of meaning" %}

    I put up with the noise.
