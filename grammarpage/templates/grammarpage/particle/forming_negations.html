{% translate "Forming Negations" %}

{% translate "The negative particle not is essential for creating negative statements in English, working in conjunction with auxiliary verbs to reverse the meaning of sentences." %}

{% translate "Basic Negation Structure" %}

[Auxiliary] + not + [Main Verb]

{% translate "Key Rules:" %}

  • {% translate "Always follows an auxiliary/modal verb" %}
  • {% translate "Precedes the main verb in the construction" %}
  • {% translate "Can be contracted with the auxiliary verb in informal contexts" %}
  • {% translate "With simple present/past, requires the dummy auxiliary 'do'" %}

{% translate "Common Negative Constructions" %}

{% translate "1. With 'Be' Verbs" %}

  • I am not ready.
  • She is not coming.
  • They were not happy.

{% translate "2. With Other Auxiliaries" %}

  • I have not seen it.
  • We should not go.
  • He can not swim.

{% translate "3. With Main Verbs (using 'do')" %}

  • I do not like coffee.
  • She did not call.

{% translate "Negative Contractions" %}

{% translate "Auxiliary Verb" %} {% translate "Full Form" %} {% translate "Contracted Form" %}
am am not aren't (in questions)
is is not isn't
are are not aren't
do do not don't
does does not doesn't
have have not haven't
will will not won't

{% translate "Special Cases and Exceptions" %}

  • {% translate "Negative imperatives: Use 'do not' or 'don't'" %}

    Don't touch that!

  • {% translate "Non-auxiliary 'have': Requires 'do' support" %}

    I do not have a car.

  • {% translate "Negative infinitives: 'not' comes before 'to'" %}

    I decided not to go.

  • {% translate "Negative questions: Inversion with contraction" %}

    Didn't you see it?

{% translate "Common Mistakes" %}

  • {% translate "Omitting auxiliary 'do':" %}

    Incorrect: I not like it.

    Correct: I do not like it.

  • {% translate "Wrong contraction form:" %}

    Incorrect: I amn't ready.

    Correct: I'm not ready.

  • {% translate "Double negatives:" %}

    Incorrect: I don't know nothing.

    Correct: I don't know anything.
