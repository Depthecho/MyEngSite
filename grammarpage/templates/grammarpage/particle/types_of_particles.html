{% translate "Types of Particles" %}

{% translate "Particles serve various grammatical functions in English. Understanding these types helps master sentence structure and meaning." %}

{% translate "1. Infinitive Particle (to)" %}

{% translate "Marks the infinitive form of verbs, creating the 'to + verb' construction:" %}

{% translate "Characteristics:" %}

  • {% translate "Always precedes the base form of a verb" %}
  • {% translate "Doesn't carry meaning on its own" %}
  • {% translate "Essential for many verb constructions" %}

{% translate "Examples:" %}

  • I want to eat.
  • She needs to study.
  • It's nice to meet you.

{% translate "Special Cases:" %}

  • {% translate "Bare infinitives (without 'to') after modal verbs:" %}

    I can swim.

  • {% translate "Split infinitives (adverbs between 'to' and verb):" %}

    I need to quickly finish.

{% translate "2. Negative Particle (not)" %}

{% translate "Creates negative statements by negating verbs or other elements:" %}

{% translate "Usage Patterns:" %}

  • {% translate "After auxiliary verbs in standard negation" %}

    She does not (doesn't) know.

  • {% translate "With non-auxiliary verbs (requires 'do' support)" %}

    I do not (don't) understand.

  • {% translate "Contracted forms common in speech" %}

    They aren't (are not) coming.

{% translate "Positioning Rules:" %}

  • {% translate "Before main verbs with auxiliaries" %}

    You should not go.

  • {% translate "After 'to be' when it's the main verb" %}

    He isn't happy.

{% translate "3. Phrasal Verb Particles" %}

{% translate "Combine with verbs to create new meanings, often idiomatic:" %}

{% translate "Common Particles:" %}

  • up: give up, clean up
  • out: find out, work out
  • off: put off, show off
  • on: put on, try on

{% translate "Types of Phrasal Verbs:" %}

{% translate "Type" %} {% translate "Example" %} {% translate "Note" %}
{% translate "Intransitive" %} show up {% translate "No direct object" %}
{% translate "Transitive separable" %} turn off the light {% translate "Object can separate verb and particle" %}
{% translate "Transitive inseparable" %} look after the child {% translate "Particle must stay with verb" %}

{% translate "4. Discourse Particles" %}

{% translate "Organize conversation flow and express attitudes:" %}

{% translate "Common Functions:" %}

  • {% translate "Hesitation markers: um, uh, er" %}

    "I was thinking, uh, maybe we should leave."

  • {% translate "Response signals: yeah, okay, right" %}

    "Right, I understand now."

  • {% translate "Emphasis markers: just, even, only" %}

    "I just don't know!"

  • {% translate "Conversation managers: well, so, anyway" %}

    "Well, what do you think?"

{% translate "Usage Notes:" %}

  • {% translate "More common in spoken than written English" %}
  • {% translate "Often considered informal" %}
  • {% translate "Vary significantly by dialect and region" %}
  • {% translate "Help maintain conversational rhythm" %}
