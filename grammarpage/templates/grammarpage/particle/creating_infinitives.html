{% translate "Creating Infinitives" %}

{% translate "The particle to serves as the infinitive marker in English, creating the base form of verbs used in various grammatical constructions." %}

{% translate "Infinitive Structure" %}

to + [Base Verb] = [Infinitive]

{% translate "Key Characteristics:" %}

  • {% translate "Unconjugated:" %} {% translate "Doesn't change for tense or subject" %}
  • {% translate "Non-finite:" %} {% translate "Doesn't indicate time by itself" %}
  • {% translate "Versatile:" %} {% translate "Can function as noun, adjective, or adverb" %}

{% translate "Primary Uses of Infinitives" %}

{% translate "1. After Certain Verbs" %}

{% translate "Many verbs require an infinitive complement:" %}

  • I want to learn.
  • She decided to stay.
  • We hope to see you soon.

{% translate "2. After Adjectives" %}

{% translate "Expressing purpose or reaction:" %}

  • It's important to practice.
  • I'm happy to help.

{% translate "3. Expressing Purpose" %}

{% translate "Answering 'why' questions:" %}

  • I came to study.
  • This tool is used to cut wood.

{% translate "Types of Infinitives" %}

{% translate "Type" %} {% translate "Structure" %} {% translate "Example" %}
{% translate "Full Infinitive" %} to + verb to read
{% translate "Bare Infinitive" %} Base verb alone Let me help
{% translate "Continuous Infinitive" %} to be + -ing to be working
{% translate "Perfect Infinitive" %} to have + past participle to have seen

{% translate "Special Cases and Exceptions" %}

  • {% translate "Bare infinitives:" %} {% translate "Used after modal verbs and some special verbs" %}

    I can swim. (Not "can to swim")

  • {% translate "Split infinitives:" %} {% translate "Adverbs between 'to' and the verb" %}

    I need to quickly finish.

  • {% translate "'To' as preposition:" %} {% translate "Distinguishing from infinitive marker" %}

    I look forward to seeing you. (Gerund after preposition)

{% translate "Common Mistakes" %}

  • {% translate "Adding 'to' after modal verbs:" %}

    Incorrect: I must to go.

    Correct: I must go.

  • {% translate "Confusing with gerunds:" %}

    Incorrect: I enjoy to swim.

    Correct: I enjoy swimming.

  • {% translate "Omitting 'to' when required:" %}

    Incorrect: I want go.

    Correct: I want to go.
