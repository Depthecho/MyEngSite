{% translate "Comprehensive Guide to Conjunction Types" %}

{% translate "Understanding the different types of conjunctions is essential for constructing clear, complex sentences in English. Each type serves specific grammatical functions and follows distinct rules." %}

{% translate "1. Coordinating Conjunctions (FANBOYS)" %}

{% translate "These conjunctions connect elements of equal grammatical rank: words with words, phrases with phrases, or independent clauses with independent clauses." %}

{% translate "The FANBOYS Acronym:" %}

  • F - for (indicates reason or cause)
  • A - and (adds information)
  • N - nor (presents negative alternative)
  • B - but (shows contrast)
  • O - or (presents alternative)
  • Y - yet (shows unexpected contrast)
  • S - so (indicates result)

{% translate "Usage Examples:" %}

{% translate "Function" %} {% translate "Example" %}
{% translate "Joining words" %} tea and coffee
{% translate "Joining phrases" %} in the morning or in the evening
{% translate "Joining clauses" %} I wanted to go, but I was too tired.

{% translate "Punctuation Rules:" %}

  • {% translate "Use a comma before the conjunction when joining two independent clauses" %}
  • {% translate "No comma needed when joining words or short phrases" %}
  • {% translate "With 'nor', the second clause typically uses inverted word order" %}

{% translate "2. Subordinating Conjunctions" %}

{% translate "These conjunctions introduce dependent clauses and show their relationship to the main clause (time, cause, condition, contrast, etc.)." %}

{% translate "Common Subordinating Conjunctions:" %}

{% translate "Relationship" %} {% translate "Conjunctions" %}
{% translate "Time" %} when, before, after, while, as, until, since
{% translate "Cause/Reason" %} because, since, as
{% translate "Condition" %} if, unless, provided that
{% translate "Contrast" %} although, though, even though, whereas
{% translate "Purpose" %} so that, in order that

{% translate "Sentence Patterns:" %}

  • {% translate "Main clause first: I'll call you when I arrive." %}
  • {% translate "Subordinate clause first: Although it was late, we continued working." %}

{% translate "Punctuation Note:" %}

{% translate "When the subordinate clause comes first, use a comma after it. When it follows the main clause, no comma is needed." %}

{% translate "3. Correlative Conjunctions" %}

{% translate "These are conjunction pairs that work together to connect equivalent sentence elements, requiring parallel structure." %}

{% translate "Common Correlative Pairs:" %}

  • both...and
  • either...or
  • neither...nor
  • not only...but also
  • whether...or
  • as...as

{% translate "Examples with Parallel Structure:" %}

  • Both my sister and my brother are doctors.
  • You must either study hard or fail the exam.
  • She is not only intelligent but also kind.

{% translate "Important Rules:" %}

  • {% translate "Maintain parallel grammatical structure between the connected elements" %}
  • {% translate "With 'either/or' and 'neither/nor', the verb agrees with the closer subject" %}
  • {% translate "Place the correlative conjunctions immediately before the elements being connected" %}

{% translate "Comparison of Conjunction Types" %}

{% translate "Type" %} {% translate "Function" %} {% translate "Examples" %} {% translate "Punctuation" %}
{% translate "Coordinating" %} {% translate "Join equal elements" %} and, but, or, so {% translate "Comma before when joining clauses" %}
{% translate "Subordinating" %} {% translate "Introduce dependent clauses" %} because, although, when {% translate "Comma after initial dependent clause" %}
{% translate "Correlative" %} {% translate "Paired connectors" %} both...and, either...or {% translate "No special punctuation" %}
