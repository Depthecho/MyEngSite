{% translate "Correlative Conjunctions: The Power of Pairs" %}

{% translate "Correlative conjunctions are powerful grammatical tools that work in pairs to connect equivalent elements in a sentence, creating balanced and parallel structures that enhance clarity and rhythm." %}

{% translate "Essential Correlative Conjunction Pairs" %}

{% translate "Pair" %} {% translate "Function" %} {% translate "Examples" %}
both...and {% translate "Connects two positive alternatives" %} Both my brother and my sister attended.
She is both intelligent and hardworking.
either...or {% translate "Presents two alternatives (positive)" %} You can have either tea or coffee.
We'll go either today or tomorrow.
neither...nor {% translate "Presents two negative alternatives" %} I like neither coffee nor tea.
Neither the teacher nor the students knew.
not only...but also {% translate "Emphasizes two qualities/actions" %} She is not only talented but also dedicated.
He not only sings but also writes songs.
whether...or {% translate "Indicates alternatives in a choice" %} I don't know whether to laugh or cry.
Whether you go or stay, please decide soon.
as...as {% translate "Makes comparisons between equals" %} She is as tall as her brother.
This test is as difficult as the last one.

{% translate "The Golden Rule: Parallel Structure" %}

{% translate "Elements connected by correlative conjunctions must be parallel in grammatical form:" %}

{% translate "Noun + Noun, Verb + Verb, Adjective + Adjective, etc." %}

{% translate "Examples of Parallel Structure" %}

{% translate "Correct (Parallel)" %} {% translate "Incorrect (Non-Parallel)" %}
She not only sings but also dances.
(verb + verb)		 She not only sings but also a dancer.
(verb + noun)
I want either to stay or to leave.
(infinitive + infinitive)		 I want either to stay or leaving.
(infinitive + gerund)
He is both kind and generous.
(adjective + adjective)		 He is both kind and has generosity.
(adjective + verb phrase)

{% translate "Pro Tip:" %}

{% translate "When using 'not only...but also', the element after 'not only' determines what should follow 'but also'. Keep them in the same grammatical form." %}

{% translate "Special Grammar Rules" %}

{% translate "1. Subject-Verb Agreement" %}

{% translate "With 'either...or' and 'neither...nor', the verb agrees with the subject closer to it:" %}

  • Neither the teacher nor the students were there.
  • Neither the students nor the teacher was there.

{% translate "2. Positioning for Clarity" %}

{% translate "Place correlative conjunctions immediately before the words/phrases being connected:" %}

  • She not only speaks French but also Spanish. (Correct)
  • She speaks not only French but also Spanish. (Better)

{% translate "3. Avoiding Double Negatives" %}

{% translate "With 'neither...nor', don't use other negative words:" %}

  • I don't like neither tea nor coffee. (Incorrect)
  • I like neither tea nor coffee. (Correct)
