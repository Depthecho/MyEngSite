{% translate "Mastering Coordinating Conjunctions (FANBOYS)" %}

{% translate "Coordinating conjunctions are the fundamental connectors in English grammar, joining words, phrases, or independent clauses of equal grammatical rank. The seven coordinating conjunctions can be remembered with the acronym FANBOYS." %}

{% translate "The Complete FANBOYS Breakdown" %}

{% translate "FANBOYS Acronym:" %}

For, And, Nor, But, Or, Yet, So

{% translate "Conjunction" %} {% translate "Function" %} {% translate "Examples" %}
For {% translate "Indicates reason or cause (similar to 'because')" %} {% translate "I stayed home, for it was raining." %}
{% translate "He must be tired, for he worked all day." %}
And {% translate "Adds information or connects similar ideas" %} {% translate "She bought milk and eggs." %}
{% translate "I woke up early, and I went for a run." %}
Nor {% translate "Connects negative alternatives (requires inverted word order)" %} {% translate "He didn't call, nor did he text." %}
{% translate "I have never been to Paris, nor have I visited London." %}
But {% translate "Shows contrast or opposition between ideas" %} {% translate "I wanted to go, but I was too busy." %}
{% translate "She's small but strong." %}
Or {% translate "Presents alternatives or choices" %} {% translate "Would you like coffee or tea?" %}
{% translate "We can leave now, or we can wait until later." %}
Yet {% translate "Shows unexpected contrast (stronger than 'but')" %} {% translate "He's wealthy, yet he lives simply." %}
{% translate "She practiced daily, yet she didn't improve." %}
So {% translate "Indicates result or consequence" %} {% translate "It was cold, so I wore a jacket." %}
{% translate "I was hungry, so I ate a snack." %}

{% translate "Punctuation Rules for Coordinating Conjunctions" %}

{% translate "1. Joining Independent Clauses" %}

{% translate "When connecting two complete sentences (independent clauses), use a comma before the conjunction:" %}

  • {% translate "I wanted to go to the party, but I had too much work." %}
  • {% translate "She finished her homework, so she went to bed." %}

{% translate "Exception:" %}

{% translate "When the clauses are very short, the comma may be omitted:" %}

{% translate "I cooked and she cleaned." %}

{% translate "2. Joining Words or Phrases" %}

{% translate "No comma is needed when connecting words or short phrases:" %}

  • {% translate "bread and butter" %}
  • {% translate "quickly but carefully" %}
  • {% translate "in the morning or in the evening" %}

{% translate "3. Serial Commas (Oxford Comma)" %}

{% translate "When listing three or more items, use commas between them, with a conjunction before the last item:" %}

  • {% translate "I bought apples, oranges, and bananas." %}
  • {% translate "She can sing, dance, or play piano." %}

{% translate "Special Cases and Common Mistakes" %}

{% translate "1. Starting Sentences with Coordinating Conjunctions" %}

{% translate "While traditionally discouraged, starting sentences with coordinating conjunctions can be effective for emphasis in informal writing:" %}

  • {% translate "But I really wanted to go!" %}
  • {% translate "And that's how the story ends." %}

{% translate "Avoid overusing this in formal writing." %}

{% translate "2. 'Nor' with Inverted Word Order" %}

{% translate "After 'nor', the auxiliary verb comes before the subject:" %}

  • {% translate "He didn't call, nor did he send a message." %}
  • {% translate "I haven't seen that movie, nor have I read the book." %}

{% translate "3. 'For' vs. 'Because'" %}

{% translate "'For' as a conjunction is more formal and less common than 'because'. It typically appears between the clauses it connects:" %}

  • {% translate "I stayed inside, for it was raining heavily." %}
