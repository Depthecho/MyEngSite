{% translate "Conjunctions: The Connectors of English" %}

{% translate "Conjunctions are the grammatical glue that holds sentences together, allowing us to express complex thoughts and relationships between ideas." %}

{% translate "Definition and Core Functions" %}

{% translate "A conjunction is a word that connects words, phrases, clauses, or sentences. They serve three primary functions:" %}

{% translate "1. Joining Words" %}

  • {% translate "Connects two or more words of the same type (nouns, verbs, adjectives, etc.)" %}
  • {% translate "I like coffee and tea." %}
  • {% translate "She is quick but careful." %}

{% translate "2. Joining Phrases" %}

  • {% translate "Links prepositional phrases, verb phrases, etc." %}
  • {% translate "We can go by car or by train." %}
  • {% translate "He was not only late but also unprepared." %}

{% translate "3. Joining Clauses" %}

  • {% translate "Connects independent and/or dependent clauses" %}
  • {% translate "I wanted to go, but I was too tired." %}
  • {% translate "I'll call you when I arrive." %}

{% translate "Types of Conjunctions" %}

{% translate "1. Coordinating Conjunctions (FANBOYS)" %}

{% translate "Join elements of equal grammatical rank (words with words, phrases with phrases, etc.)" %}

FANBOYS - {% translate "The 7 coordinating conjunctions:" %}

{% translate "For, And, Nor, But, Or, Yet, So" %}

{% translate "Examples:" %}

  • {% translate "She likes tea but I prefer coffee." %}
  • {% translate "You can study now or you can fail the test." %}
  • {% translate "He was tired, yet he continued working." %}

{% translate "Usage Rules:" %}

  • {% translate "Use a comma before the conjunction when joining two independent clauses" %}
  • {% translate "No comma needed when joining words or short phrases" %}
  • {% translate "\"Nor\" requires inversion of the following auxiliary verb" %}

{% translate "2. Subordinating Conjunctions" %}

{% translate "Introduce dependent clauses and show their relationship to the main clause" %}

{% translate "Common Subordinating Conjunctions:" %}

{% translate "after, although, as, because, before, if, since, though, unless, until, when, where, while" %}

{% translate "Examples:" %}

  • {% translate "Although it was raining, we went for a walk." %}
  • {% translate "I'll stay until you return." %}
  • {% translate "She failed the test because she didn't study." %}

{% translate "Key Functions:" %}

{% translate "Relationship" %} {% translate "Conjunctions" %} {% translate "Example" %}
{% translate "Time" %} when, before, after, while {% translate "When she arrives, we'll eat." %}
{% translate "Cause/Effect" %} because, since, as {% translate "He left because he was tired." %}
{% translate "Condition" %} if, unless {% translate "If it rains, we'll cancel." %}
{% translate "Contrast" %} although, though, whereas {% translate "Although it's expensive, it's worth it." %}

{% translate "3. Correlative Conjunctions" %}

{% translate "Pairs of conjunctions that work together to join elements" %}

{% translate "Common Correlative Pairs:" %}

  • both...and
  • either...or
  • neither...nor
  • not only...but also
  • whether...or

{% translate "Examples:" %}

  • {% translate "Both my sister and my brother are doctors." %}
  • {% translate "You must either study hard or fail the exam." %}
  • {% translate "She is not only intelligent but also kind." %}

{% translate "Important Rules:" %}

  • {% translate "Maintain parallel structure between the connected elements" %}
  • {% translate "Subject-verb agreement follows the second element in 'or/nor' pairs" %}
  • {% translate "Place the correlative conjunctions immediately before the elements being connected" %}

{% translate "Common Errors and How to Avoid Them" %}

{% translate "1. Comma Splices" %}

{% translate "Using a comma alone to join two independent clauses (a comma splice)" %}

{% translate "Incorrect: I like coffee, she prefers tea." %}

{% translate "Correct: I like coffee, but she prefers tea." %}

{% translate "2. Misplaced Correlatives" %}

{% translate "Breaking the parallel structure in correlative conjunctions" %}

{% translate "Incorrect: She not only speaks French but also Spanish." %}

{% translate "Correct: She speaks not only French but also Spanish." %}

{% translate "3. Overusing 'And'" %}

{% translate "Stringing together too many ideas with 'and'" %}

{% translate "Weak: I woke up and I brushed my teeth and I had breakfast and I went to work." %}

{% translate "Better: I woke up, brushed my teeth, had breakfast, and then went to work." %}
