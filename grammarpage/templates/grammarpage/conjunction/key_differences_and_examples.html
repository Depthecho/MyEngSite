{% translate "Understanding the differences between these word types is crucial for proper sentence construction. Many words can function as different parts of speech depending on context." %}

{% translate "Conjunctions" %}

[Clause/Phrase] + [Conjunction] + [Clause/Phrase]

{% translate "Key Characteristics:" %}

  • {% translate "Join words, phrases, or clauses of equal importance" %}
  • {% translate "Coordinating conjunctions (FANBOYS) connect independent clauses" %}
  • {% translate "Subordinating conjunctions introduce dependent clauses" %}
  • {% translate "Correlative conjunctions work in pairs (both...and, either...or)" %}

{% translate "Examples:" %}

  • I like tea and coffee. (coordinating)
  • We'll go if it stops raining. (subordinating)
  • Both my parents and my siblings came. (correlative)

{% translate "Prepositions" %}

[Noun/Pronoun] + [Preposition] + [Noun Phrase]

{% translate "Key Characteristics:" %}

  • {% translate "Show relationship between a noun/pronoun and other words" %}
  • {% translate "Always followed by a noun or pronoun (the object of the preposition)" %}
  • {% translate "Form prepositional phrases that function as adjectives or adverbs" %}
  • {% translate "Cannot connect clauses or sentences" %}

{% translate "Common Prepositions:" %}

about, above, across, after, against, along, among, around, at, before, behind, below, beneath, beside, between, beyond, by, down, during, except, for, from, in, inside, into, like, near, of, off, on, onto, out, outside, over, past, since, through, throughout, to, toward, under, underneath, until, up, upon, with, within, without

{% translate "Examples:" %}

  • The book is on the table.
  • We walked through the park.
  • She arrived after dinner.

{% translate "Adverbs" %}

[Verb/Adjective/Adverb] + [Adverb]

{% translate "Key Characteristics:" %}

  • {% translate "Modify verbs, adjectives, or other adverbs" %}
  • {% translate "Answer questions like how, when, where, why, or to what extent" %}
  • {% translate "Often end in -ly (but not always)" %}
  • {% translate "Can appear in various positions in a sentence" %}

{% translate "Examples:" %}

  • She ran quickly. (modifies verb)
  • That was extremely difficult. (modifies adjective)
  • He spoke very softly. (modifies adverb)

{% translate "Words That Can Be Both" %}

{% translate "1. After" %}

  • We left after the movie. (preposition)
  • We left after the movie ended. (conjunction)

{% translate "2. Before" %}

  • Call me before dinner. (preposition)
  • Call me before you leave. (conjunction)

{% translate "3. Since" %}

  • I've been here since Tuesday. (preposition)
  • I've been here since you left. (conjunction)
  • I haven't seen him since. (adverb)
