{% translate "Tag Questions in English" %}

{% translate "A tag question is a short question added to the end of a statement, used to confirm information or seek agreement. It always ends with a question mark." %}

{% translate "Basic Structure" %}

{% translate "Tag questions follow this pattern:" %}

{% translate "Statement" %}, {% translate "auxiliary verb" %} + {% translate "pronoun" %}?

{% translate "Examples:" %}

  • {% translate 'You\'re coming with us, aren\'t you' %}?
  • {% translate 'She doesn\'t know about this, does she' %}?
  • {% translate 'They\'ve finished their work, haven\'t they' %}?

{% translate "Formation Rules" %}

1. {% translate "Positive-Negative Pattern" %}

{% translate "When the statement is positive, the tag is negative:" %}

  • {% translate 'You like coffee, don\'t you' %}?

2. {% translate "Negative-Positive Pattern" %}

{% translate "When the statement is negative, the tag is positive:" %}

  • {% translate 'You don\'t like tea, do you' %}?

3. {% translate "Auxiliary Verb Matching" %}

{% translate "The tag uses the same auxiliary verb as the main clause:" %}

  • {% translate 'She is coming, isn\'t she' %}?
  • {% translate 'You can swim, can\'t you' %}?

{% translate "Special Cases and Exceptions" %}

{% translate "Case" %} {% translate "Example" %} {% translate "Rule" %}
{% translate "Imperatives" %} {% translate 'Let\'s go, shall we' %}? {% translate "Use 'shall we' for suggestions" %}
{% translate "I am" %} {% translate 'I\'m late, aren\'t I' %}? {% translate "Exception to auxiliary matching" %}
{% translate "No auxiliary" %} {% translate 'You like it, don\'t you' %}? {% translate "Use 'do/does/did' when no auxiliary" %}
{% translate "Negative words" %} {% translate 'You never listen, do you' %}? {% translate "Words like 'never' count as negative" %}

{% translate "Intonation and Meaning" %}

  • {% translate "Falling intonation: Speaker expects agreement" %}

    {% translate 'It\'s cold today, isn\'t it' %}↘?

  • {% translate "Rising intonation: Genuine question" %}

    {% translate 'You haven\'t seen my keys, have you' %}↗?

  • {% translate "Sarcastic use: When the answer is obvious" %}

    {% translate 'Oh, you broke it, did you' %}?
