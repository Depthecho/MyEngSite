{% translate "Quotation Marks for Direct Speech" %}

{% translate "Quotation marks (\"\" or '') enclose exact spoken or written words, preserving the original phrasing and punctuation of the quoted material." %}

{% translate "Primary Usage" %}

{% translate "Quotation marks are used for:" %}

  • {% translate "Direct speech (dialogues)" %}
  • {% translate "Quoting text verbatim" %}
  • {% translate "Highlighting words as words" %}
  • {% translate "Titles of short works" %}

{% translate "Basic Structure:" %}

{% translate "Speaker" %} + {% translate "reporting clause" %} + "{% translate "Direct speech" %}"

{% translate "Detailed Rules" %}

1. {% translate "Punctuation Placement" %}

{% translate "In American English, punctuation goes inside quotes:" %}

  • She said, "I\'ll be there by 5."
  • Did he say "Wait"?

2. {% translate "Quotes Within Quotes" %}

{% translate "Alternate between double and single quotes:" %}

  • He explained, "She told me \'No\' immediately."

3. {% translate "Paragraph Breaks" %}

{% translate "For multi-paragraph quotes, start each paragraph with opening quotes but only close the final paragraph:" %}

  • "First paragraph...\n"Second paragraph..."

{% translate "American vs British Usage" %}

{% translate "Feature" %} {% translate "American English" %} {% translate "British English" %}
{% translate "Primary Quotes" %} "Double" \'Single\'
{% translate "Nested Quotes" %} \'Single\' "Double"
{% translate "Punctuation" %} {% translate "Inside quotes" %} {% translate "Outside unless part of quote" %}
{% translate "Examples" %} She said, "Hello." She said, \'Hello\'.

{% translate "Special Cases" %}

  • {% translate "Block quotes: For long quotes (usually 4+ lines), indent without quotation marks" %}
  • {% translate "Scare quotes: Use to distance yourself from terminology" %}

    The so-called "experts" disagreed.

  • {% translate "Thoughts: Often italicized rather than quoted" %}
  • {% translate "Dialogue tags: Use commas to separate from quotes" %}

    "I\'m tired," she said.
