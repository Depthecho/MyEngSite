{% translate "The Dash: Usage and Rules" %}

{% translate "The dash — is a versatile punctuation mark used for emphasis, interruption, or to set off parenthetical information. There are two main types: the em dash (—) and the en dash (–), with the em dash being most common for these purposes." %}

{% translate "1. Primary Uses of the Em Dash" %}

{% translate "A. Emphasis and Dramatic Pause" %}

{% translate "Dashes create stronger breaks than commas, adding dramatic emphasis to the enclosed text." %}

  • After years of waiting — at long last — the day had arrived.
  • There was only one thing left to do — run!

{% translate "B. Interruptions and Abrupt Changes" %}

{% translate "Dashes can indicate an interruption in thought or speech." %}

  • "I was going to say — oh never mind."
  • "If you would just let me fin — " "No, I won't!"

{% translate "2. Parenthetical and Additional Information" %}

{% translate "A. Parenthetical Statements" %}

{% translate "Dashes can set off parenthetical information with more emphasis than commas or parentheses." %}

  • The ingredients — flour, sugar, and eggs — were all on the counter.

{% translate "B. Appositives with Extra Emphasis" %}

{% translate "Dashes highlight appositives (renaming phrases) more dramatically than commas." %}

  • Three countries — France, Spain, and Italy — border the Mediterranean.

{% translate "Dash vs. Other Punctuation" %}

{% translate "Punctuation" %} {% translate "Effect" %} {% translate "Example" %}
{% translate "Comma" %} {% translate "Standard separation" %} The book, which was heavy, fell.
{% translate "Parentheses" %} {% translate "Subdued, supplementary info" %} The book (a first edition) fell.
{% translate "Dash" %} {% translate "Dramatic, emphasized info" %} The book — a priceless antique — fell.

{% translate "3. Special Cases and Rules" %}

{% translate "A. En Dash Usage" %}

{% translate "The shorter en dash (–) has specific uses:" %}

  • {% translate "Number ranges:" %} pages 10 – 15
  • {% translate "Connections between words:" %} the London – Paris flight
  • {% translate "Scores and conflicts:" %} The Lakers won 98 – 95

{% translate "B. Spacing Rules" %}

{% translate "Style guides differ on spacing around dashes:" %}

  • {% translate "AP Style:" %} {% translate "space before and after em dash" %}
  • {% translate "Chicago Manual:" %} {% translate "no spaces around em dash" %}
  • {% translate "En dashes:" %} {% translate "typically have spaces around them" %}

{% translate "Common Mistakes to Avoid" %}

  • {% translate "Overuse: Too many dashes can make writing seem choppy" %}

    Poor: The day — bright and sunny — was perfect — or so we thought — for the picnic.

  • {% translate "Confusing dashes: Don't mix up hyphens (-), en dashes (–), and em dashes (—)" %}

    Incorrect: 1990-2000 (should be en dash)

    Incorrect: The answer-no-was unexpected (should be em dashes)

  • {% translate "Inconsistent usage: Choose one style and stick with it" %}

    Inconsistent: The results — surprising to all — were announced (but later uses parentheses for similar constructions)
