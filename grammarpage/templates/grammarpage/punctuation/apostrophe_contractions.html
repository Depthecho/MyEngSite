{% translate "Contractions in English" %}

{% translate "The apostrophe in contractions replaces omitted letters when combining words, making speech more natural and writing more conversational." %}

{% translate "Basic Contractions" %}

{% translate "Common contractions combine pronouns with verbs or negate verbs:" %}

{% translate "Structure:" %}

{% translate "Word1 + Word2" %} → {% translate "Combined form" %}'{% translate "contraction" %}

{% translate "Examples:" %}

  • I am → I'm
  • You are → You're
  • They have → They've

{% translate "Types of Contractions" %}

1. {% translate "Verb + Not" %}

{% translate "Negating verbs with 'not':" %}

  • Do not → Don't
  • Cannot → Can't
  • Will not → Won't

2. {% translate "Pronoun + Verb" %}

{% translate "Combining pronouns with auxiliary verbs:" %}

  • She is → She's
  • We have → We've
  • They would → They'd

3. {% translate "Question Words + Is/Has" %}

{% translate "Combining question words with verbs:" %}

  • What is → What's
  • Who has → Who's

{% translate "Contraction Usage Guidelines" %}

{% translate "Context" %} {% translate "Appropriate" %} {% translate "Inappropriate" %}
{% translate "Informal Writing" %} I don't know I do not know
{% translate "Formal Writing" %} Do not enter Don't enter
{% translate "Dialogue" %} "You're late," she said. "You are late," she said.
{% translate "Academic Papers" %} The results are significant The results're significant

{% translate "Special Cases and Exceptions" %}

  • {% translate "Homophones: Some contractions sound like other words" %}

    they're (they are) vs their vs there

    who's (who is/has) vs whose

  • {% translate "Double Contractions: Informal forms like 'shouldn't've'" %}

    Avoid in formal writing

  • {% translate "Non-Standard Forms: Dialectal forms like 'ain't' should be used carefully" %}
  • {% translate "Possessive Pronouns: Don't confuse with contractions (its vs it's)" %}

{% translate "Convert these to contractions where appropriate:" %}

  1. She will not be coming
  2. I would have told you
  3. It is raining ({% translate "informal context" %})
  4. They are going to the park
