{% translate "The use of full stops in abbreviations varies significantly between style guides and English variants, with American English generally using more periods in abbreviations than British English." %}

{% translate "General Rules for Abbreviations" %}

{% translate "Full stops are used differently depending on the type of abbreviation and English variant:" %}

{% translate "1. Initialisms vs. Acronyms" %}

{% translate "Initialisms (pronounced as individual letters) and acronyms (pronounced as words) have different conventions:" %}

{% translate "Type" %} {% translate "Example" %} {% translate "Punctuation" %}
Initialism (American) U.S.A., F.B.I. {% translate "Periods after each letter" %}
Initialism (British) USA, FBI {% translate "No periods" %}
Acronym NASA, NATO {% translate "No periods in any variant" %}

{% translate "2. Latin Abbreviations" %}

{% translate "Common Latin abbreviations typically use periods in all English variants:" %}

  • e.g. (exempli gratia - for example)
  • i.e. (id est - that is)
  • etc. (et cetera - and so forth)

{% translate "3. Titles and Measurements" %}

{% translate "Usage varies between American and British English:" %}

{% translate "Abbreviation" %} {% translate "American" %} {% translate "British" %}
Mister Mr. Mr
Doctor Dr. Dr
Ante meridiem a.m. am
Post meridiem p.m. pm

{% translate "Modern Trends and Style Guides" %}

{% translate "Contemporary usage shows these patterns in different contexts:" %}

{% translate "1. Digital Communication" %}

{% translate "Periods are increasingly omitted in informal digital communication:" %}

  • txt msg (text message)
  • info (information)
  • app (application)

{% translate "2. Academic and Formal Writing" %}

{% translate "Style guides differ in their recommendations:" %}

{% translate "Style Guide" %} {% translate "Approach" %} {% translate "Example" %}
APA (American) {% translate "Minimal periods" %} US, PhD, eg
Chicago Manual {% translate "Traditional periods" %} U.S., Ph.D., e.g.
Oxford Style {% translate "No periods in acronyms" %} UK, EU, BBC

{% translate "Special Cases and Exceptions" %}

{% translate "These situations require special attention:" %}

{% translate "1. Abbreviations at Sentence End" %}

{% translate "When an abbreviation ends a sentence, use only one period:" %}

  • The meeting is at 3 p.m. (not "3 p.m..")

{% translate "2. Plural Abbreviations" %}

{% translate "Form plurals of abbreviations without apostrophes:" %}

  • CDs (not CD's)
  • the 1990s (not 1990's)

{% translate "3. Measurements and Symbols" %}

{% translate "Units of measurement typically don't use periods:" %}

  • 5 kg (not 5 kg.)
  • 10 cm (not 10 cm.)
