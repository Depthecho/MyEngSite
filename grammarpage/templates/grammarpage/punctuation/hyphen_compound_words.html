{% translate "The Hyphen: Usage and Rules" %}

{% translate "The hyphen - is a punctuation mark used to join words and to separate syllables of a single word. It plays crucial roles in forming compound words, avoiding ambiguity, and ensuring proper word division." %}

{% translate "1. Compound Words with Hyphens" %}

{% translate "A. Compound Adjectives Before Nouns" %}

{% translate "Hyphens connect words that work together to modify a noun that follows them." %}

  • a well-known author
  • a five-year-old child
  • an up-to-date report

{% translate "B. Permanent Compound Words" %}

{% translate "Some words are always hyphenated, regardless of their position in a sentence." %}

  • mother--in--law
  • editor--in--chief
  • merry--go--round

{% translate "2. Numbers and Fractions" %}

{% translate "A. Compound Numbers (21-99)" %}

{% translate "Hyphens are used in written-out numbers between twenty-one and ninety-nine." %}

  • thirty--two
  • one hundred twenty--five

{% translate "B. Fractions" %}

{% translate "Hyphens connect the numerator and denominator in written fractions." %}

  • two--thirds
  • three--fourths

{% translate "Prefixes before capitalized words" %} {% translate "Always use a hyphen" %} un--American

{% translate "3. Special Cases" %}

{% translate "A. Word Division" %}

{% translate "Hyphens indicate where a word breaks at the end of a line." %}

  • The document contained important infor-
    mation about the project.

{% translate "B. Avoiding Confusion" %}

{% translate "Hyphens can prevent misreading of words." %}

  • a small--business owner

    (vs. a small business owner, which could imply the person is short)

{% translate "C. Prefixes with the Same Letter" %}

{% translate "Hyphens are used when a prefix ends with the same letter the root word begins with." %}

  • anti--inflammatory
  • re--elect

{% translate "Common Mistakes to Avoid" %}

  • {% translate "Hyphen vs. dash: Don't confuse hyphens (-) with en dashes (–) or em dashes (—)" %}

    Incorrect: 1990-2000 (should be en dash)

  • {% translate "Over-hyphenation: Many compound words don't need hyphens" %}

    Incorrect: e--mail (correct: email)

  • {% translate "Inconsistent hyphenation: Be consistent within a document" %}

    Inconsistent: decision-making and decision making

  • {% translate "Hyphenating adverbs: Don't hyphenate adverbs ending in -ly" %}

    Incorrect: a highly--paid executive

    Correct: a highly paid executive
