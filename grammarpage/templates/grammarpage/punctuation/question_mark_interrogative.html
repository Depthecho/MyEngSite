{% translate "The Question Mark in English" %}

{% translate "The question mark (?) is a punctuation mark that indicates an interrogative sentence or clause. It replaces the period at the end of direct questions." %}

{% translate "Primary Usage" %}

{% translate "Question marks are used to end direct questions that expect an answer." %}

{% translate "Examples:" %}

  • What time is it?
  • Are you coming to the party?
  • How did you solve this problem?

{% translate "Types of Questions That Require Question Marks" %}

1. {% translate "Yes/No Questions" %}

{% translate "Questions that can be answered with 'yes' or 'no'." %}

  • Do you like coffee?
  • Has she finished her work?

2. {% translate "Wh- Questions" %}

{% translate "Questions beginning with question words (who, what, when, where, why, how)." %}

  • Where did you put the keys?
  • Why are you late?

3. {% translate "Tag Questions" %}

{% translate "Statements turned into questions with question tags." %}

  • You're coming with us, aren't you?
  • She doesn't know about this, does she?

{% translate "Special Cases and Exceptions" %}

{% translate "Case" %} {% translate "Example" %} {% translate "Explanation" %}
{% translate "Indirect questions" %} I wonder what time it is. {% translate "No question mark - these are statements" %}
{% translate "Polite requests" %} Would you please send me the file. {% translate "Period is acceptable for polite commands" %}
{% translate "Rhetorical questions" %} Who wouldn't want free vacation days? {% translate "Question mark required despite no answer expected" %}
{% translate "Question within a sentence" %} The question "Where are we going?" kept repeating in my mind. {% translate "Question mark stays with the quoted question" %}

{% translate "Advanced Usage" %}

  • {% translate "Multiple questions: Each short question gets its own mark" %}

    Where to? When? With whom?

  • {% translate "Question marks with quotes: The mark goes inside if it's part of the quote" %}

    She asked, 'What time is it?'

    Did she say, 'I'll be late'?

  • {% translate "Spanish influence: Inverted question mark (¿) is not used in English" %}

    Incorrect: ¿How are you?
