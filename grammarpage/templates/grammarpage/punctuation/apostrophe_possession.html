{% translate "Possessive Apostrophes in English" %}

{% translate "The apostrophe (') is used with 's' to indicate ownership or possession. Proper use clarifies relationships between nouns and their possessors." %}

{% translate "Basic Rules" %}

{% translate "The apostrophe placement changes based on whether the noun is singular or plural:" %}

{% translate "Structure:" %}

{% translate "Owner" %}'s {% translate "or" %} s' {% translate "+ possessed item" %}

{% translate "Examples:" %}

  • The teacher's desk
  • The teachers' lounge

{% translate "Detailed Usage" %}

1. {% translate "Singular Nouns" %}

{% translate "Add 's to all singular nouns, including those ending in s:" %}

  • The cat's tail
  • James's car ({% translate "or James'" %})

2. {% translate "Regular Plural Nouns" %}

{% translate "Add only an apostrophe after the existing s:" %}

  • The students' projects
  • The companies' policies

3. {% translate "Irregular Plural Nouns" %}

{% translate "Add 's to plurals that don't end in s:" %}

  • The children's toys
  • The women's rights

{% translate "Special Cases" %}

{% translate "Case" %} {% translate "Example" %} {% translate "Rule" %}
{% translate "Joint Possession" %} {% translate 'John and Mary\'s house' %} {% translate "Add 's only to last name" %}
{% translate "Separate Possession" %} {% translate 'John\'s and Mary\'s cars' %} {% translate "Add 's to each name" %}
{% translate "Names Ending in S" %} {% translate 'Charles\'s book' %}
{% translate 'or Charles\' book' %}		 {% translate "Both forms acceptable" %}
{% translate "Time Expressions" %} {% translate 'a week\'s vacation' %} {% translate "Treat time periods as possessors" %}

{% translate "Common Mistakes" %}

  • {% translate 'Its vs. It\'s: \'Its\' is possessive, \'it\'s\' means \'it is\'' %}

    {% translate 'Correct: The dog wagged its tail.' %}

    {% translate 'Correct: It\'s (it is) raining.' %}

  • {% translate 'Plural Nouns: Don\'t use apostrophes for simple plurals' %}

    {% translate 'Incorrect: banana\'s for sale' %}

  • {% translate 'Pronouns: Possessive pronouns (yours, hers, theirs) don\'t use apostrophes' %}
