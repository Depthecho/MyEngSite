{% translate "Brackets: Usage and Rules" %}

{% translate "Brackets [ ] are punctuation marks used primarily for editorial clarification within quoted material or to insert explanatory information within parentheses." %}

{% translate "1. Primary Uses of Brackets" %}

{% translate "A. Clarifying Quotations" %}

{% translate "Brackets are used to add explanatory words or phrases to quoted material when the original meaning might be unclear." %}

  • Original: "He arrived yesterday."
    With clarification: "He [the defendant] arrived yesterday."
  • "The results [of the experiment] were inconclusive."

{% translate "B. Indicating Errors with [sic]" %}

{% translate "The Latin term 'sic' (meaning 'thus') is used in brackets to show that an error in a quotation appears in the original source." %}

  • "The committee has it's [sic] regular meeting on Tuesday."
  • "She was know [sic] for her generosity."

{% translate "2. Specialized Uses" %}

{% translate "A. Translations in Quotations" %}

{% translate "Brackets can provide translations of foreign words or phrases within quotes." %}

  • "He exclaimed 'Vive la France' [Long live France] as he left."

{% translate "B. Editorial Comments" %}

{% translate "Editors may use brackets to insert comments or notes within quoted text." %}

  • "The event [emphasis added] changed everything."

{% translate "C. Nested Parentheses" %}

{% translate "When additional information is needed within parentheses, brackets are used to avoid confusion." %}

  • The author made this claim (based on earlier research [see Smith, 1999]).

{% translate "Punctuation with Brackets" %}

{% translate "Case" %} {% translate "Rule" %} {% translate "Example" %}
{% translate "Brackets within a sentence" %} {% translate "No additional punctuation needed inside brackets" %} "The document [dated 1945] was recently discovered."
{% translate "Complete sentence in brackets" %} {% translate "Capitalize first word and end with period inside" %} "The results were unexpected. [See Appendix B for details.]"
{% translate "Brackets at end of quoted sentence" %} {% translate "Period goes outside the closing bracket" %} "The meeting is scheduled for noon [EST]."
{% translate "Brackets with other punctuation" %} {% translate "Other punctuation remains outside unless part of bracketed material" %} "Did he really say 'I won't go' [sic]?"

{% translate "3. Brackets vs. Parentheses" %}

{% translate "Key Differences" %}

  • Brackets [ ]: {% translate "Used primarily within quotations to add editorial comments or clarifications" %}

    "The witness stated: 'He [the defendant] was present that night.'"

  • Parentheses ( ): {% translate "Used to add supplementary information to your own writing" %}

    The results (shown in Table 3) support our hypothesis.

{% translate "Common Mistakes to Avoid" %}

  • {% translate "Overuse: Excessive brackets can make text difficult to read" %}

    Poor: "The study [conducted in 2020] showed [contrary to expectations]"

  • {% translate "Misuse of [sic]: Only use for actual errors in the original" %}

    Incorrect: "The color [sic] was beautiful."

    (Unless 'color' was actually an error in the original)

  • {% translate "Formatting errors: Remember proper spacing and punctuation" %}

    Incorrect: "The event was[important]."

    Correct: "The event was [important]."
