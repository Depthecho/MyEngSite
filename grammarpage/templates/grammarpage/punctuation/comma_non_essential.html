{% translate "Commas for Non-Essential Information" %}

{% translate "Commas play a crucial role in setting off information that adds extra details to a sentence but isn't necessary for its core meaning. This is called non-restrictive or non-essential information." %}

{% translate "Core Concept" %}

{% translate "Non-essential elements can be removed without changing the fundamental meaning of the sentence. These are always set off with commas." %}

{% translate "Key Characteristics:" %}

  • {% translate "The information is additional, not required to identify the noun" %}
  • {% translate "The clause often begins with who, which, whose, where, when" %}
  • {% translate "The sentence would remain grammatically correct without it" %}

{% translate "Types of Non-Essential Elements" %}

1. {% translate "Non-Restrictive Relative Clauses" %}

{% translate "These clauses provide extra information about a noun that's already clearly identified." %}

  • {% translate "Example:" %}
    My car, which is parked outside, needs washing.

    {% translate "(We know which car - the additional information about its location isn't necessary to identify it)" %}

2. {% translate "Appositives" %}

{% translate "Noun phrases that rename or explain another noun right beside it." %}

  • {% translate "Example:" %}
    Paris, the capital of France, is beautiful in spring.

3. {% translate "Parenthetical Expressions" %}

{% translate "Words or phrases that interrupt the flow of a sentence." %}

  • {% translate "Example:" %}
    The solution, however, was quite simple.

{% translate "Essential vs. Non-Essential Comparison" %}

{% translate "Feature" %} {% translate "Non-Essential" %} {% translate "Essential" %}
{% translate "Commas" %} {% translate "Required" %} {% translate "Not used" %}
{% translate "Removability" %} {% translate "Can be removed without changing core meaning" %} {% translate "Cannot be removed - changes meaning" %}
{% translate "Relative Pronouns" %} Which, who(m), whose, where, when That, who(m), whose, where, when
{% translate "Example" %} My doctor, who graduated from Harvard, is excellent. The doctor who graduated from Harvard is excellent.

{% translate "Special Cases and Exceptions" %}

  • {% translate "Which vs. That: 'Which' typically introduces non-essential clauses, while 'that' introduces essential ones" %}

    The book that I borrowed is interesting. (essential)

    The book, which I borrowed yesterday, is interesting. (non-essential)

  • {% translate "Proper Nouns: Information following proper nouns is usually non-essential" %}

    London, where I was born, is crowded.

  • {% translate "Adjective Clauses: When the information is clearly additional" %}

    The elderly man, who seemed lost, asked for directions.

{% translate "Practice Examples" %}

{% translate "Identify whether these sentences need commas for non-essential information:" %}

  1. My sister who lives in Boston is visiting.
  2. The Empire State Building which has 102 floors is in New York.
  3. Shakespeare's play Hamlet is his longest.
  4. The professor wearing the red tie is my advisor.
