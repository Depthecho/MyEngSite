{% translate "Mastering the Prepositions: By, For, and From" %}

{% translate "These three prepositions are among the most frequently used in English, each serving distinct grammatical functions to express relationships of agency, purpose, and origin." %}

{% translate "1. The Preposition 'By'" %}

{% translate "Core Meanings and Uses" %}

{% translate "Function" %} {% translate "Explanation" %} {% translate "Examples" %}
{% translate "Agent in passive voice" %} {% translate "Indicates who performs the action" %} The novel was written by Hemingway.
{% translate "Means/method" %} {% translate "Shows how something is done" %} We traveled by plane.
{% translate "Time deadline" %} {% translate "Indicates a time before which something must happen" %} Submit your work by Friday.
{% translate "Proximity" %} {% translate "Shows physical closeness" %} She sat by the window.

{% translate "Special Constructions with 'By'" %}

  • {% translate "By + reflexive pronoun:" %} {% translate "To mean 'alone'" %}

    I did it by myself.

  • {% translate "By + time expressions:" %} {% translate "For regular intervals" %}

    He gets paid by the hour.

  • {% translate "By + gerund:" %} {% translate "To describe methods" %}

    You can improve by practicing daily.

{% translate "2. The Preposition 'For'" %}

{% translate "Primary Functions" %}

{% translate "Function" %} {% translate "Explanation" %} {% translate "Examples" %}
{% translate "Purpose" %} {% translate "Indicates why something is done" %} This tool is for cutting wood.
{% translate "Beneficiary" %} {% translate "Shows who benefits from an action" %} I bought flowers for my mother.
{% translate "Duration" %} {% translate "Expresses how long something lasts" %} We waited for three hours.
{% translate "Exchange" %} {% translate "Indicates a trade or substitution" %} I'll trade you my sandwich for your apple.

{% translate "Idiomatic Uses of 'For'" %}

  • {% translate "For + noun:" %} {% translate "To express purpose" %}

    This is for emergencies only.

  • {% translate "For + gerund:" %} {% translate "After certain adjectives" %}

    This tool is useful for opening jars.

  • {% translate "For + period:" %} {% translate "With present perfect" %}

    I've lived here for five years.

{% translate "3. The Preposition 'From'" %}

{% translate "Key Applications" %}

{% translate "Function" %} {% translate "Explanation" %} {% translate "Examples" %}
{% translate "Origin" %} {% translate "Shows where someone/something comes from" %} She's from Canada.
{% translate "Source" %} {% translate "Indicates where something originates" %} I got this information from the internet.
{% translate "Time starting point" %} {% translate "Marks when something begins" %} The store is open from 9 a.m.
{% translate "Material change" %} {% translate "Shows transformation" %} Wine is made from grapes.

{% translate "Special Patterns with 'From'" %}

  • {% translate "From...to:" %} {% translate "For ranges" %}

    We work from Monday to Friday.

  • {% translate "Prevent/protect from:" %} {% translate "For avoidance" %}

    This cream protects from sunburn.

  • {% translate "Different from:" %} {% translate "For comparisons" %}

    Your opinion is different from mine.

{% translate "Comparative Usage: By vs. For vs. From" %}

{% translate "These prepositions sometimes overlap in usage, leading to common confusion. Here's how to distinguish them:" %}

{% translate "Context" %} {% translate "Correct Preposition" %} {% translate "Explanation" %}
{% translate "Creator vs. Purpose" %} by / for {% translate "\"A painting by Van Gogh\" (creator) vs. \"A brush for painting\" (purpose)" %}
{% translate "Duration vs. Deadline" %} for / by {% translate "\"I studied for two hours\" (duration) vs. \"Finish by 5 PM\" (deadline)" %}
{% translate "Origin vs. Purpose" %} from / for {% translate "\"A letter from my friend\" (origin) vs. \"Paper for writing letters\" (purpose)" %}

{% translate "Common Mistakes to Avoid" %}

  • {% translate "Incorrect:" %} This book was written from Shakespeare.

    {% translate "Correct:" %} This book was written by Shakespeare.

  • {% translate "Incorrect:" %} I need this report for tomorrow.

    {% translate "Correct:" %} I need this report by tomorrow.

  • {% translate "Incorrect:" %} She borrowed money by her friend.

    {% translate "Correct:" %} She borrowed money from her friend.
