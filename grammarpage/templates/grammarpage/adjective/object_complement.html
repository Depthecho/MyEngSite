{% translate "Adjectives: Object Complement" %}

{% translate "In some sentences, an adjective can function as an object complement. This means the adjective follows a direct object and describes or renames that object, not the subject of the sentence. This structure is used with verbs that describe a state or change of state, such as " %}make, consider, find, call{% translate ", etc." %}

{% translate "1. The Basic Structure" %}

{% translate "The structure for an object complement is: Subject + Verb + Direct Object + Adjective." %}

{% translate "Sentence Structure" %} {% translate "Example" %}
{% translate "Subject + Verb + Direct Object + Adjective" %} He painted the wall red.
{% translate "Subject + Verb + Direct Object + Adjective" %} They found the movie boring.
{% translate "Subject + Verb + Direct Object + Adjective" %} She considers him kind.

{% translate "In these examples, the adjectives " %}red, boring, {% translate "and" %} kind {% translate "are describing the direct objects (" %}the wall, the movie, him{% translate "), not the subjects of the sentences." %}

{% translate "2. Distinguishing from Other Functions" %}

{% translate "It's important to differentiate an object complement from a predicative adjective, which describes the subject. The position of the adjective and the verb used are key." %}

{% translate "Function" %} {% translate "Example" %} {% translate "What the Adjective Describes" %}
Predicative The house is white. {% translate "The subject, 'house'." %}
Object Complement They painted the house white. {% translate "The direct object, 'house'." %}

{% translate "3. Common Verbs Used with Object Complements" %}

{% translate "Here is a list of some common verbs that can be used with an adjective as an object complement:" %}

  • make: The noise made me angry.
  • consider: I consider the project complete.
  • find: We find the story interesting.
  • elect: They elected him president.
  • call: You can call me lucky.
