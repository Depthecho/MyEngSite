{% translate "Adjectives: Attributive Function" %}

{% translate "The attributive function is the most common use of an adjective. When an adjective is used attributively, it is placed directly before the noun it modifies. The adjective is considered a part of the noun phrase, providing a direct description or quality of the noun." %}

{% translate "1. Basic Structure" %}

{% translate "The standard structure is a simple noun phrase: Article + Adjective + Noun." %}

  • {% translate "a red car" %}
  • {% translate "the tall man" %}
  • {% translate "many difficult questions" %}

{% translate "In these examples, the adjectives red, tall, and difficult are placed before the nouns they describe, acting as direct attributes of those nouns." %}

{% translate "2. Multiple Attributive Adjectives" %}

{% translate "When multiple adjectives are used to describe a single noun, they are all placed before the noun and follow a specific order (e.g., Opinion, Size, Age, Color, etc.)." %}

  • {% translate "a beautiful old black car" %}
  • {% translate "the huge round wooden table" %}

{% translate "This structure remains attributive, as all adjectives are part of the noun phrase that comes before the noun itself." %}

{% translate "3. Adjectives That Are ONLY Attributive" %}

{% translate "Some adjectives can only be used in an attributive position and cannot be used predicatively (after a verb). Common examples include:" %}

  • {% translate "main" %}: {% translate "the main reason" %}, not the reason is main
  • {% translate "chief" %}: {% translate "the chief complaint" %}, not the complaint is chief
  • {% translate "former" %}: {% translate "the former president" %}, not the president is former

{% translate "This is an important distinction to avoid grammatical errors when using these specific words." %}
