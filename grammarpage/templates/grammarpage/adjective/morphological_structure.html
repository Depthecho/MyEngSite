{% translate "1. Simple and Derived Adjectives" %}

{% translate "Most adjectives in English are simple words that don't have affixes (e.g., " %}big, tall, happy{% translate "). However, many adjectives are formed from other parts of speech by adding a suffix." %}

{% translate "Suffix" %} {% translate "From" %} {% translate "Example" %}

-less {% translate "Noun" %} hope → hopeless
-able / -ible {% translate "Verb" %} read → readable
-ic {% translate "Noun" %} economy → economic
-ous {% translate "Noun" %} fame → famous
