{% translate "Adjectives: Predicative Function" %}

{% translate "When an adjective is used predicatively, it describes the subject of the sentence but is located after a verb. The adjective is not part of the noun phrase; instead, it functions as a subject complement, providing more information about the subject." %}

{% translate "1. Basic Structure" %}

{% translate "The predicative adjective typically follows a linking verb, such as to be, to seem, to appear, to become, or sensory verbs like to taste, to feel, to smell, to sound, to look." %}

{% translate "Sentence Structure" %} {% translate "Example" %}
{% translate "Subject + Linking Verb + Adjective" %} {% translate "The car is red." %}
{% translate "Subject + Linking Verb + Adjective" %} {% translate "The soup tastes delicious." %}
{% translate "Subject + Linking Verb + Adjective" %} {% translate "She seems happy." %}

{% translate "In these examples, the adjectives red, delicious, and happy describe the subjects (car, soup, she) but are separated from them by a verb." %}

{% translate "2. Comparing Attributive vs. Predicative" %}

{% translate "Understanding the difference is crucial for proper sentence construction. The two functions can often be used to convey the same information, but with a different emphasis." %}

  • {% translate "Attributive:" %} {% translate "A happy dog barked loudly." %}
  • {% translate "Predicative:" %} {% translate "The dog was happy and barked loudly." %}

{% translate "While both sentences describe a happy dog, the attributive form is more direct, while the predicative form often feels like a statement of a condition." %}

{% translate "3. Adjectives That Are ONLY Predicative" %}

{% translate "Just as some adjectives are only attributive, others can only be used predicatively. These adjectives often start with the prefix a-." %}

  • {% translate "afraid" %}: {% translate "The child is afraid." %}, not the afraid child
  • {% translate "asleep" %}: {% translate "The baby is asleep." %}, not the asleep baby
  • {% translate "alive" %}: {% translate "The patient is alive." %}, not the alive patient

{% translate "Knowing these specific words helps to prevent common grammatical errors." %}
