{% translate "Adjective: Formation Rules" %}

{% translate "The way comparative and superlative degrees are formed depends on the number of syllables in the adjective. There are three main rules to follow." %}

{% translate "1. One-Syllable Adjectives" %}

{% translate "For most one-syllable adjectives, you form the comparative by adding -er and the superlative by adding -est." %}

{% translate "Positive" %} {% translate "Comparative" %} {% translate "Superlative" %}
tall taller tallest
old older oldest

{% translate "Spelling Rule: If the adjective ends in a single vowel + consonant, double the final consonant before adding the suffix (e.g., " %}big → bigger → biggest{% translate ")." %}

{% translate "2. Two-Syllable Adjectives" %}

{% translate "For most two-syllable adjectives, you use more for the comparative and most for the superlative. However, some two-syllable adjectives can use both forms." %}

{% translate "Positive" %} {% translate "Comparative" %} {% translate "Superlative" %}
beautiful more beautiful most beautiful
careful more careful most careful

{% translate "Note: Adjectives ending in -y often follow the one-syllable rule, changing the y to i (e.g., " %}happy → happier → happiest{% translate ")." %}

{% translate "3. Three or More Syllable Adjectives" %}

{% translate "All adjectives with three or more syllables form the comparative and superlative degrees using more and most." %}

  • expensive → more expensive → most expensive
  • interesting → more interesting → most interesting
