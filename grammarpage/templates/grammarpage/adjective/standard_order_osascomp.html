{% translate "Order of Adjectives: The Standard Order (OSASCOMP)" %}

{% translate "When a noun is modified by multiple adjectives, they must follow a specific, unspoken order. English speakers use this order naturally, but for learners, a mnemonic device can be very helpful. This standard order is often referred to by the acronym OSASCOMP." %}

{% translate "Understanding OSASCOMP" %}

{% translate "Each letter in the acronym represents a category of adjective. The order is from most general opinion to most specific, factual detail." %}

{% translate "Letter" %} {% translate "Category" %} {% translate "Example Adjective" %}
O {% translate "Opinion" %} beautiful, ugly, wonderful
S {% translate "Size" %} big, small, huge
A {% translate "Age" %} old, new, ancient
S {% translate "Shape" %} round, square, triangular
C {% translate "Color" %} red, green, black
O {% translate "Origin" %} French, American, Japanese
M {% translate "Material" %} wooden, silk, metal
P {% translate "Purpose" %} sleeping (bag), running (shoes), gardening (tools)

{% translate "Putting It All Together" %}

{% translate "Not every noun phrase will have an adjective from each category, but when they do, this is the order they should follow. The most common error is mixing up the order of Opinion and Size/Age." %}

  • {% translate "Correct:" %} a beautiful big old house.
  • {% translate "Incorrect:" %} a big beautiful old house.

{% translate "Remember, an article always comes before the entire string of adjectives." %}
