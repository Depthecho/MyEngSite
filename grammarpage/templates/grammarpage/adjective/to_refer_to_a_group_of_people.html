{% translate "Adjectives Used as Nouns: To Refer to a Group of People" %}

{% translate "In English, a specific group of people who share a common characteristic can be referred to by using the definite article " %}the{% translate " followed by an adjective. In this usage, the adjective functions as a plural noun." %}

{% translate "1. The Basic Rule" %}

{% translate "The structure is simple: " %}the + adjective{% translate ". The resulting phrase acts as a plural noun, so it is always used with a plural verb." %}

{% translate "Adjective" %} {% translate "Phrase" %} {% translate "Example Sentence" %}
poor the poor The poor are often forgotten by society.
rich the rich A tax cut for the rich.
elderly the elderly Special assistance is available for the elderly.

{% translate "2. Specific vs. General Reference" %}

{% translate "This usage is for referring to people in a general sense, as a class or group. It is a concise way to describe people without having to say '" %}the rich people{% translate "' or '" %}the poor people{% translate "'." %}

  • Correct: The brave fight for freedom.
  • Correct: The brave soldiers fight for freedom.

{% translate "The first example uses '" %}the brave{% translate "' as a plural noun, while the second uses the adjective '" %}brave{% translate "' to modify the noun '" %}soldiers{% translate "'." %}

{% translate "3. Adjectives for Nationalities" %}

{% translate "Many adjectives referring to nationalities can also be used this way, especially those ending in " %}-sh, -ch, -ese{% translate "." %}

  • the British
  • the Spanish
  • the Chinese

{% translate "Note: Not all nationality adjectives follow this rule. For example, you would not say " %}the Americans{% translate " but rather '" %}Americans{% translate "' or '" %}the American people{% translate "'." %}
