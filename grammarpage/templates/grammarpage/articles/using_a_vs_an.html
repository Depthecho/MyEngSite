{% translate "Using 'A' vs. 'An'" %}

{% translate "The choice between the indefinite articles a and an is determined by the sound of the word that follows the article, not the letter it starts with. This is a simple but crucial rule to get right for natural-sounding English." %}

{% translate "Rule: Based on Sound, not Spelling" %}

{% translate "Use a when the next word starts with a consonant sound. Use an when the next word starts with a vowel sound." %}

{% translate "Article" %} {% translate "Rule" %} {% translate "Examples" %}
a {% translate "Use before a consonant sound." %} a book, a cat, a dog, a university (starts with a 'y' sound), a European (starts with a 'y' sound).
an {% translate "Use before a vowel sound." %} an apple, an elephant, an hour (the 'h' is silent), an honest person (the 'h' is silent).

{% translate "Common Exceptions and Tricky Words" %}

{% translate "Some words can be confusing because their spelling doesn't match their pronunciation. Remember to listen to the sound of the first syllable." %}

  • {% translate "Words starting with 'u' or 'eu':" %}
    {% translate "If they sound like 'yoo,' use a. If they sound like 'uh' or 'ew,' use an." %}
    {% translate "Examples:" %} a university, a unique idea. vs. an umbrella, an unwritten rule.
  • {% translate "Words starting with 'h':" %}
    {% translate "If the 'h' is silent, use an. If the 'h' is pronounced, use a." %}
    {% translate "Examples:" %} an hour, an honor. vs. a horse, a happy day.
