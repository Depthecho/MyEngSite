{% translate "Introduction to Articles" %}

{% translate "An article is a word that is used with a noun to specify its grammatical definiteness, whether it is specific or non-specific. English has three articles: a, an (indefinite articles), and the (the definite article). In some cases, no article is used (known as the zero article)." %}

{% translate "What Articles Do" %}

{% translate "Articles are essential because they tell the listener or reader whether the noun being referred to is a specific item or a general one." %}

  • {% translate "Indefinite Article (a/an):" %}
    {% translate "Refers to a non-specific or general noun. It's used when we talk about something for the first time." %}
    I saw a dog. ({% translate "This could be any dog." %})
  • {% translate "Definite Article (the):" %}
    {% translate "Refers to a specific or already known noun. It's used when we are talking about a specific item that both the speaker and listener know." %}
    I saw the dog. ({% translate "We both know which dog I'm talking about." %})

{% translate "Main Types of Articles" %}

{% translate "Article Type" %} {% translate "Forms" %} {% translate "Usage" %}
{% translate "Indefinite" %} a / an {% translate "Used with singular, countable nouns when the noun is general or mentioned for the first time." %}
{% translate "Definite" %} the {% translate "Used with any noun (singular, plural, countable, uncountable) when the noun is specific or has been previously mentioned." %}
{% translate "Zero Article" %} (no article) {% translate "Used with plural countable nouns and uncountable nouns when speaking generally, or with proper nouns like names of countries, people, and cities." %}
