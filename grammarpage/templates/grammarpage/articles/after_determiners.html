{% translate "Placement of Articles: After Determiners" %}

{% translate "A key rule to remember about articles is that they are a type of determiner. You cannot use two determiners for the same noun at the same time. Therefore, articles are not used after other determiners like possessive adjectives or demonstratives." %}

{% translate "1. With Possessive Adjectives" %}

{% translate "Possessive adjectives (my, your, his, her, its, our, their) already specify which noun is being talked about, so an article is redundant and incorrect." %}

{% translate "Correct" %} {% translate "Incorrect" %}
{% translate "This is my car." %} {% translate "This is the my car." %}
{% translate "She found her keys." %} {% translate "She found a her keys." %}

{% translate "2. With Demonstratives" %}

{% translate "Demonstratives (this, that, these, those) also act as determiners, pointing to a specific noun. Therefore, they cannot be used with an article." %}

{% translate "Correct" %} {% translate "Incorrect" %}
{% translate "I like this book." %} {% translate "I like a this book." %}
{% translate "Please take those boxes." %} {% translate "Please take the those boxes." %}

{% translate "Summary" %}

{% translate "Think of it this way: articles and other determiners are like single-use tickets for a noun. Once you use a ticket (my, this, some, etc.), you cannot use another one (a, an, the) for the same noun. This helps to avoid confusion and keeps your sentences grammatically correct." %}
