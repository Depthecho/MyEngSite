{% translate "Placement of Articles: Before Adjective" %}

{% translate "When a noun is modified by an adjective, the article moves to a position before the adjective. The structure becomes: Article + Adjective + Noun. The article still modifies the noun, but the adjective is placed between them to describe the noun." %}

{% translate "1. With Indefinite Articles ('a' and 'an')" %}

{% translate "Remember that the choice between a and an is based on the sound of the very next word. When an adjective is present, the choice depends on the sound of that adjective, not the noun." %}

{% translate "Correct" %} {% translate "Incorrect" %} {% translate "Explanation" %}
{% translate "I saw a tall man." %} {% translate "I saw an tall man." %} {% translate "The adjective 'tall' starts with a consonant sound, so we use 'a'." %}
{% translate "She is an honest woman." %} {% translate "She is a honest woman." %} {% translate "The adjective 'honest' starts with a vowel sound (silent 'h'), so we use 'an'." %}

{% translate "2. With the Definite Article ('the')" %}

{% translate "The definite article the is always placed at the very beginning of the noun phrase, before any adjectives." %}

{% translate "Structure" %} {% translate "Example" %}
{% translate "the" %} + adjective + noun {% translate "We live in the old house." %}
{% translate "the" %} + adjective + noun {% translate "He returned the blue book to me." %}

{% translate "Summary" %}

{% translate "The general rule remains the same: the article is the first word of the noun phrase. The only thing that changes is that an adjective can be placed between the article and the noun." %}
