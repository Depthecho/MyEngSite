{% translate "This table provides a quick reference guide to the usage of English articles (a, an, the) and when to use no article at all." %}

{% translate "Article" %} {% translate "Usage" %} {% translate "Example" %}
a / an
  • {% translate "For singular, countable nouns." %}
  • {% translate "When the noun is general or non-specific." %}
  • {% translate "When the noun is mentioned for the first time." %}
  • I need a new phone.
  • He is an excellent student.
the
  • {% translate "For specific nouns (singular or plural, countable or uncountable)." %}
  • {% translate "When the noun has been mentioned before." %}
  • {% translate "When there is only one of that item (e.g., the sun)." %}
  • Please close the door.
  • I saw a movie yesterday. The movie was great.
Zero Article
  • {% translate "For plural countable nouns in a general sense." %}
  • {% translate "For uncountable nouns in a general sense." %}
  • {% translate "For most proper nouns (countries, cities, names)." %}
  • Cats are independent pets.
  • I drink coffee every morning.
  • I live in Japan.

{% translate "Quick Tip" %}

{% translate "Think of articles as signposts. A/an points to 'one of many,' while the points to 'the one we both know.' The zero article is used when the noun's identity is not important or it's a general concept." %}
