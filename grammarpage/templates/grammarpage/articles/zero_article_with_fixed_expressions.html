{% translate "The Zero Article: With Fixed Expressions" %}

{% translate "The zero article is a core part of many fixed expressions and common phrases in English. These phrases have to be learned as a whole, as the lack of an article is a characteristic feature." %}

{% translate "1. Means of Transport" %}

{% translate "When talking about modes of transport using the preposition 'by,' no article is used." %}

  • {% translate "We travel by bus." %}
  • {% translate "She goes to work by car." %}
  • {% translate "I prefer to travel by train." %}

{% translate "Note: When you refer to a specific vehicle, you will use an article: 'I got on the bus.'" %}

{% translate "2. Meals" %}

{% translate "We typically do not use an article when talking about meals in a general sense." %}

  • {% translate "What time do you have breakfast?" %}
  • {% translate "We had lunch together yesterday." %}
  • {% translate "They are preparing dinner." %}

{% translate "Note: An exception is made when the meal is described by an adjective or is specific: 'We had a big lunch.' or 'The dinner we had was excellent.'" %}

{% translate "3. Institutions and Places" %}

{% translate "The zero article is used when a noun refers to the purpose of a place, rather than the physical building itself." %}

{% translate "Zero Article (Purpose)" %} {% translate "With 'the' (Physical building)" %}
{% translate "He is in hospital." %} {% translate "I went to the hospital to visit him." %}
{% translate "Children go to school." %} {% translate "The meeting is at the school." %}
{% translate "He is a student at university." %} {% translate "They are building a new library at the university." %}
