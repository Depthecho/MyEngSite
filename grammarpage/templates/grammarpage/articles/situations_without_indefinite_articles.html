{% translate "Situations without Indefinite Articles" %}

{% translate "The indefinite articles a and an are only used with singular, countable nouns. Therefore, there are many situations where they cannot or should not be used." %}

{% translate "1. With Plural Nouns" %}

{% translate "Since a and an mean 'one,' they cannot be used with plural nouns. Instead, we use no article (the zero article) to refer to a general group." %}

{% translate "Correct" %} {% translate "Incorrect" %}
I saw cars. I saw a cars.
They bought tickets. They bought a tickets.

{% translate "2. With Uncountable Nouns" %}

{% translate "Uncountable nouns (e.g., water, information, advice, furniture) cannot be counted individually, so they do not take indefinite articles." %}

  • I need some water. ({% translate "Correct" %})
  • I need a water. ({% translate "Incorrect" %})
  • He gave me some good advice. ({% translate "Correct" %})
  • He gave me an advice. ({% translate "Incorrect" %})

{% translate "3. With Names and Titles" %}

{% translate "When referring to a specific person or a unique title, no indefinite article is used." %}

  • This is Mary.
  • President Biden gave a speech.

{% translate "4. With Possessive Adjectives and Demonstratives" %}

{% translate "You cannot use an indefinite article with a noun that is already preceded by a possessive adjective (my, your, his) or a demonstrative (this, that, these, those), as these words already specify the noun." %}

{% translate "Correct" %} {% translate "Incorrect" %}
My car is red. A my car is red.
Is this an umbrella? Is a this umbrella?
