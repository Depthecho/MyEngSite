{% translate "Key Uses of Indefinite Articles" %}

{% translate "The indefinite articles a and an have several important functions in English. They are used to introduce new information, refer to a non-specific item, and describe people's professions, among other things." %}

{% translate "1. Introducing a New or Non-Specific Noun" %}

{% translate "This is the most common use of indefinite articles. When you mention a singular, countable noun for the first time, or when you are not referring to a particular one, you use a or an." %}

{% translate "Rule" %} {% translate "Example" %}
{% translate "First mention of a noun." %} {% translate "I saw a cat and a dog in the park." %}
{% translate "Refers to any single item from a group." %} {% translate "Can I have a cup of coffee, please?" %}

{% translate "2. With Professions and Roles" %}

{% translate "We use indefinite articles to state someone's profession, job, or role. This is a crucial distinction and a very common usage." %}

  • {% translate "He is a doctor." %}
  • {% translate "She wants to be an artist." %}

{% translate "3. In Exclamations" %}

{% translate "Indefinite articles are used in exclamations with 'what' and 'such' followed by a singular countable noun." %}

  • {% translate "What a beautiful day!" %}
  • {% translate "It was such a good movie!" %}

{% translate "4. With Numerical Expressions" %}

{% translate "The indefinite article is used to mean 'one' in certain numerical expressions and phrases." %}

{% translate "Phrase" %} {% translate "Example" %}
{% translate "a hundred, a thousand" %} {% translate "It costs a hundred dollars." %}
{% translate "a dozen, a couple" %} {% translate "I need a dozen eggs." %}
