{% translate "Pronouns: Indefinite Pronouns" %}

{% translate "Indefinite pronouns refer to nonspecific people, places, or things. They do not refer to a particular person, place, or thing. They can be singular, plural, or both, which affects the verb they take." %}

{% translate "1. Common Indefinite Pronouns" %}

{% translate "Indefinite pronouns can be grouped into several categories based on their meaning." %}

{% translate "Category" %} {% translate "Pronouns" %} {% translate "Example" %}
{% translate "Singular" %} another, anybody, anything, everyone, nobody, someone, something Everyone is here.
{% translate "Plural" %} both, few, many, others, several Many are called, but few are chosen.
{% translate "Singular or Plural" %} all, any, more, most, none, some Most of the work is finished. Most of the students are here.

{% translate "2. Singular Indefinite Pronouns and Verb Agreement" %}

{% translate "Most indefinite pronouns that end in -one, -body, or -thing (e.g., everyone, somebody, nothing) are always singular and therefore require a singular verb." %}

  • Everyone loves pizza. (not love)
  • Something is wrong with this computer. (not are)

{% translate "3. Plural Indefinite Pronouns" %}

{% translate "Pronouns like both, few, many, others, and several are always plural and require a plural verb." %}

  • Both of the children are playing.
  • Several students have arrived.

{% translate "4. The 'Either'/'Or' Conundrum" %}

{% translate "When using pronouns that can be singular or plural, the verb's number depends on the noun or pronoun it refers to." %}

  • All of the cake is gone.
  • All of the cookies are gone.

{% translate "When the pronoun refers to a countable noun (cookies), it is plural. When it refers to an uncountable noun (cake), it is singular." %}
