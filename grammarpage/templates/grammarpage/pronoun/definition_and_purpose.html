{% translate "Pronouns: Definition and Purpose" %}

{% translate "A pronoun is a word that replaces a noun or a noun phrase. The word that the pronoun replaces is called its antecedent. The primary purpose of pronouns is to avoid repetition, making sentences and paragraphs flow more smoothly." %}

{% translate "1. The Core Purpose: Avoiding Repetition" %}

{% translate "Consider the following sentences:" %}

  • Incorrect: John went to the store. John bought bread. John came home.
  • Correct: John went to the store. He bought bread. He came home.

{% translate "In the second example, the pronoun He replaces the noun John, making the text much more natural and easy to read. The noun John is the antecedent for the pronoun He." %}

{% translate "2. Types of Pronouns" %}

{% translate "There are several categories of pronouns, each with a specific function. The main types include:" %}

{% translate "Type of Pronoun" %} {% translate "Function" %} {% translate "Examples" %}
{% translate "Personal" %} {% translate "Refers to people or things (I, you, he, she, it, we, they)." %} She is a great singer.
{% translate "Possessive" %} {% translate "Shows ownership (mine, yours, his, hers, its, ours, theirs)." %} That book is mine.
{% translate "Demonstrative" %} {% translate "Points to specific things (this, that, these, those)." %} I like this dress.
{% translate "Interrogative" %} {% translate "Asks questions (who, whom, whose, which, what)." %} Who is at the door?
