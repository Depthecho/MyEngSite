{% translate "Pronouns: Grammatical Features (Case, Number, etc.)" %}

{% translate "Pronouns in English are not static words; they change their form based on several grammatical features. The most important of these are person, number, gender, and case. Understanding these features is essential for using pronouns correctly." %}

{% translate "1. Case: Subject vs. Object" %}

{% translate "Case refers to the function of the pronoun in a sentence. There are two main cases for personal pronouns:" %}

  • {% translate "Subjective Case" %}: {% translate "Used when the pronoun is the subject of the sentence (the one performing the action)." %}
  • {% translate "Objective Case" %}: {% translate "Used when the pronoun is the object of the sentence (the one receiving the action)." %}
{% translate "Subjective" %} {% translate "Objective" %} {% translate "Example" %}
I me I saw him.
She her She gave the book to me.
They them They helped us.

{% translate "2. Number: Singular vs. Plural" %}

{% translate "Pronouns can be either singular (referring to one) or plural (referring to more than one). It's crucial to match the pronoun's number to its antecedent." %}

{% translate "Singular" %} {% translate "Plural" %} {% translate "Example" %}
I we I went to the store. We all went to the store.
he/she/it they He is a doctor. They are doctors.

{% translate "3. Person: First, Second, and Third" %}

{% translate "Person indicates who is speaking, who is being spoken to, or who is being spoken about." %}

  • {% translate "First Person" %}: {% translate "The speaker (I, we)." %}
  • {% translate "Second Person" %}: {% translate "The person or people being spoken to (you)." %}
  • {% translate "Third Person" %}: {% translate "The person or thing being spoken about (he, she, it, they)." %}

{% translate "4. Gender" %}

{% translate "Gender-specific pronouns are used for the third person singular." %}

  • He/him/his: {% translate "Masculine gender." %}
  • She/her/hers: {% translate "Feminine gender." %}
  • It/its: {% translate "Neutral or inanimate objects." %}

{% translate "The pronoun they/them/their is now also commonly used as a singular, gender-neutral pronoun." %}
