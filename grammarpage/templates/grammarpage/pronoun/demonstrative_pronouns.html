{% translate "Pronouns: Demonstrative Pronouns" %}

{% translate "Demonstrative pronouns are used to point to or identify a noun or a group of nouns. They indicate the distance of the item(s) from the speaker, in terms of both physical and temporal distance." %}

{% translate "1. The Four Demonstrative Pronouns" %}

{% translate "There are four main demonstrative pronouns, which are categorized by number (singular or plural) and distance (near or far)." %}

{% translate "Number" %} {% translate "Near to Speaker" %} {% translate "Far from Speaker" %}
{% translate "Singular" %} {% translate "this" %} {% translate "that" %}
{% translate "Plural" %} {% translate "these" %} {% translate "those" %}

{% translate "These pronouns can refer to both tangible objects and abstract concepts." %}

{% translate "2. Examples of Usage" %}

{% translate "Demonstrative pronouns stand alone in a sentence, replacing a noun. They do not modify a noun, which would make them demonstrative adjectives." %}

  • {% translate "This" %}: {% translate "used for a singular noun that is near." %}
    {% translate "This is my favorite song. (I'm listening to it now.)" %}
  • {% translate "That" %}: {% translate "used for a singular noun that is far." %}
    {% translate "That was a difficult year. (referring to a time in the past.)" %}
  • {% translate "These" %}: {% translate "used for plural nouns that are near." %}
    {% translate "These are the pictures from our trip." %}
  • {% translate "Those" %}: {% translate "used for plural nouns that are far." %}
    {% translate "Those are the stars I was telling you about." %}

{% translate "3. Demonstrative Pronouns vs. Adjectives" %}

{% translate "It's important to distinguish between demonstrative pronouns and demonstrative adjectives. The word is the same, but its function in the sentence is different." %}

  • {% translate "Pronoun" %}: {% translate "replaces a noun. The word stands alone." %}
    {% translate "This is my car." %}
  • {% translate "Adjective" %}: {% translate "modifies a noun. The word is followed by a noun." %}
    {% translate "This car is mine." %}

{% translate "In the first example, This is a pronoun standing in for the car. In the second, This is an adjective modifying the noun car." %}
