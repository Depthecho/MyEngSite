{% translate "Pronouns: Interrogative and Relative Pronouns" %}

{% translate "Interrogative and relative pronouns are two distinct types of pronouns that often use the same words (e.g., who, whom, whose, which, that). Their function, however, is entirely different." %}

{% translate "1. Interrogative Pronouns" %}

{% translate "Interrogative pronouns are used to ask questions. They stand in for the noun about which the question is being asked. The most common interrogative pronouns are:" %}

  • Who: {% translate "Refers to a person (subject)." %}
    Who is coming to the party?
  • Whom: {% translate "Refers to a person (object)." %}
    Whom did you see at the store?
  • Whose: {% translate "Asks about possession." %}
    Whose car is parked outside?
  • Which: {% translate "Refers to a choice between a limited number of things or people." %}
    Which of these books do you want?
  • What: {% translate "Refers to things or ideas in a general sense." %}
    What happened?

{% translate "An interrogative pronoun always begins a question and is followed by a question mark." %}

{% translate "2. Relative Pronouns" %}

{% translate "Relative pronouns connect a relative clause to the main clause of a sentence. They refer to a noun (the antecedent) mentioned earlier in the sentence. The most common relative pronouns are:" %}

  • Who / Whom / Whose: {% translate "Refer to people." %}
    The man who called is my boss.
    The woman whom I saw was happy.
  • Which: {% translate "Refers to things or animals." %}
    The book which is on the table is mine.
  • That: {% translate "Can refer to people, things, or animals." %}
    The car that I bought is fast.

{% translate "Relative pronouns act as a bridge, linking two parts of a sentence together and making the sentence more detailed and complex." %}

{% translate "3. Interrogative vs. Relative: A Simple Test" %}

{% translate "The key difference is their function in a sentence. Here is a simple way to tell them apart:" %}

  • {% translate "If the pronoun is asking a question, it's interrogative." %}
  • {% translate "If the pronoun is connecting a clause and referring to a noun, it's relative." %}
