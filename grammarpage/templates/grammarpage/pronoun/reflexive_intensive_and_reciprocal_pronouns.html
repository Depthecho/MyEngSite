{% translate "Pronouns: Reflexive, Intensive, and Reciprocal Pronouns" %}

{% translate "While some pronouns replace nouns, others serve more specialized functions, such as referring back to the subject or emphasizing a noun. This page covers reflexive, intensive, and reciprocal pronouns, which are often confused but serve distinct purposes." %}

{% translate "1. Reflexive Pronouns" %}

{% translate "A reflexive pronoun is used as a direct or indirect object when the object is the same person or thing as the subject of the verb. It 'reflects' the action back to the subject. They are formed by adding -self (singular) or -selves (plural) to a personal pronoun." %}

{% translate "Singular" %} {% translate "Plural" %}
myself ourselves
yourself yourselves
himself, herself, itself themselves

{% translate "Example: He hurt himself while cooking." %}
{% translate "In this sentence, He is both the subject and the object of the action." %}

{% translate "2. Intensive Pronouns" %}

{% translate "Intensive pronouns have the exact same form as reflexive pronouns but are used for a different purpose: to add emphasis to a noun or pronoun that is already stated. They are not essential to the meaning of the sentence and can be removed without changing the core meaning." %}

{% translate "Example: The president herself announced the news." %}
{% translate "The pronoun herself emphasizes that the president, and no one else, made the announcement. The sentence still makes sense without it: The president announced the news." %}

{% translate "A common error is using intensive pronouns in place of personal pronouns, for example, My friend and myself went to the store. The correct form is My friend and I went to the store." %}

{% translate "3. Reciprocal Pronouns" %}

{% translate "Reciprocal pronouns are used to show that two or more people or things are performing the same action to or for each other. There are only two reciprocal pronouns in English:" %}

  • each other: {% translate "Used for two people or things." %}
  • one another: {% translate "Used for more than two people or things." %}

{% translate "Examples:" %}

  • John and Mary gave gifts to each other.
  • The members of the team respect one another.
