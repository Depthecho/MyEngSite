{% translate "The Attribute in English Grammar" %}

{% translate "An attribute is a word, phrase, or clause that modifies a noun or pronoun by providing additional information about its qualities, characteristics, or identity." %}

{% translate "Core Characteristics" %}

[Attribute] + [Noun] OR [Noun] + [Attribute]

{% translate "Key Functions:" %}

  • {% translate "Describes qualities (color, size, shape, etc.)" %}
  • {% translate "Specifies quantity or number" %}
  • {% translate "Indicates possession or ownership" %}
  • {% translate "Provides classifying information" %}
  • {% translate "Expresses subjective evaluation" %}

{% translate "Examples:" %}

  • The three cats slept peacefully. (quantity)
  • She wore a blue dress. (quality)
  • Shakespeare's sonnets are famous. (possession)

{% translate "Types of Attributes" %}

{% translate "1. Adjectives" %}

{% translate "Single-word descriptors before nouns:" %}

  • A beautiful sunset
  • An interesting book

{% translate "2. Noun Modifiers" %}

{% translate "Nouns functioning as adjectives:" %}

  • Computer science
  • Chicken soup

{% translate "3. Participial Phrases" %}

{% translate "Verbs in participle form (-ing or -ed) as modifiers:" %}

  • The barking dog
  • A broken window

{% translate "4. Prepositional Phrases" %}

{% translate "Phrases beginning with prepositions:" %}

  • The man in the corner
  • A book about whales

{% translate "5. Relative Clauses" %}

{% translate "Clauses beginning with relative pronouns:" %}

  • The person who called you
  • The ideas that we discussed

{% translate "Special Cases and Challenges" %}

{% translate "1. Adjective Order" %}

{% translate "English follows a specific sequence for multiple adjectives:" %}

  1. Opinion (beautiful, ugly)
  2. Size (large, small)
  3. Age (young, old)
  4. Shape (round, square)
  5. Color (red, blue)
  6. Origin (French, solar)
  7. Material (wooden, silk)
  8. Purpose (sleeping bag, frying pan)

A lovely little old rectangular green French silver whittling knife

{% translate "2. Restrictive vs Non-restrictive" %}

  • {% translate "Restrictive: Essential for identifying the noun (no commas)" %}

    Students who cheat will fail.

  • {% translate "Non-restrictive: Additional information (set off with commas)" %}

    Mr. Smith, who lives next door, is a doctor.

{% translate "3. Common Errors" %}

  • {% translate "Misplaced modifiers:" %}

    Wrong: She served sandwiches to the children on paper plates.
    Right: She served the children sandwiches on paper plates.

  • {% translate "Dangling modifiers:" %}

    Wrong: Walking to school, the bus passed by.
    Right: Walking to school, I saw the bus pass by.

  • {% translate "Adjective/adverb confusion:" %}

    Wrong: She feels badly.
    Right: She feels bad.
