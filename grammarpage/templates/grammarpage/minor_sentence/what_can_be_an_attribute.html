{% translate "Grammatical Forms That Can Function as Attributes" %}

{% translate "Attributes can be expressed through various grammatical forms, each providing different types of information about the nouns they modify." %}

{% translate "1. Adjectives" %}

[Adjective] + [Noun]

{% translate "Characteristics:" %}

  • {% translate "Most common attributive form" %}
  • {% translate "Typically precede the noun they modify" %}
  • {% translate "Can be gradable (comparative/superlative forms)" %}
  • {% translate "Can be compound (dark-blue, ice-cold)" %}

{% translate "Examples:" %}

  • A happy child
  • The tallest building
  • An extremely difficult problem

{% translate "Special Cases:" %}

  • {% translate "Postpositive adjectives: Some adjectives follow the noun" %}

    The president elect

  • {% translate "Adjectives beginning with 'a':" %}

    The child asleep

{% translate "2. Nouns and Noun Phrases" %}

{% translate "A. Possessive Nouns" %}

[Noun+'s] + [Noun]
  • Mary's car
  • The company's policy

{% translate "B. Noun Modifiers" %}

[Noun] + [Noun]
  • Chocolate cake
  • Computer science

{% translate "C. Compound Nouns" %}

[Noun]+[Noun]
  • Toothbrush
  • Swimming pool

{% translate "3. Participles" %}

{% translate "A. Present Participles (-ing)" %}

[Verb+ing] + [Noun]
  • A barking dog
  • The setting sun

{% translate "B. Past Participles (-ed/irregular)" %}

[Verb+ed/irregular] + [Noun]
  • A broken window
  • The stolen jewelry

{% translate "4. Prepositional Phrases" %}

[Noun] + [Preposition + Noun]

{% translate "Characteristics:" %}

  • {% translate "Always follow the noun they modify" %}
  • {% translate "Can be restrictive or non-restrictive" %}
  • {% translate "Often equivalent to relative clauses" %}

{% translate "Examples:" %}

  • The book on the shelf
  • A woman of great wisdom
  • The house across the street

{% translate "5. Other Attributive Forms" %}

{% translate "A. Relative Clauses" %}

  • The man who lives next door
  • The ideas that we discussed

{% translate "B. Infinitives" %}

  • The best way to learn
  • A book to read

{% translate "C. Adverbs" %}

  • The people here
  • The meeting yesterday

{% translate "D. Whole Sentences" %}

  • A do-it-yourself project
  • The I-told-you-so look
