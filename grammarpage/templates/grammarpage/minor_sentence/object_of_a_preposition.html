{% translate "Object of a Preposition" %}

{% translate "The object of a preposition is the noun, pronoun, or noun phrase that follows a preposition to complete its meaning and form a prepositional phrase." %}

{% translate "Core Structure" %}

[Preposition] + [Object] = Prepositional Phrase

{% translate "Key Characteristics:" %}

  • {% translate "Always follows a preposition (in, on, at, with, to, etc.)" %}
  • {% translate "Can be a noun, pronoun, gerund, or noun clause" %}
  • {% translate "Never a verb in its base form" %}
  • {% translate "Answers what? or whom? after the preposition" %}

{% translate "Examples:" %}

  • She sat on the bench.
  • We talked about her promotion.
  • He's interested in learning French. (gerund as object)

{% translate "Types of Prepositional Objects" %}

{% translate "1. Noun Objects" %}

{% translate "The most common type of prepositional object:" %}

  • The keys are in the drawer.
  • They arrived after midnight.

{% translate "2. Pronoun Objects" %}

{% translate "Must be in the objective case:" %}

  • This gift is for you.
  • She stood near them.

{% translate "3. Gerund Objects" %}

{% translate "Verbs in -ing form functioning as nouns:" %}

  • He's good at swimming.
  • Before leaving, she turned off the lights.

{% translate "4. Noun Clause Objects" %}

{% translate "Whole clauses functioning as objects:" %}

  • We disagreed about what needed to be done.
  • It depends on whether they'll agree.

{% translate "Grammatical Functions" %}

{% translate "Function" %} {% translate "Example" %} {% translate "Explanation" %}
Adverbial She slept during the meeting. {% translate "Modifies the verb (when she slept)" %}
Adjectival The man with the hat waved. {% translate "Modifies the noun (which man)" %}
Nominal After lunch is my free time. {% translate "Functions as a noun (subject of sentence)" %}

{% translate "Special Cases and Challenges" %}

{% translate "1. Multiple Prepositional Phrases" %}

{% translate "Sentences can contain chains of prepositional phrases:" %}

The book on the shelf in the corner of the room is mine.

{% translate "2. Preposition Stranding" %}

{% translate "When the object moves but leaves the preposition behind:" %}

  • Standard: This is the book about which I told you.
  • Stranded: This is the book which I told you about.

{% translate "3. Common Errors" %}

  • {% translate "Incorrect pronoun case:" %}

    Wrong: Between you and I
    Right: Between you and me

  • {% translate "Missing objects:" %}

    Wrong: She's looking forward to.
    Right: She's looking forward to your visit.

  • {% translate "Wrong prepositions:" %}

    Wrong: We're waiting on the bus.
    Right: We're waiting for the bus.
