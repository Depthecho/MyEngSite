{% translate "Cardinal Numerals Functioning as Nouns" %}

{% translate "While cardinal numerals typically modify nouns as adjectives, they can also serve as nouns themselves in various contexts. This grammatical flexibility allows for more nuanced expression in English." %}

{% translate "1. Numerals as Independent Digits and Figures" %}

{% translate "When referring to numbers as abstract concepts or physical representations, cardinals function as nouns and can be pluralized." %}

{% translate "Context" %} {% translate "Example" %} {% translate "Explanation" %}
{% translate "Abstract numbers" %} Seven is a prime number {% translate "Referring to the number itself as a concept" %}
{% translate "Physical digits" %} The clock showed three threes (3:33) {% translate "Referring to the digit representations" %}
{% translate "Game scores" %} He rolled three sixes in a row {% translate "Referring to multiple instances of the number" %}

{% translate "Special Cases with Digits" %}

  • {% translate "Binary digits:" %} The code contains five ones and three zeros
  • {% translate "Phone numbers:" %} My number ends with two eights
  • {% translate "Mathematical operations:" %} Add the two fives first

{% translate "2. Approximate Large Quantities" %}

{% translate "Pluralized cardinal numbers can express indefinite large quantities, often with an implied sense of approximation." %}

{% translate "Quantity" %} {% translate "Example" %} {% translate "Implied Meaning" %}
hundreds Hundreds attended the rally 200-900 {% translate "approximately" %}
thousands Thousands protested downtown 2,000-9,000 {% translate "approximately" %}
millions Millions watched the event 2,000,000+ {% translate "approximately" %}

{% translate "Intensifying Expressions" %}

  • {% translate "With 'of':" %} Tens of thousands celebrated
  • {% translate "With adjectives:" %} Countless hundreds gathered
  • {% translate "Compound forms:" %} Hundreds upon hundreds applied

{% translate "Important Notes" %}

  • {% translate "Don't use articles with these plural forms: a hundreds → hundreds" %}
  • {% translate "Smaller numbers (below 100) typically don't work this way: tens of people (unless in 'tens of thousands')" %}
  • {% translate "These forms imply approximation - for exact counts, use the adjective form: three hundred people" %}

{% translate "3. Pronoun-like Numerical Nouns" %}

{% translate "Numerals can replace previously mentioned nouns, functioning similarly to pronouns in context." %}

{% translate "Common Usage Patterns" %}

  • {% translate "After quantity specification:" %} We have ten tickets - I need two
  • {% translate "In comparisons:" %} She bought five books but read only three
  • {% translate "With implied objects:" %} Take three (cookies) if you want

{% translate "Special Pronoun Cases" %}

{% translate "Type" %} {% translate "Example" %} {% translate "Function" %}
{% translate "Subject" %} Three of the apples were rotten {% translate "Replaces 'apple' as subject" %}
{% translate "Object" %} I'll take two {% translate "Replaces understood object" %}
{% translate "Possessive" %} The five's combined value {% translate "Shows possession" %}
