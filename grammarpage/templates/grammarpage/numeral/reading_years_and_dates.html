{% translate "Reading Years and Dates in English" %}

{% translate "Understanding how to properly read years and dates is essential for clear communication in English. This guide covers all the conventions and special cases you need to know." %}

{% translate "1. Reading Years" %}

{% translate "English has specific patterns for reading years that vary depending on the century and digit configuration." %}

{% translate "Four-Digit Years" %}

{% translate "Most four-digit years are read as two two-digit numbers:" %}

  • 1999: nineteen ninety-nine
  • 1845: eighteen forty-five
  • 1066: ten sixty-six ({% translate "Battle of Hastings" %})

{% translate "21st Century Years" %}

{% translate "Years from 2000-2099 have two common reading patterns:" %}

  • 2005: two thousand five ({% translate "more common" %}) or twenty oh-five
  • 2023: two thousand twenty-three or twenty twenty-three

{% translate "Special Year Cases" %}

  • {% translate "Years ending in 00:" %} 1900 - nineteen hundred
  • {% translate "The year 2000:" %} 2000 - two thousand ({% translate "never \"twenty hundred\"" %})
  • {% translate "Years before 1000:" %} 876 - eight seventy-six or eight hundred seventy-six

{% translate "2. Reading Full Dates" %}

{% translate "Date formats differ between American and British English, affecting how they're read aloud." %}

{% translate "Format" %} {% translate "Written" %} {% translate "Spoken" %}
{% translate "American English (Month-Day-Year)" %} March 10, 2024 {% translate "March tenth, twenty twenty-four" %}
{% translate "British English (Day-Month-Year)" %} 10 March 2024 {% translate "the tenth of March twenty twenty-four" %}

{% translate "Ordinal Numbers in Dates" %}

{% translate "Days are always read as ordinal numbers, even when written as cardinals:" %}

  • May 1 → May first ({% translate "not \"May one\"" %})
  • July 4 → July fourth (U.S. Independence Day)
  • November 11 → November eleventh (Veterans Day)

{% translate "3. Special Date Formats" %}

{% translate "Decades" %}

{% translate "Decades are typically referred to in these patterns:" %}

  • 1990s → the nineties or the nineteen nineties
  • 2000s → the two thousands
  • 2010s → the twenty-tens

{% translate "Centuries" %}

{% translate "Centuries are ordinal numbers one higher than the year's first digits:" %}

  • 1800-1899 → the nineteenth century
  • 2000-2099 → the twenty-first century

{% translate "Historical Events" %}

{% translate "Some historical events have special names:" %}

  • 9/11 → nine eleven ({% translate "September 11 attacks" %})
  • 7/7 → seven seven ({% translate "London bombings" %})

{% translate "Important Notes" %}

  • {% translate "Never use \"and\" when reading years: 1995 is nineteen ninety-five, not \"nineteen and ninety-five\"" %}
  • {% translate "For years 2001-2009, \"two thousand\" is more common than \"twenty oh\"" %}
  • {% translate "In formal writing, always write dates with the year: January 1, 2023 (not just \"January 1\")" %}
  • {% translate "When writing dates numerically, remember the format differences: 01/02/2023 is January 2 in US, February 1 in UK" %}
