{% translate "Ordinal Numerals: Definition and Formation" %}

{% translate "Ordinal numerals indicate position or order in a sequence, answering the question 'which one?' They are essential for describing rankings, dates, floors in buildings, and competition results." %}

{% translate "1. Key Characteristics" %}

  • {% translate "Always modify nouns (function as adjectives)" %}
  • {% translate "Can be written as words (third) or numerals with suffixes (3rd)" %}
  • {% translate "Typically precede the noun they modify" %}
  • {% translate "Often used with definite articles (the first, the second)" %}
  • {% translate "Formed differently from cardinal numbers" %}

{% translate "2. Formation Rules" %}

{% translate "Number Range" %} {% translate "Pattern" %} {% translate "Examples" %} {% translate "Notes" %}
1-3 {% translate "Irregular" %} first, second, third {% translate "Must be memorized" %}
4-20 {% translate "Add -th" %} fourth, eleventh, twentieth {% translate "Note: fifth (not fiveth), twelfth (not twelveth)" %}
21-99 {% translate "Tens + ordinal unit" %} twenty-first, thirty-second {% translate "Hyphenate between tens and unit" %}
100+ {% translate "Cardinal + ordinal" %} one hundred first, two hundred twenty-second {% translate "No hyphen after hundred" %}

{% translate "Special Spelling Cases" %}

  • {% translate "five → fifth (ve → f)" %}
  • {% translate "eight → eighth (drop one t)" %}
  • {% translate "nine → ninth (drop e)" %}
  • {% translate "twelve → twelfth (ve → f)" %}
  • {% translate "twenty → twentieth (y → ie)" %}

{% translate "3. Written Forms" %}

{% translate "Numerical Representation" %}

  • {% translate "Use last two letters of written word:" %}

    1st, 2nd, 3rd, 4th, 21st, 32nd, 43rd, 100th

  • {% translate "In dates, always use this form:" %}

    May 1st, November 22nd

  • {% translate "In formal writing, often spelled out:" %}

    the first day, the twenty-second of November

{% translate "When to Spell Out vs. Use Numerals" %}

  • {% translate "Spell out in formal writing and at sentence beginnings" %}

    {% translate "Twenty-fifth Anniversary" %}

  • {% translate "Use numerals in dates, addresses, and footnotes" %}

    {% translate "January 15th, 2023" %}

  • {% translate "Use numerals for large ordinals" %}

    {% translate "the 125th edition" %}
