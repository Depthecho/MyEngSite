{% translate "Ordinal Numerals: Spelling Rules and Exceptions" %}

{% translate "Mastering ordinal numeral spelling requires understanding both the regular patterns and important exceptions. These rules are essential for proper writing in academic, professional, and formal contexts." %}

{% translate "1. Standard Formation Rules" %}

{% translate "Basic '-th' Suffix Rule" %}

  • {% translate "For most numbers, add -th to the cardinal form:" %}

    four → fourth
    ten → tenth
    sixteen → sixteenth

  • {% translate "Numbers ending in -y change to -ieth:" %}

    twenty → twentieth
    fifty → fiftieth

  • {% translate "Numbers ending in -ve change to -fth:" %}

    five → fifth
    twelve → twelfth

{% translate "Pronunciation Patterns" %}

  • {% translate "The -th ending is always unvoiced (/θ/)" %}
  • {% translate "Stress typically remains on the same syllable as the cardinal number" %}
  • {% translate "Compound ordinals have secondary stress on both parts" %}

    twenty-FIFTH
    thirty-SECOND

{% translate "2. Key Exceptions and Irregular Forms" %}

{% translate "Cardinal" %} {% translate "Ordinal" %} {% translate "Spelling Change" %} {% translate "Notes" %}
one first {% translate "Complete change" %} {% translate "Not 'oneth'" %}
two second {% translate "Complete change" %} {% translate "Not 'twoth'" %}
three third {% translate "Drop -ee, add -ird" %} {% translate "Not 'threeth'" %}
five fifth {% translate "ve → f, add -th" %} {% translate "Not 'fiveth'" %}
eight eighth {% translate "Add -h only" %} {% translate "One 't' disappears" %}
nine ninth {% translate "Drop -e" %} {% translate "Not 'nineth'" %}
twelve twelfth {% translate "ve → f, add -th" %} {% translate "Not 'twelveth'" %}

{% translate "Compound Number Exceptions" %}

  • {% translate "Only the last number becomes ordinal:" %}

    twenty-one → twenty-first
    fifty-three → fifty-third

  • {% translate "Exception for 11-13 (use -th):" %}

    eleven → eleventh
    twelve → twelfth
    thirteen → thirteenth

2 2nd second 2nd floor
3 3rd third 3rd attempt

{% translate "Writing Style Guidelines" %}

  • {% translate "In formal writing, spell out ordinals under 10:" %}

    the third chapter ({% translate "not 3rd chapter" %})

  • {% translate "Use numerals with suffixes for dates and rankings:" %}

    May 15th
    42nd President

  • {% translate "Superscripts are acceptable in informal contexts:" %}

    1st, 2nd, 3rd

{% translate "4. Common Mistakes to Avoid" %}

  • {% translate "Incorrect double ordinalization" %} {% translate "in compounds:" %}

    Incorrect: twenty-oneth
    Correct: twenty-first

  • {% translate "Misspelling irregular forms" %}:

    Incorrect: nineth, twelveth, fiveth
    Correct: ninth, twelfth, fifth

  • {% translate "Incorrect abbreviations" %}:

    Incorrect: 22th, 43th
    Correct: 22nd, 43rd

  • {% translate "Wrong suffix after hundred/thousand" %}:

    Incorrect: one hundredth first
    Correct: one hundred first
