{% translate "Ordinal Numerals: Usage with Articles and Prepositions" %}

{% translate "Proper use of articles and prepositions with ordinal numerals is essential for precise communication. These grammatical elements help establish clear relationships between items in a sequence." %}

{% translate "1. Article Usage with Ordinals" %}

{% translate "Definite Article 'the'" %}

  • {% translate "Used in 99% of cases with ordinals" %}
  • {% translate "Specifies exact position in a sequence" %}

    She won the first prize.

  • {% translate "Required with superlatives" %}

    This is the second largest city.

  • {% translate "Used in dates" %}

    the fifteenth of March

{% translate "Indefinite Article 'a/an'" %}

  • {% translate "Used when referring to another item in sequence" %}

    We need a third opinion.

  • {% translate "Indicates non-specific order" %}

    He made a second attempt.

  • {% translate "Used in expressions of approximation" %}

    This happened a fifth time.

{% translate "Context" %} {% translate "Article" %} {% translate "Example" %}
{% translate "Specific position" %} the the third chapter
{% translate "Another in sequence" %} a/an a third option
{% translate "Superlatives" %} the the second tallest
{% translate "Approximation" %} a/an a fifth time

{% translate "Special Prepositional Phrases" %}

  • {% translate "For the first/second time" %}

    I visited Paris for the first time last year.

  • {% translate "In the first/second place" %}

    In the first place, we should consider costs.

  • {% translate "On the one hand... on the other hand" %}

    On the one hand it's cheap, on the other hand it's low quality.

{% translate "3. Common Mistakes and Special Cases" %}

  • {% translate "Omitting 'the' with ordinals" %}

    Incorrect: He won first prize.
    Correct: He won the first prize.

  • {% translate "Confusing 'in' and 'on' with dates" %}

    Incorrect: in the 5th of June
    Correct: on the 5th of June

  • {% translate "Misusing articles with proper nouns" %}

    Incorrect: the First Avenue
    Correct: First Avenue

  • {% translate "Special cases without articles" %}

    He became the first president. ({% translate "position" %})
    He was elected president first. ({% translate "adverbial" %})
