{% translate "Decimal Fractions" %}

{% translate "Decimal fractions represent parts of whole numbers using a decimal point system. They are essential for precise measurements, financial transactions, and scientific calculations in English." %}

{% translate "1. Reading Decimal Numbers" %}

{% translate "Standard Methods" %}

{% translate "Decimal" %} {% translate "Point Method" %} {% translate "Place Value Method" %} {% translate "Context" %}
0.5 zero point five five tenths General/Math
3.14 three point one four three and fourteen hundredths Pi (π)
25.75 twenty-five point seven five twenty-five and seventy-five hundredths Money/Measurements
100.001 one hundred point zero zero one one hundred and one thousandth Precise measurements

{% translate "Special Cases" %}

  • {% translate "Money amounts often drop 'point':" %}

    $4.99 → four ninety-nine (not four point nine nine)

  • {% translate "Temperatures often use 'point':" %}

    98.6°F → ninety-eight point six

  • {% translate "Sports scores use individual digits:" %}

    7.5 → seven five (gymnastics scoring)

{% translate "2. Practical Usage" %}

{% translate "Money and Financial Contexts" %}

{% translate "Amount" %} {% translate "Formal Reading" %} {% translate "Informal Reading" %} {% translate "Notes" %}
$1.50 one dollar and fifty cents one fifty Common in prices
€2.75 two euros and seventy-five cents two seventy-five European format
£3.99 three pounds and ninety-nine pence three ninety-nine UK currency
$0.99 ninety-nine cents ninety-nine Common pricing

{% translate "Measurements and Quantities" %}

  • {% translate "Metric measurements:" %}

    1.5 liters → one point five liters or one and a half liters

  • {% translate "Time measurements:" %}

    2.5 hours → two point five hours or two and a half hours

  • {% translate "Weight measurements:" %}

    0.5 kg → zero point five kilograms or half a kilogram

{% translate "3. International Variations" %}

{% translate "Decimal Separators" %}

{% translate "Country/Region" %} {% translate "Decimal Separator" %} {% translate "Thousand Separator" %} {% translate "Example" %}
USA, UK, Australia Point (.) Comma (,) 1,234.56
Most European countries Comma (,) Point (.) or space 1.234,56 or 1 234,56
Switzerland Point (.) Apostrophe (') 1'234.56

{% translate "Important Notes" %}

  • {% translate "In English writing, always use point as decimal separator" %}
  • {% translate "Financial documents may use comma as thousand separator" %}

    $1,234.56 (correct English format)

  • {% translate "Some style guides recommend spaces for thousands in scientific writing" %}

    1 234.56 (alternative format)

{% translate "4. Common Mistakes to Avoid" %}

  • {% translate "Mixing separators:" %}

    Incorrect: 1.234,56 (English)
    Correct: 1,234.56

  • {% translate "Omitting 'and' in place value method:" %}

    Incorrect: three fourteen hundredths
    Correct: three and fourteen hundredths

  • {% translate "Incorrect money readings:" %}

    Incorrect: one point fifty dollars
    Correct: one dollar and fifty cents
