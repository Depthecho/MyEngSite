{% translate "Cardinal Numerals: Spelling and Hyphenation" %}

{% translate "Proper spelling and hyphenation of numbers are essential for clear communication in formal writing, academic papers, and professional documents. While digits are often acceptable, knowing when and how to spell out numbers is crucial." %}

{% translate "When to Spell Out Numbers" %}

  • {% translate "In formal writing and literature" %}
  • {% translate "At the beginning of sentences" %}
  • {% translate "For round numbers (hundred, thousand)" %}
  • {% translate "In dialogue and quoted speech" %}
  • {% translate "When numbers are part of proper names" %}

{% translate "1. General Spelling Rules" %}

{% translate "Style Guide" %} {% translate "Rule" %} {% translate "Example" %}
{% translate "General" %} {% translate "Spell out 0-9, use digits for 10+" %} three apples, 15 oranges
{% translate "Chicago" %} {% translate "Spell out 0-99, digits for 100+" %} ninety-nine problems, 100 solutions
{% translate "AP Style" %} {% translate "Spell out 0-9, digits for 10+ (except at sentence start)" %} Five cars parked near 12 trucks
{% translate "Scientific" %} {% translate "Use digits for all exact measurements" %} 5 ml, 3 kg, 2 cm

{% translate "Special Cases for Spelling Out Numbers" %}

  • {% translate "Always spell out numbers at sentence beginnings" %}

    {% translate "Twenty-five students passed the exam." %}

  • {% translate "Spell out indefinite numbers" %}

    {% translate "Hundreds of people attended." %}

  • {% translate "Spell out numbers in proper names" %}

    {% translate "Twenty-First Century Fox" %}

{% translate "2. Hyphenation Rules" %}

{% translate "When to Use Hyphens" %}

  • {% translate "Compound numbers 21-99" %}

    twenty-one, forty-two, ninety-nine

  • {% translate "Fractions (when written out)" %}

    two-thirds, three-fourths

  • {% translate "Age descriptions" %}

    a five-year-old child

  • {% translate "Combined numbers with nouns" %}

    a twenty-page document

{% translate "Correct" %} {% translate "Incorrect" %} {% translate "Rule" %}
twenty-one twenty one {% translate "Hyphenate 21-99" %}
one hundred twenty-five one-hundred-twenty-five {% translate "No hyphens in hundreds" %}
a three-year contract a three year contract {% translate "Hyphenate number-noun combinations" %}

{% translate "3. Punctuation with Numbers" %}

{% translate "Commas in Large Numbers" %}

  • {% translate "Use commas to separate thousands:" %}

    1,000 (one thousand)
    25,500 (twenty-five thousand, five hundred)
    1,500,000 (one million, five hundred thousand)

  • {% translate "No commas in years:" %}

    1999 (nineteen ninety-nine)

  • {% translate "No commas in page numbers or addresses:" %}

    page 1050
    1250 Main Street

{% translate "International Variations" %}

  • {% translate "Some European countries use spaces instead of commas:" %}

    1 000 000 (one million)

  • {% translate "Some use periods as thousand separators:" %}

    1.000.000 (one million)

  • {% translate "In English, always use commas for thousands and periods for decimals" %}
