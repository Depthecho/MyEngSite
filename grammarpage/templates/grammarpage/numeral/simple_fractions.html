{% translate "Simple Fractions (Common Fractions)" %}

{% translate "Simple fractions represent parts of a whole and are essential for precise communication in mathematics, measurements, and everyday situations. Mastering their correct usage improves both written and spoken English." %}

{% translate "1. Formation and Reading Rules" %}

{% translate "Basic Structure" %}

  • {% translate "Numerator (top number): cardinal number" %}

    1 (one), 2 (two), 3 (three)

  • {% translate "Denominator (bottom number): ordinal number + 's' if numerator > 1" %}

    3 (third), 4 (fourth), 5 (fifth)

  • {% translate "Special cases:" %}

    2 → half (not second)
    4 → quarter (common alternative to fourth)

{% translate "Fraction" %} {% translate "Spoken Form" %} {% translate "Rule Applied" %}
1/2 one half Special case (never "one second")
1/3 one third Standard formation
2/3 two thirds Plural denominator
1/4 one quarter/fourth Alternative forms
3/4 three quarters/fourths Plural alternatives
5/8 five eighths Irregular plural

{% translate "2. Special Fraction Forms" %}

{% translate "Common Fraction Names" %}

  • {% translate "Half (1/2) - used in:" %}

    half an hour
    half the population
    one and a half

  • {% translate "Quarter (1/4) - used in:" %}

    a quarter past three
    quarter final (sports)
    three-quarter length

  • {% translate "Third (1/3) - used in:" %}

    third party
    third world
    two-thirds majority

{% translate "Fractions in Measurements" %}

  • {% translate "With units of measurement:" %}

    1½ meters (one and a half meters)
    3¼ cups (three and a quarter cups)

  • {% translate "In time expressions:" %}

    7:45 (quarter to eight)
    12:30 (half past twelve)

{% translate "3. Grammatical Usage" %}

{% translate "Hyphenation Rules" %}

{% translate "Usage" %} {% translate "Example" %} {% translate "Rule" %}
Adjective a two-thirds majority Hyphenate when modifying a noun
Noun two thirds of the students No hyphen when standalone
Mixed numbers 1½ cups (one and a half cups) No space between whole number and fraction

{% translate "Verb Agreement" %}

  • {% translate "Singular verb when referring to a single quantity:" %}

    {% translate "Half of the cake is gone." %}

  • {% translate "Plural verb when referring to multiple items:" %}

    {% translate "Two thirds of the students are present." %}

{% translate "4. Common Mistakes to Avoid" %}

  • {% translate "Incorrect denominator forms:" %}

    Incorrect: two third
    Correct: two thirds

  • {% translate "Misusing 'half':" %}

    Incorrect: one second
    Correct: one half

  • {% translate "Wrong hyphenation:" %}

    Incorrect: a two thirds majority
    Correct: a two-thirds majority

  • {% translate "Incorrect verb agreement:" %}

    Incorrect: Two-thirds of the water are gone.
    Correct: Two-thirds of the water is gone.
