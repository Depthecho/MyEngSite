{% translate "1. Royal and Papal Numerals" %}

{% translate "Monarchs and popes use Roman numerals in their official titles, which are always read as ordinal numbers in speech." %}

{% translate "Title Type" %} {% translate "Written Form" %} {% translate "Spoken Form" %} {% translate "Example" %}
{% translate "British Monarch" %} Elizabeth II Elizabeth the Second {% translate "Queen Elizabeth the Second's coronation was in 1953" %}
{% translate "Pope" %} John Paul II John Paul the Second {% translate "Pope John Paul the Second visited over 100 countries" %}
{% translate "European Monarch" %} Louis XIV Louis the Fourteenth {% translate "Louis the Fourteenth ruled France for 72 years" %}

{% translate "Special Royal Cases" %}

  • First of name: Typically written without numeral but spoken as "the First" if distinction needed
  • Current monarchs: Spoken without ordinal until their reign ends (e.g., "King Charles" not "the Third" in daily speech)
  • Regnal numbers: Reset for different countries (Elizabeth II was only first Elizabeth of UK, not Scotland)

{% translate "2. Sequential Numbering in Titles" %}

{% translate "The same ordinal convention applies to numbered items in series, though practice varies by context." %}

{% translate "Cultural and Historical References" %}

  • World War II: World War Two or the Second World War
  • Henry VIII: Henry the Eighth (English king)
  • Pope Benedict XVI: Benedict the Sixteenth

{% translate "Modern Sequential Titles" %}

  • Rocky IV: Rocky Four (common for movies)
  • iPhone 14: iPhone fourteen (tech products)
  • Chapter XI: Chapter Eleven (books/legal)

{% translate "When Ordinals Aren't Used" %}

  • {% translate "With family names: John Smith III → John Smith the Third (formal) or John Smith Three (informal)" %}
  • {% translate "For centuries: 21st century always read as ordinal" %}
  • {% translate "With 'Junior/Senior': No numerals used" %}

  • {% translate "Never read Roman numerals as cardinals: Elizabeth Two → Elizabeth the Second" %}
  • {% translate "Don't omit 'the' with ordinals in royal names" %}
  • {% translate "For popes, always use ordinal even if numeral isn't present: Pope Francis (no number needed yet)" %}
