{% translate "Cardinal Numerals: Usage and Pluralization" %}

{% translate "Mastering the usage of cardinal numerals involves understanding their grammatical roles and proper pluralization. These rules are essential for precise communication in both spoken and written English." %}

{% translate "1. Grammatical Roles of Cardinal Numerals" %}

{% translate "As Determiners (Most Common)" %}

  • {% translate "Modify nouns directly" %}

    She has three cats.

  • {% translate "Always precede the noun they modify" %}

    Five students arrived late.

  • {% translate "Never take plural form in this role" %}

    Seven days ({% translate "not sevens days" %})

{% translate "As Pronouns" %}

  • {% translate "Stand alone without modifying a noun" %}

    {% translate "How many do you need? I need five." %}

  • {% translate "Can refer to previously mentioned items" %}

    {% translate "She bought twelve apples but only needed six." %}

{% translate "2. Pluralization Rules" %}

{% translate "When to Pluralize Numerals" %}

  • {% translate "Referring to the numbers themselves" %}

    {% translate "All the sevens in this document need correction." %}

  • {% translate "Referring to decades" %}

    {% translate "The nineties were a time of economic growth." %}

  • {% translate "Referring to age groups" %}

    {% translate "This game is popular with tens and twelves." %}

  • {% translate "Referring to playing cards or dice" %}

    {% translate "He rolled three sixes in a row." %}

{% translate "Context" %} {% translate "Singular" %} {% translate "Plural" %}
{% translate "General counting" %} three books N/A
{% translate "Referring to digits" %} the number seven all the sevens
{% translate "Decades" %} the year 1990 the nineties
{% translate "Approximate quantities" %} a hundred times hundreds of times

{% translate "3. Special Usage Cases" %}

{% translate "With Units of Measurement" %}

  • {% translate "Hyphenated compound adjectives" %}

    a five-liter container

  • {% translate "Non-hyphenated when not modifying a noun" %}

    The container holds five liters.

  • {% translate "Special cases with foot/feet" %}

    a six-foot tall man ({% translate "adjective" %})
    The man is six feet tall ({% translate "noun" %})

{% translate "With Money and Currency" %}

  • {% translate "Singular form for exact amounts" %}

    fifty dollars ({% translate "not fifties dollars" %})

  • {% translate "Plural for approximate amounts" %}

    hundreds of dollars

  • {% translate "Special forms for some currencies" %}

    five pence ({% translate "not pences" %})

{% translate "Common Mistakes and Exceptions" %}

  • {% translate "One" %} {% translate "is never pluralized" %}

    {% translate "Correct: the number one" %}
    {% translate "Incorrect: the number ones" %}

  • {% translate "Zero" %} {% translate "has two plural forms" %}

    zeros ({% translate "mathematics" %})
    zeroes ({% translate "general usage" %})

  • {% translate "Collective numbers have fixed forms" %}

    a dozen eggs ({% translate "not dozens eggs" %})
    dozens of eggs ({% translate "when approximate" %})

  • {% translate "Avoid double pluralization" %}

    {% translate "Correct: five million people" %}
    {% translate "Incorrect: five millions people" %}
