{% translate "Ordinal Numerals Functioning as Nouns" %}

{% translate "Ordinal numerals frequently serve as nouns when they represent positions or ranks independently, without modifying another noun. This usage is particularly common in competitive, historical, and temporal contexts." %}

{% translate "1. Positional References" %}

{% translate "Ordinals as nouns typically refer to people or items in a ranked sequence when the category is understood from context." %}

{% translate "Context" %} {% translate "Example" %} {% translate "Implied Noun" %}
{% translate "Competitions" %} She finished third (place/position)
{% translate "Family" %} I'm the eldest of four (child/sibling)
{% translate "Dates" %} See you on the first (day of month)
{% translate "Historical events" %} The twenty-third of the dynasty (ruler/monarch)

{% translate "Important Rules" %}

  • {% translate "Always use the definite article the with ordinal nouns: the first, not first" %}
  • {% translate "When the noun is expressed, ordinals function as adjectives: the first chapter" %}
  • {% translate "For dates, the is required: the fifteenth of June" %}

{% translate "2. Plural Ordinal Nouns" %}

{% translate "Pluralized ordinal nouns have specialized uses referring to multiple items sharing a rank or representing decades." %}

{% translate "Shared Rankings" %}

  • {% translate "Competitions:" %} There were three firsts this year ({% translate "three first-place winners" %})
  • {% translate "Sports:" %} The team had five seconds last season ({% translate "five second-place finishes" %})

{% translate "Decades and Centuries" %}

  • {% translate "Years:" %} The roaring twenties (1920s)
  • {% translate "Centuries:" %} The sixteen hundreds (1600s)
  • {% translate "Temperature:" %} The low thirties (30-34°F)
{% translate "Singular vs. Plural Ordinal Nouns" %}
{% translate "Form" %} {% translate "Example" %} {% translate "Meaning" %}
{% translate "Singular" %} the tenth {% translate "One item in 10th position" %}
{% translate "Plural" %} the tenths {% translate "Multiple items in 10th position" %}
