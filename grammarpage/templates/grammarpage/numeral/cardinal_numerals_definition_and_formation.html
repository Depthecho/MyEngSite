{% translate "Cardinal Numerals: Definition and Formation" %}

{% translate "Cardinal numerals (also known as cardinal numbers) are words that represent specific quantities. They answer the question 'how many?' and are fundamental for counting, measuring, and quantifying in English." %}

{% translate "Key Characteristics" %}

  • {% translate "Represent exact quantities (one, two, three)" %}
  • {% translate "Used for counting objects, people, or abstract concepts" %}
  • {% translate "Form the basis for mathematical operations" %}
  • {% translate "Can function as determiners or pronouns" %}
  • {% translate "Distinct from ordinal numbers (first, second, third)" %}

{% translate "1. Basic Number Formation" %}

{% translate "English cardinal numbers follow specific patterns that make them relatively easy to learn, though there are some irregularities." %}

{% translate "Range" %} {% translate "Pattern" %} {% translate "Examples" %} {% translate "Notes" %}
0-12 {% translate "Unique words" %} zero, one, five, twelve {% translate "Must be memorized" %}
13-19 {% translate "Base + -teen" %} thirteen, fifteen, nineteen {% translate "Note: thirteen (not threeteen)" %}
20-90 (tens) {% translate "Base + -ty" %} twenty, fifty, ninety {% translate "Note: forty (not fourty)" %}
21-99 {% translate "Tens + hyphen + unit" %} twenty-one, sixty-five, ninety-nine {% translate "Hyphen required" %}

{% translate "Pronunciation Tips" %}

  • {% translate "-teen" %} {% translate "numbers have stress on the last syllable: thirTEEN" %}
  • {% translate "Tens numbers have stress on the first syllable: TWENty" %}
  • {% translate "In compound numbers (21-99), both parts are stressed: TWENty-ONE" %}

{% translate "2. Large Number Formation" %}

{% translate "English uses specific terms for large numbers, with some differences between American and British English." %}

{% translate "Number" %} {% translate "American English" %} {% translate "British English" %}
100 one hundred one hundred
1,000 one thousand one thousand
1,000,000 one million one million
1,000,000,000 one billion one thousand million
1,000,000,000,000 one trillion one billion

{% translate "Forming Large Numbers" %}

{% translate "When writing or saying large numbers:" %}

  • {% translate "Start with the largest place value" %}
  • {% translate "Use commas to separate thousands, millions, etc." %}
  • {% translate "Use 'and' before the last two digits in British English" %}
  • {% translate "Don't use 'and' in American English except for decimals" %}

{% translate "Examples:" %}

  • 125: one hundred twenty-five (US) / one hundred and twenty-five (UK)
  • 3,450: three thousand, four hundred fifty (US) / three thousand, four hundred and fifty (UK)
  • 2,000,307: two million, three hundred seven (US) / two million, three hundred and seven (UK)

{% translate "3. Special Cases and Exceptions" %}

{% translate "Irregular Forms" %}

  • 0: zero, nought (UK), oh (in phone numbers)
  • 11, 12: eleven, twelve (not oneteen, twoteen)
  • 13, 15: thirteen, fifteen (not threeteen, fiveteen)
  • 20, 30, 40, 50: twenty, thirty, forty, fifty

{% translate "Collective Numerals" %}

{% translate "Some numbers have special forms when counting specific items:" %}

  • a pair of (for two items usually used together)
  • a dozen (12)
  • a score (20 - archaic)
  • a gross (144)

{% translate "Common Mistakes to Avoid" %}

  • {% translate "Using plural forms incorrectly: three hundred (not three hundreds)" %}
  • {% translate "Forgetting hyphens in compound numbers: twenty-one (not twenty one)" %}
  • {% translate "Mixing American and British conventions in the same text" %}
  • {% translate "Using 'and' incorrectly in American English: one hundred five (not one hundred and five)" %}

{% translate "4. Practical Usage Examples" %}

{% translate "Context" %} {% translate "Example" %} {% translate "Notes" %}
{% translate "Counting objects" %} There are twenty-four students in the class. {% translate "Hyphen required" %}
{% translate "Age" %} She is sixteen (years old). "Years old" {% translate "is optional" %}
{% translate "Temperature" %} It's minus five degrees. {% translate "Use \"minus\" for negative" %}
{% translate "Money" %} $4.50 - four dollars and fifty cents "And" {% translate "used before cents" %}
