{% translate "Noun Agreement with Fractions" %}

{% translate "Proper noun agreement with fractions ensures grammatical accuracy in both written and spoken English. These rules govern how nouns and verbs interact with fractional quantities." %}

{% translate "1. Core Agreement Rules" %}

{% translate "Fraction Value Determines Agreement" %}

{% translate "Fraction Value" %} {% translate "Noun Form" %} {% translate "Verb Form" %} {% translate "Example" %}
Less than 1 Singular Singular Two-thirds of the cake is eaten
Equal to 1 Singular Singular One whole pie is missing
Greater than 1 Plural Plural One and a half cups are needed

{% translate "Special Cases" %}

  • {% translate "Fractions of one with plural nouns:" %}

    Half of the students are present

  • {% translate "Mixed numbers with measurement units:" %}

    1½ teaspoons were added

{% translate "2. Countable vs. Uncountable Nouns" %}

{% translate "Countable Nouns" %}

{% translate "Example" %} {% translate "Noun Form" %} {% translate "Verb Agreement" %} {% translate "Rule" %}
Three-quarters of the apples Plural Plural Portion of countable plural
One-third of the team Singular Singular Collective noun treated as unit
Two-fifths of the books Plural Plural Multiple countable items

{% translate "Uncountable Nouns" %}

  • {% translate "Always singular:" %}

    Three-fourths of the water is contaminated

  • {% translate "No plural form:" %}

    Half of the information is inaccurate

  • {% translate "Measurement units:" %}

    0.75 grams of gold was used

{% translate "3. Special Considerations" %}

{% translate "Collective Nouns" %}

  • {% translate "Can take singular or plural depending on meaning:" %}

    Half of the team is injured (unit)
    Half of the team are arguing (individuals)

{% translate "Percentage Equivalents" %}

  • {% translate "Follow same rules as fractions:" %}

    50% of the data is missing
    50% of the participants are women

{% translate "Money and Measurements" %}

  • {% translate "Treat amounts as singular:" %}

    Three-quarters of a million dollars was spent

{% translate "4. Common Errors" %}

{% translate "Incorrect" %} {% translate "Correct" %} {% translate "Reason" %}
Two-thirds of the students is here Two-thirds of the students are here Plural countable noun
One and a half hour passed One and a half hours passed Fraction >1 requires plural
Three-fourths of the equipment are broken Three-fourths of the equipment is broken Uncountable noun
